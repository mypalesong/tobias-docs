SQLAlchemy + PostgreSQL 완벽 가이드

Python에서 가장 강력한 ORM으로 PostgreSQL을 우아하게 다루세요!

SQLAlchemy는 Python의 사실상 표준 ORM(Object-Relational Mapping) 라이브러리입니다. 이 가이드에서는 PostgreSQL 데이터베이스와 함께 SQLAlchemy를 실전에서 활용하는 방법을 다룹니다.

🚀 왜 SQLAlchemy인가?​

Raw SQL vs ORM 비교​

Raw SQL:

# SQL 직접 작성
cursor.execute("""
    SELECT users.*, posts.title
    FROM users
    LEFT JOIN posts ON users.id = posts.user_id
    WHERE users.active = true
""")
results = cursor.fetchall()

# 딕셔너리로 수동 변환 필요
users = [dict(zip([col[0] for col in cursor.description], row))
         for row in results]

SQLAlchemy ORM:

# Python 객체로 작업
users = session.query(User)\
    .join(Post)\
    .filter(User.active == True)\
    .all()

# 이미 객체로 변환됨!
for user in users:
    print(user.name, user.posts)

SQLAlchemy의 장점​

  1. 타입 안전성: Python 타입 힌트와 IDE 자동완성
  2. 데이터베이스 독립성: PostgreSQL, MySQL, SQLite 등 전환 용이
  3. 자동 SQL 생성: 복잡한 쿼리도 Python 코드로
  4. 객체 지향: 데이터를 객체로 다룸
  5. 마이그레이션: Alembic으로 스키마 변경 관리

🌟 SQLAlchemy 2.0 vs 1.4​

이 가이드는 SQLAlchemy 2.0 최신 문법을 사용합니다.

주요 변화​

1.4 (레거시):

# 구식 Query API
users = session.query(User).filter(User.name == 'Alice').all()

2.0 (최신):

# 새로운 select API
from sqlalchemy import select

users = session.execute(
    select(User).where(User.name == 'Alice')
).scalars().all()

이 가이드는 2.0 문법을 기본으로 하되, 중요한 차이점을 설명합니다.

🎯 빠른 시작​

5분 만에 시작하기​

from sqlalchemy import create_engine, Column, Integer, String
from sqlalchemy.orm import declarative_base, Session

# 1. 데이터베이스 연결
engine = create_engine('postgresql://user:password@localhost/mydb')
Base = declarative_base()

# 2. 모델 정의
class User(Base):
    __tablename__ = 'users'

    id = Column(Integer, primary_key=True)
    name = Column(String(50), nullable=False)
    email = Column(String(100), unique=True)

# 3. 테이블 생성
Base.metadata.create_all(engine)

# 4. 데이터 삽입
with Session(engine) as session:
    user = User(name='Alice', email='alice@example.com')
    session.add(user)
    session.commit()

# 5. 데이터 조회
with Session(engine) as session:
    users = session.query(User).all()
    for user in users:
        print(user.name, user.email)
