고급편 소개

SQLAlchemy의 고급 기능을 마스터하여 전문가 수준의 데이터베이스 애플리케이션을 개발하세요.

🎯 학습 목표​

고급편에서는 실무에서 복잡한 요구사항을 해결하는 전문가 수준의 기술을 배웁니다:

  • 고급 쿼리 기법: Window Functions, CTE, Raw SQL
  • 성능 최적화: 인덱스, 쿼리 분석, 캐싱 전략
  • 트랜잭션 관리: ACID, 격리 수준, 분산 트랜잭션
  • 고급 패턴: 이벤트 시스템, 하이브리드 속성

📚 학습 순서​

1. 고급 쿼리 기법​

PostgreSQL과 SQLAlchemy의 고급 기능을 활용합니다.

from sqlalchemy import select, func

# Window Functions
stmt = select(
    User.username,
    func.row_number().over(
        partition_by=User.department,
        order_by=User.salary.desc()
    ).label('rank')
)

# CTE (Common Table Expression)
popular_posts = select(Post).where(Post.views > 1000).cte()
stmt = select(User).join(popular_posts, User.id == popular_posts.c.user_id)

2. 성능 최적화​

쿼리 성능을 극대화하는 기법들을 배웁니다.

# 인덱스 활용
class User(Base):
    __table_args__ = (
        Index('idx_email_active', 'email', 'is_active'),
    )

# 부분 로딩
stmt = select(User.id, User.username)  # 필요한 컬럼만

# Bulk 작업
session.bulk_insert_mappings(User, data)  # 대량 삽입 최적화

💡 고급편의 핵심​

PostgreSQL 특화 기능​

# Full-Text Search
stmt = select(Article).where(
    func.to_tsvector('english', Article.content).match('python')
)

# JSONB 연산자
stmt = select(Product).where(
    Product.specs['cpu'].astext == 'Intel i7'
)

# Array 함수
stmt = select(Post).where(
    Post.tags.contains(['python', 'tutorial'])
)

성능 측정과 최적화​

# EXPLAIN 분석
from sqlalchemy import event

@event.listens_for(engine, "before_cursor_execute")
def receive_before_cursor_execute(conn, cursor, statement, params, context, executemany):
    if 'EXPLAIN' not in statement:
        statement = f"EXPLAIN ANALYZE {statement}"
    print(statement)

동시성 제어​

# 재고 관리 (Race Condition 방지)
with session.begin():
    product = session.query(Product).with_for_update().get(product_id)

    if product.stock >= quantity:
        product.stock -= quantity
        order = Order(product=product, quantity=quantity)
        session.add(order)
    else:
        raise OutOfStockError()

stmt = select(User).options(

2. 대량 데이터 처리​

# 메모리 효율적 처리
def process_all_users():
    page = 0
    while True:
        stmt = select(User).limit(1000).offset(page * 1000)
        users = session.execute(stmt).scalars().all()

        if not users:
            break

        for user in users:
            process(user)

        session.expunge_all()  # 메모리 해제
        page += 1

3. 쿼리 캐싱​

from dogpile.cache import make_region

region = make_region().configure('dogpile.cache.memory')

@region.cache_on_arguments()
def get_popular_posts():
    return session.execute(
        select(Post).where(Post.views > 1000)
    ).scalars().all()
