교육 및 학습 시나리오

AI를 활용한 효과적인 학습과 교육

교육 분야에서 Prompt Engineering은 개인화 학습, 개념 설명, 퀴즈 생성 등에 활용됩니다. 이 가이드에서는 다양한 교육 상황에서 효과적인 프롬프트를 작성하는 방법을 다룹니다.

📚 시나리오 1: 개념 설명​

"""
개념 설명 요청:

주제: 파이썬의 데코레이터
학습자: 프로그래밍 6개월 경험 (함수 개념은 이해)
학습 목표: 데코레이터를 이해하고 간단히 사용할 수 있다

설명 방식:
1. 일상생활 비유로 시작
   - 쉬운 예: 선물 포장
   - 연결: 함수 '꾸미기'

2. 기본 개념
   - 정의: 함수를 받아 함수를 반환
   - 용도: 기능 추가 without 수정

3. 간단한 예제 (3단계)

   ## 단계 1: 기본 함수
   ```python
   def greet(name):
       return f"Hello, {name}"

단계 2: 수동 래핑​

def wrapper(func):
    def inner(name):
        print("Before")
        result = func(name)
        print("After")
        return result
    return inner

greet = wrapper(greet)

단계 3: @ 문법​

@wrapper
def greet(name):
    return f"Hello, {name}"

  1. 실용 예제

    • 로깅
    • 실행 시간 측정
    • 권한 체크

  2. 연습 문제 (3개)

    • 난이도: 쉬움 → 중간 → 어려움
    • 각 문제에 힌트 포함

  3. 요약

    • 핵심 3가지
    • 다음 학습: 클래스 데코레이터

요구사항:

  • 코드 예제마다 주석
  • 실행 가능한 코드
  • "왜?"에 대한 설명 포함
  • 흔한 실수와 해결법
  • 추가 학습 자료 링크 """

numbers = [1, 2, 3, 4, 5]

모범 답안:

odd_numbers = [x for x in range(1, 21) if x % 2 != 0]

  • 리스트 컴프리헨션 사용 (3점)

📚 시나리오 5: 맞춤형 피드백​

"""
학습자 코드 피드백 제공:

학습자 레벨: 초급 (3개월 경험)
제출 코드:

```python
def calculate_average(numbers):
    sum = 0
    for i in range(len(numbers)):
        sum = sum + numbers[i]
    return sum / len(numbers)

과제: "리스트의 평균 계산 함수"

피드백 요청:

🎯 종합 평가​

  • 기능 작동: ✅ / ❌
  • 코드 품질: ⭐⭐⭐ / 5
  • 학습 목표 달성: ✅ / ❌

✅ 잘한 점​

  1. 함수 구조 올바름
  2. 변수명 의미있게 작성
  3. 기본 로직 정확

💡 개선 제안 (초급자 맞춤)​

1. Pythonic한 방식 사용 (중요도: 중)​

현재:

sum = 0
for i in range(len(numbers)):
    sum = sum + numbers[i]

개선:

total = sum(numbers)  # 내장 함수 활용

왜?:

  • 더 읽기 쉬움
  • 실수 줄임
  • Python스러움

2. 에러 처리 추가 (중요도: 상)​

현재: 빈 리스트 시 ZeroDivisionError

개선:

def calculate_average(numbers):
    if not numbers:  # 빈 리스트 체크
        return 0  # 또는 raise ValueError
    return sum(numbers) / len(numbers)

3. 타입 힌트 (중요도: 하)​

def calculate_average(numbers: list[float]) -> float:
    ...

📚 다음 학습 단계​

  1. Python 내장 함수 탐색 (sum, len, max, min)
  2. 예외 처리 (try-except)
  3. 타입 힌트 기초

💪 연습 과제​

같은 방식으로 다음 함수들을 작성해보세요:

  • calculate_median (중앙값)
  • calculate_mode (최빈값)

피드백 원칙:

  • 격려와 동기 부여 """
