Few-shot Learning

예시로 AI를 가르치는 강력한 기법

Few-shot Learning은 몇 개의 예시를 제공하여 AI가 패턴을 학습하고 유사한 작업을 수행하도록 하는 기법입니다. Zero-shot보다 정확하고 일관된 결과를 얻을 수 있습니다.

🎯 학습 목표​

  • Few-shot Learning의 개념과 장점
  • Zero-shot vs One-shot vs Few-shot 비교
  • 효과적인 예제 작성 방법
  • 다양한 작업에 Few-shot 적용
  • Few-shot 성능 최적화

📚 개념 이해​

정의​

few_shot_learning = {
    "정의": "몇 개의 예시를 통해 AI에게 패턴을 학습시키는 기법",
    "구성": "입력-출력 쌍의 예시 + 새로운 입력",
    "목적": "일관성 있고 정확한 결과 도출",
    "예시 수": "보통 2-5개 (상황에 따라 조정)"
}

학습 방식 비교​

방식예시 개수장점단점사용 시기
Zero-shot0개빠르고 간단일관성 낮음일반적 작업
One-shot1개적은 예시로 패턴 제시패턴 학습 제한적간단한 패턴
Few-shot2-10개높은 정확도와 일관성프롬프트 길어짐복잡한 패턴

📊 Few-shot vs Zero-shot 선택 기준​

def choose_approach(task):
    if task.is_standard and task.is_simple:
        return "Zero-shot"  # 예: 번역, 요약

    if task.requires_specific_format:
        return "Few-shot"  # 예: JSON 생성, 특정 스타일

    if task.has_edge_cases:
        return "Few-shot"  # 예: 복잡한 분류

    if task.needs_consistency:
        return "Few-shot"  # 예: 대량 데이터 처리

    if task.is_creative:
        return "Zero-shot 또는 Few-shot"  # 상황에 따라

    return "Few-shot 우선, 필요시 Zero-shot"

의사결정 플로우차트​

시작
  ↓
일관된 형식이 필요한가?
  ↓ Yes → Few-shot
  ↓ No
복잡한 패턴인가?
  ↓ Yes → Few-shot
  ↓ No
예외 케이스가 많은가?
  ↓ Yes → Few-shot
  ↓ No
  → Zero-shot

✅ Few-shot 체크리스트​

예제 품질​

  • 예제가 다양한가? (다른 케이스 포함)
  • 예제가 명확한가? (모호함 없음)
