실전 시나리오 소개

실무에서 바로 사용할 수 있는 WSL 2 활용 시나리오

이 섹션에서는 다양한 개발 환경을 WSL 2에서 구축하는 실전 가이드를 제공합니다. 각 시나리오는 실제 개발자들의 작업 환경을 기반으로 작성되었습니다.

🎯 시나리오 목록​

1. 풀스택 개발 환경 구축​

대상: 프론트엔드 + 백엔드 개발자

Julia는 새로운 스타트업에 풀스택 개발자로 입사했습니다. Windows PC에서 React 프론트엔드와 Node.js 백엔드를 모두 개발해야 합니다.

배우는 내용:

  • Node.js, npm, nvm 설치
  • VS Code + WSL 통합
  • Git 설정 및 SSH 키 생성
  • 프론트엔드 + 백엔드 동시 실행
  • 포트 포워딩 및 네트워킹

👉 시나리오 보기

2. Docker 및 컨테이너 개발​

대상: 백엔드 개발자, DevOps 엔지니어

Dockerize된 마이크로서비스 아키텍처를 개발하는 민수. WSL 2에서 Docker를 활용하여 로컬 개발 환경을 구축합니다.

배우는 내용:

  • Docker Desktop 설정
  • Docker Compose로 멀티 컨테이너 구성
  • 컨테이너 네트워킹
  • 볼륨 마운트 및 데이터 관리
  • 성능 최적화

👉 시나리오 보기

3. Node.js 웹 개발 환경​

대상: 웹 개발자, 프론트엔드 개발자

지수는 최신 웹 프레임워크로 프로젝트를 시작합니다. Next.js, React, TypeScript 환경을 WSL 2에서 구축합니다.

배우는 내용:

  • nvm으로 Node.js 버전 관리
  • Next.js/React 프로젝트 설정
  • TypeScript 개발 환경
  • Hot reload 및 빠른 빌드
  • 프로덕션 배포 준비

👉 시나리오 보기

4. Python 데이터베이스 백엔드 개발​

대상: 백엔드 개발자, 데이터 엔지니어

수진은 FastAPI로 REST API를 개발하고 PostgreSQL 데이터베이스를 사용합니다. WSL 2에서 완벽한 Python 백엔드 환경을 만듭니다.

배우는 내용:

  • Python 가상 환경 (venv, pyenv)
  • PostgreSQL 설치 및 관리
  • FastAPI/Django 개발
  • DB 마이그레이션 (Alembic)
  • API 테스팅

👉 시나리오 보기

5. DevOps 및 CI/CD 자동화​

대상: DevOps 엔지니어, SRE

현우는 팀의 CI/CD 파이프라인을 관리합니다. WSL 2에서 Kubernetes, Terraform, Ansible 등 DevOps 도구를 활용합니다.

배우는 내용:

  • kubectl 및 minikube 설정
  • Terraform 인프라 코드
  • Ansible 플레이북 실행
  • GitHub Actions 로컬 테스트
  • 자동화 스크립트

👉 시나리오 보기

6. 머신러닝 & AI 개발 환경​

대상: 데이터 과학자, ML 엔지니어

ML 모델을 훈련시키는 예진. WSL 2에서 GPU를 활용하여 TensorFlow와 PyTorch 환경을 구축합니다.

배우는 내용:

  • CUDA 및 cuDNN 설정
  • Jupyter Notebook 환경
  • TensorFlow/PyTorch 설치
  • GPU 가속 활용
  • 대용량 데이터 처리

👉 시나리오 보기

📋 시나리오 활용 방법​

1. 목적에 맞는 시나리오 선택​

자신의 개발 스택과 가장 가까운 시나리오를 선택하세요.

2. 단계별로 따라하기​

각 시나리오는 실습 위주로 구성되어 있습니다. 코드를 복사하여 직접 실행해보세요.

3. 커스터마이징​

기본 설정을 이해한 후, 자신의 프로젝트에 맞게 수정하세요.

4. 문제 해결​

막히는 부분이 있다면 문제 해결 섹션을 확인하세요.
