파일 시스템 탐색

Linux 파일 시스템을 자유자재로 이동하는 방법을 배웁니다.

🗺️ Linux 디렉토리 구조​

Linux는 트리 구조로 되어 있습니다:

/                    (루트 - 모든 것의 시작)
├── home/           (사용자 홈 디렉토리)
│   └── username/   (당신의 홈)
├── etc/            (설정 파일)
├── var/            (변하는 데이터)
├── usr/            (사용자 프로그램)
├── bin/            (필수 명령어)
├── tmp/            (임시 파일)
└── mnt/            (마운트 포인트)
    └── c/          (WSL: Windows C 드라이브)

📍 pwd - 현재 위치 확인​

Print Working Directory

pwd
# 출력: /home/username

사용 예:

# 어디 있는지 모를 때
pwd

# 절대 경로 확인
cd /etc
pwd
# /etc

📂 ls - 파일 목록 보기​

LiSt

기본 사용법​

# 현재 디렉토리 파일 목록
ls

# 특정 디렉토리 목록
ls /home

# 숨김 파일 포함 (. 으로 시작)
ls -a

# 자세한 정보
ls -l

# 조합 (자주 사용!)
ls -la

# 사람이 읽기 쉬운 크기
ls -lh

출력 읽기​

ls -l
drwxr-xr-x 2 user group 4096 Jan 15 10:30 Documents
-rw-r--r-- 1 user group  234 Jan 15 09:20 file.txt
└─┬──┘└┬─┘ └┬ └─┬─┘ └─┬─┘ └────┬────┘ └────┬────┘ └───┬───┘
  │    │    │   │     │         │            │         │
  │    │    │   │     │         │            │         파일명
  │    │    │   │     │         │            수정 시간
  │    │    │   │     │         크기 (bytes)
  │    │    │   │     그룹
  │    │    │   소유자
  │    │    링크 수
  │    권한
  파일 타입 (d: 디렉토리, -: 파일, l: 링크)

유용한 옵션​

# 최신 파일 먼저
ls -lt

# 크기순 정렬
ls -lS

# 역순 정렬
ls -lr

# 재귀적 (하위 디렉토리 포함)
ls -R

# 트리 형태 (tree 설치 필요)
tree

# 색상 추가 (대부분 기본 활성화)
ls --color=auto

🚶 cd - 디렉토리 이동​

Change Directory

기본 사용법​

# 홈 디렉토리로
cd ~
# 또는 그냥
cd

# 특정 디렉토리로
cd /etc

# 상대 경로로 이동
cd Documents

# 한 단계 위로
cd ..

# 두 단계 위로
cd ../..

# 이전 위치로
cd -

# 루트로
cd /

절대 경로 vs 상대 경로​

# 절대 경로 (/ 로 시작)
cd /home/username/Documents

# 상대 경로 (현재 위치 기준)
cd Documents  # 현재 위치에서 Documents로

# 예시
pwd  # /home/username
cd Documents  # /home/username/Documents로 이동
cd ../Downloads  # /home/username/Downloads로 이동

특수 디렉토리​

# ~ : 홈 디렉토리
cd ~/Desktop

# . : 현재 디렉토리
cd .  # 아무 변화 없음 (하지만 다른 명령어에서 유용)

# .. : 부모 디렉토리
cd ..

# - : 이전 디렉토리
cd -  # 바로 전에 있던 위치로

🎯 실전 연습​

연습 1: 파일 시스템 탐색​

# 1. 현재 위치 확인
pwd

# 2. 파일 목록 보기
ls -la

# 3. 루트로 이동
cd /

# 4. 루트 구조 확인
ls -l

# 5. etc 디렉토리 내용 확인
cd etc
ls

# 6. 홈으로 돌아가기
cd ~

# 7. 위치 확인
pwd

연습 2: WSL Windows 파일 접근​

# Windows C 드라이브로
cd /mnt/c

# 사용자 폴더로
cd /mnt/c/Users/YourName

# 문서 폴더
cd /mnt/c/Users/YourName/Documents

# 다운로드 폴더
cd /mnt/c/Users/YourName/Downloads

연습 3: 디렉토리 생성 및 탐색​

# 연습용 디렉토리 생성
mkdir -p ~/practice/level1/level2/level3

# 한 번에 깊이 들어가기
cd ~/practice/level1/level2/level3

# 위치 확인
pwd
# /home/username/practice/level1/level2/level3

# 단계적으로 올라가기
cd ..
pwd  # /home/username/practice/level1/level2

cd ../..
pwd  # /home/username/practice

# 홈으로
cd

💡 프로 팁​

1. Tab 자동완성​

# Doc 타이핑 후 Tab
cd Doc<Tab>
# → cd Documents/

# 여러 옵션이 있으면 Tab 두 번
cd D<Tab><Tab>
# Desktop/  Documents/  Downloads/

2. 와일드카드 사용​

# 모든 .txt 파일
ls *.txt

# 모든 .jpg와 .png 파일
ls *.{jpg,png}

# a로 시작하는 모든 파일
ls a*

# 단일 문자 매칭
ls file?.txt  # file1.txt, fileA.txt 등

3. 유용한 Alias 추가​

~/.bashrc에 추가:

# 자주 사용하는 ls 옵션
alias ll='ls -alF'
alias la='ls -A'
alias l='ls -CF'

# 이동 단축키
alias ..='cd ..'
alias ...='cd ../..'
alias ....='cd ../../..'

# 안전한 조작
alias rm='rm -i'
alias cp='cp -i'
alias mv='mv -i'

적용:

source ~/.bashrc

4. pushd/popd - 디렉토리 스택​

여러 디렉토리를 오갈 때:

# 현재 위치를 스택에 저장하고 이동
pushd /etc
pushd /var/log
pushd ~/Documents

# 디렉토리 스택 확인
dirs -v

# 이전 디렉토리로 (스택에서 꺼내기)
popd

🔍 고급 기능​

find로 파일 찾기​

# 이름으로 찾기
find ~ -name "*.txt"

# 타입으로 찾기
find ~ -type f  # 파일
find ~ -type d  # 디렉토리

# 크기로 찾기
find ~ -size +10M  # 10MB 이상

# 최근 수정된 파일
find ~ -mtime -7  # 7일 이내

which - 명령어 위치 찾기​

which python
# /usr/bin/python

which node
# /home/username/.nvm/versions/node/v20.10.0/bin/node

locate - 빠른 파일 검색​

# 데이터베이스 업데이트 (처음 한 번)
sudo updatedb

# 파일 찾기 (find보다 빠름)
locate file.txt
