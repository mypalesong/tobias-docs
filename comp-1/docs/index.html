GitHub 마스터 완벽 가이드

기초부터 고급까지, GitHub의 모든 것을 마스터하세요!

이 가이드는 Git과 GitHub를 처음 접하는 초보자부터 고급 기능을 활용하고 싶은 개발자까지 모두를 위한 완벽한 학습 자료입니다.

🎯 학습 목표​

  • 기초편: Git 설치부터 기본적인 워크플로우까지
  • 중급편: Pull Request, Issues, GitHub Actions 등 협업 도구
  • 고급편: CI/CD, API, Hooks 등 고급 기능 마스터

📚 학습 순서​

1️⃣ 기초편 (Basic)​

Git의 기본 개념과 필수 명령어를 배웁니다.

  • Git 설치 및 초기 설정
  • 저장소 생성 및 클론
  • Add, Commit, Push 워크플로우
  • 브랜치 기초
  • 변경 이력 확인
  • 변경 사항 취소
  • .gitignore 사용

2️⃣ 중급편 (Intermediate)​

GitHub를 활용한 협업과 자동화를 배웁니다.

  • Pull Request 생성 및 관리
  • Issues 활용
  • 협업 워크플로우
  • GitHub Actions 기초
  • Rebase 활용
  • Stash 사용
  • Tag 관리
  • GitHub Projects

3️⃣ 고급편 (Advanced)​

전문가 수준의 Git 활용법을 배웁니다.

  • Git Hooks
  • CI/CD 파이프라인
  • GitHub API
  • Submodules
  • Worktree
  • Bisect
  • GitHub Packages
  • Advanced Commands
  • Security Features
  • Monorepo 관리

💡 이 가이드의 특징​

  • ✅ 실습 위주: 모든 예제는 실제로 따라할 수 있습니다
  • ✅ 단계별 학습: 쉬운 것부터 어려운 것까지 체계적으로
  • ✅ 실무 중심: 실제 프로젝트에서 바로 사용할 수 있는 내용
  • ✅ 한글 지원: 모든 내용이 한글로 작성되어 있습니다

🚀 시작하기​

왼쪽 사이드바에서 원하는 주제를 선택하여 학습을 시작하세요!

처음이시라면 Git 설치 및 초기 설정부터 시작하는 것을 추천합니다.
