React 고급

React를 마스터하기 위한 고급 개념과 패턴을 학습합니다.

고급 과정 개요​

이 과정에서는 React의 고급 기능과 성능 최적화, 복잡한 패턴을 다룹니다.

학습 내용​

1. 성능 최적화​

React 애플리케이션의 성능을 최적화하는 방법을 배웁니다.

  • React.memo
  • useMemo와 useCallback
  • 가상화 (Virtualization)
  • Code Splitting

2. Code Splitting​

애플리케이션 번들을 분할하여 로딩 성능을 개선합니다.

  • React.lazy
  • Suspense
  • 동적 import

3. 디자인 패턴​

React에서 자주 사용되는 디자인 패턴을 학습합니다.

  • Compound Components
  • Render Props
  • Higher-Order Components (HOC)
  • Container/Presentational Pattern

4. TypeScript with React​

TypeScript를 React와 함께 사용하는 방법을 배웁니다.

  • 타입 정의
  • Props 타입
  • Hooks 타입
  • Generic Components

5. 테스팅​

React 컴포넌트를 테스트하는 방법을 학습합니다.

  • Jest와 React Testing Library
  • 단위 테스트
  • 통합 테스트
  • E2E 테스트

6. Server-Side Rendering​

서버 사이드 렌더링의 개념과 구현 방법을 배웁니다.

  • Next.js
  • SEO 최적화
  • Hydration

7. 고급 Hooks 패턴​

복잡한 Hooks 패턴과 사용법을 학습합니다.

  • useImperativeHandle
  • useLayoutEffect
  • useDebugValue
  • Custom Hook 고급 패턴

사전 요구사항​

  • Hooks 완벽 이해
  • useEffect 숙달
  • Custom Hooks 작성 경험
  • Context API 사용 경험

학습 목표​

이 과정을 완료하면 다음을 할 수 있습니다:

  • ✓ 대규모 React 애플리케이션 아키텍처 설계
  • ✓ 성능 병목 지점 식별 및 최적화
  • ✓ 복잡한 상태 관리 패턴 구현
  • ✓ TypeScript로 타입 안전한 코드 작성
  • ✓ 포괄적인 테스트 작성
  • ✓ SSR 애플리케이션 구축

실전 프로젝트​

고급 과정에서는 다음과 같은 프로젝트를 권장합니다:

  1. E-commerce 플랫폼: 복잡한 상태 관리와 최적화
  2. 소셜 미디어 앱: 실시간 업데이트와 무한 스크롤
  3. 대시보드: 데이터 시각화와 복잡한 UI
  4. 블로그 플랫폼: SSR과 SEO 최적화
