Props 사용하기

Props(properties의 줄임말)는 컴포넌트에 데이터를 전달하는 방법입니다.

Props란?​

Props는 부모 컴포넌트에서 자식 컴포넌트로 데이터를 전달하는 읽기 전용 객체입니다.

function Welcome(props) {
  return <h1>안녕하세요, {props.name}님!</h1>;
}

// 사용
<Welcome name="홍길동" />

Props 전달하기​

기본 사용법​

function Greeting({ name, age }) {
  return (
    <div>
      <h1>안녕하세요, {name}님!</h1>
      <p>나이: {age}세</p>
    </div>
  );
}

// 사용
<Greeting name="홍길동" age={25} />

여러 Props 전달​

function UserCard({ name, email, avatar, role }) {
  return (
    <div className="user-card">
      <img src={avatar} alt={name} />
      <h3>{name}</h3>
      <p>{email}</p>
      <span>{role}</span>
    </div>
  );
}

<UserCard
  name="홍길동"
  email="hong@example.com"
  avatar="avatar.jpg"
  role="개발자"
/>

Props 구조 분해​

매개변수에서 구조 분해​

// ✅ 권장: 구조 분해 사용
function UserInfo({ name, email }) {
  return (
    <div>
      <h3>{name}</h3>
      <p>{email}</p>
    </div>
  );
}

// ❌ 비권장: props 객체 사용
function UserInfo(props) {
  return (
    <div>
      <h3>{props.name}</h3>
      <p>{props.email}</p>
    </div>
  );
}

함수 본문에서 구조 분해​

function UserInfo(props) {
  const { name, email, avatar } = props;

  return (
    <div>
      <img src={avatar} alt={name} />
      <h3>{name}</h3>
      <p>{email}</p>
    </div>
  );
}

Props 기본값​

함수 매개변수 기본값​

function Button({ text = "클릭", variant = "primary" }) {
  return (
    <button className={`btn-${variant}`}>
      {text}
    </button>
  );
}

// 기본값 사용
<Button />  // "클릭", "primary"

// 값 전달
<Button text="저장" variant="success" />

defaultProps (클래스 컴포넌트)​

function Button({ text, variant }) {
  return (
    <button className={`btn-${variant}`}>
      {text}
    </button>
  );
}

Button.defaultProps = {
  text: "클릭",
  variant: "primary"
};

Props의 타입​

다양한 타입 전달​

function Demo({
  stringProp,      // 문자열
  numberProp,      // 숫자
  booleanProp,     // 불리언
  arrayProp,       // 배열
  objectProp,      // 객체
  functionProp,    // 함수
  elementProp      // React 엘리먼트
}) {
  return <div>Demo Component</div>;
}

// 사용
<Demo
  stringProp="안녕하세요"
  numberProp={42}
  booleanProp={true}
  arrayProp={[1, 2, 3]}
  objectProp={{ key: 'value' }}
  functionProp={() => console.log('클릭')}
  elementProp={<span>자식 요소</span>}
/>

문자열 Props​

// 따옴표 사용
<Welcome name="홍길동" />

// 중괄호 사용 (변수)
const userName = "홍길동";
<Welcome name={userName} />

숫자 Props​

// 반드시 중괄호 사용
<Counter count={10} />
<Price amount={19.99} />

불리언 Props​

// true는 속성만 써도 됨
<Button disabled />
<Button disabled={true} />

// false
<Button disabled={false} />

children Prop​

특별한 prop으로, 컴포넌트 태그 사이의 내용을 받습니다.

function Card({ children, title }) {
  return (
    <div className="card">
      <h3>{title}</h3>
      <div className="card-content">
        {children}
      </div>
    </div>
  );
}

// 사용
<Card title="제목">
  <p>여기는 children입니다.</p>
  <button>버튼</button>
</Card>

children 활용 예제​

function Container({ children }) {
  return <div className="container">{children}</div>;
}

function Layout({ children }) {
  return (
    <div className="layout">
      <Header />
      <main>{children}</main>
      <Footer />
    </div>
  );
}

// 사용
<Layout>
  <Container>
    <h1>페이지 제목</h1>
    <p>내용</p>
  </Container>
</Layout>

Props는 읽기 전용​

Props는 절대 수정할 수 없습니다.

function User({ name }) {
  // ❌ 잘못된 예 - Props 수정
  name = name.toUpperCase();

  // ✅ 올바른 예 - 새로운 변수 사용
  const upperName = name.toUpperCase();

  return <h1>{upperName}</h1>;
}

Spread 연산자로 Props 전달​

const user = {
  name: "홍길동",
  email: "hong@example.com",
  age: 25
};

// 개별 전달
<UserCard name={user.name} email={user.email} age={user.age} />

// Spread 연산자 사용
<UserCard {...user} />

일부 Props 재정의​

const props = {
  name: "홍길동",
  age: 25,
  role: "개발자"
};

// role만 재정의
<UserCard {...props} role="시니어 개발자" />

조건부 Props​

function Button({ isPrimary, children }) {
  return (
    <button className={isPrimary ? 'btn-primary' : 'btn-secondary'}>
      {children}
    </button>
  );
}

// 또는
function Button({ variant = 'primary', children }) {
  return (
    <button className={`btn-${variant}`}>
      {children}
    </button>
  );
}

Props Drilling​

Props를 여러 단계의 컴포넌트를 통해 전달하는 것을 Props Drilling이라고 합니다.

function App() {
  const user = { name: "홍길동" };
  return <Parent user={user} />;
}

function Parent({ user }) {
  return <Child user={user} />;
}

function Child({ user }) {
  return <GrandChild user={user} />;
}

function GrandChild({ user }) {
  return <div>{user.name}</div>;
}

주의: Props Drilling이 깊어지면 Context API나 상태 관리 라이브러리 사용을 고려하세요.

실전 예제​

Product 카드​

function ProductCard({
  name,
  price,
  image,
  rating,
  onAddToCart,
  inStock = true
}) {
  return (
    <div className="product-card">
      <img src={image} alt={name} />
      <h3>{name}</h3>
      <div className="price">${price}</div>
      <div className="rating">⭐ {rating}</div>
      <button
        onClick={() => onAddToCart(name)}
        disabled={!inStock}
      >
        {inStock ? '장바구니에 담기' : '품절'}
      </button>
    </div>
  );
}

// 사용
<ProductCard
  name="무선 이어폰"
  price={99.99}
  image="earphone.jpg"
  rating={4.5}
  onAddToCart={(name) => console.log(`${name} 추가됨`)}
  inStock={true}
/>

Alert 컴포넌트​

function Alert({ type = 'info', message, onClose }) {
  const icons = {
    success: '✓',
    error: '✗',
    warning: '⚠',
    info: 'ℹ'
  };

  return (
    <div className={`alert alert-${type}`}>
      <span className="icon">{icons[type]}</span>
      <span className="message">{message}</span>
      {onClose && (
        <button onClick={onClose} className="close">×</button>
      )}
    </div>
  );
}

// 사용
<Alert type="success" message="저장되었습니다!" />
<Alert
  type="error"
  message="오류가 발생했습니다"
  onClose={() => console.log('닫기')}
/>

TypeScript와 Props​

interface UserCardProps {
  name: string;
  email: string;
  age?: number;  // 선택적 prop
  role: 'admin' | 'user';  // 유니온 타입
}

function UserCard({ name, email, age, role }: UserCardProps) {
  return (
    <div>
      <h3>{name}</h3>
      <p>{email}</p>
      {age && <p>나이: {age}</p>}
      <span>{role}</span>
    </div>
  );
}
