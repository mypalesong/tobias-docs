JSX 문법

JSX(JavaScript XML)는 JavaScript를 확장한 문법으로, React에서 UI를 작성하는 데 사용됩니다.

JSX란?​

JSX는 JavaScript 코드 안에서 HTML과 유사한 마크업을 작성할 수 있게 해주는 문법입니다.

const element = <h1>안녕하세요, React!</h1>;

JSX vs HTML​

유사점​

  • HTML과 비슷한 태그 구조
  • 중첩 가능
  • 속성 사용 가능

차이점​

1. className vs class​

// JSX
<div className="container">내용</div>

// HTML
<div class="container">내용</div>

2. camelCase 속성명​

// JSX
<button onClick={handleClick}>클릭</button>
<label htmlFor="input">라벨</label>

// HTML
<button onclick="handleClick()">클릭</button>
<label for="input">라벨</label>

3. 자동 닫기 태그​

// JSX - 반드시 닫아야 함
<img src="image.jpg" />
<input type="text" />
<br />

// HTML - 선택적
<img src="image.jpg">
<input type="text">
<br>

JSX에 표현식 포함하기​

중괄호 {}를 사용하여 JavaScript 표현식을 포함할 수 있습니다.

변수 사용​

const name = "홍길동";
const element = <h1>안녕하세요, {name}님!</h1>;

함수 호출​

function formatName(user) {
  return user.firstName + ' ' + user.lastName;
}

const user = {
  firstName: '길동',
  lastName: '홍'
};

const element = (
  <h1>안녕하세요, {formatName(user)}님!</h1>
);

연산​

const count = 5;
const element = <p>총 {count * 2}개의 아이템이 있습니다.</p>;

조건부 렌더링​

const isLoggedIn = true;
const element = (
  <div>
    {isLoggedIn ? <p>환영합니다!</p> : <p>로그인해주세요.</p>}
  </div>
);

JSX 속성​

문자열 속성​

<img src="profile.jpg" alt="프로필" />

표현식 속성​

const imageUrl = "profile.jpg";
<img src={imageUrl} alt="프로필" />

스타일 속성​

// 객체로 전달
const divStyle = {
  color: 'blue',
  backgroundColor: 'lightgray',
  padding: '10px'
};

<div style={divStyle}>스타일 적용</div>

// 인라인으로 전달
<div style={{ color: 'red', fontSize: '20px' }}>
  인라인 스타일
</div>

JSX 자식 요소​

중첩 요소​

const element = (
  <div>
    <h1>제목</h1>
    <p>내용</p>
    <ul>
      <li>항목 1</li>
      <li>항목 2</li>
    </ul>
  </div>
);

자식이 없는 경우​

// 둘 다 동일
<div></div>
<div />

자식으로 표현식​

const items = ['사과', '바나나', '오렌지'];

const element = (
  <ul>
    {items.map((item, index) => (
      <li key={index}>{item}</li>
    ))}
  </ul>
);

JSX는 객체를 표현합니다​

Babel은 JSX를 React.createElement() 호출로 컴파일합니다.

// 이 두 예제는 동일합니다
const element = (
  <h1 className="greeting">
    안녕하세요!
  </h1>
);

const element = React.createElement(
  'h1',
  {className: 'greeting'},
  '안녕하세요!'
);

JSX 주의사항​

1. 하나의 부모 요소​

JSX는 반드시 하나의 부모 요소를 가져야 합니다.

// ❌ 잘못된 예
return (
  <h1>제목</h1>
  <p>내용</p>
);

// ✅ 올바른 예
return (
  <div>
    <h1>제목</h1>
    <p>내용</p>
  </div>
);

// ✅ Fragment 사용
return (
  <>
    <h1>제목</h1>
    <p>내용</p>
  </>
);

2. 주석​

return (
  <div>
    {/* 이것은 JSX 주석입니다 */}
    <h1>제목</h1>
    {/*
      여러 줄
      주석
    */}
  </div>
);

3. Boolean 속성​

// 이 두 예제는 동일
<input type="checkbox" checked={true} />
<input type="checkbox" checked />

// false인 경우
<input type="checkbox" checked={false} />

4. undefined, null, boolean은 렌더링되지 않음​

<div>
  {undefined}  {/* 렌더링 안 됨 */}
  {null}       {/* 렌더링 안 됨 */}
  {true}       {/* 렌더링 안 됨 */}
  {false}      {/* 렌더링 안 됨 */}
</div>

실전 예제​

사용자 카드 컴포넌트​

function UserCard({ user }) {
  const { name, email, avatar, isOnline } = user;

  return (
    <div className="user-card">
      <img src={avatar} alt={`${name}의 프로필`} />
      <div className="user-info">
        <h3>{name}</h3>
        <p>{email}</p>
        <span className={isOnline ? 'online' : 'offline'}>
          {isOnline ? '온라인' : '오프라인'}
        </span>
      </div>
    </div>
  );
}

// 사용
const user = {
  name: '홍길동',
  email: 'hong@example.com',
  avatar: 'avatar.jpg',
  isOnline: true
};

<UserCard user={user} />
