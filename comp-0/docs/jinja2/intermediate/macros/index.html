매크로

재사용 가능한 템플릿 함수를 만들어봅시다!

🎯 매크로란?​

매크로는 템플릿 내에서 재사용 가능한 함수입니다. Python의 함수와 비슷하게 동작합니다.

핵심 개념​

  • 정의: {% macro name(params) %}로 정의
  • 호출: {{ name(args) }}로 호출
  • 스코프: 매크로 내부는 독립적인 스코프
  • 재사용: 여러 템플릿에서 import하여 사용

매크로 정의​

{% macro greeting(name) %}
    안녕하세요, {{ name }}님!
{% endmacro %}

{{ greeting('홍길동') }}

📦 caller 기능​

기본 caller​

{% macro dialog(title) %}
<div class="dialog">
    <div class="dialog-header">
        <h3>{{ title }}</h3>
    </div>
    <div class="dialog-body">
        {{ caller() }}
    </div>
</div>
{% endmacro %}

{% call dialog('알림') %}
    <p>이것은 대화상자 내용입니다.</p>
    <p>여러 줄을 포함할 수 있습니다.</p>
{% endcall %}

출력:

<div class="dialog">
    <div class="dialog-header">
        <h3>알림</h3>
    </div>
    <div class="dialog-body">
        <p>이것은 대화상자 내용입니다.</p>
        <p>여러 줄을 포함할 수 있습니다.</p>
    </div>
</div>

caller에 인자 전달​

{% macro list_users(users) %}
<ul class="user-list">
{% for user in users %}
    <li>{{ caller(user) }}</li>
{% endfor %}
</ul>
{% endmacro %}

{% call(user) list_users(users) %}
    <strong>{{ user.name }}</strong> ({{ user.email }})
{% endcall %}

🎨 실전 예제​

UI 컴포넌트 라이브러리​

from jinja2 import Environment, FileSystemLoader

env = Environment(loader=FileSystemLoader('templates'))

# components.html
components = """
{# 버튼 컴포넌트 #}
{% macro button(text, type='primary', size='md', disabled=false) %}
<button class="btn btn-{{ type }} btn-{{ size }}"
        {% if disabled %}disabled{% endif %}>
    {{ text }}
</button>
{% endmacro %}

{# 알림 컴포넌트 #}
{% macro alert(message, type='info', dismissible=false) %}
<div class="alert alert-{{ type }}{% if dismissible %} alert-dismissible{% endif %}">
    {{ message }}
    {% if dismissible %}
        <button type="button" class="close" data-dismiss="alert">
            <span>&times;</span>
        </button>
    {% endif %}
</div>
{% endmacro %}

{# 카드 컴포넌트 #}
{% macro card(title, image=none, footer=none) %}
<div class="card">
    {% if image %}
    <img class="card-img-top" src="{{ image }}" alt="{{ title }}">
    {% endif %}

    <div class="card-body">
        <h5 class="card-title">{{ title }}</h5>
        {{ caller() if caller }}
    </div>

    {% if footer %}
    <div class="card-footer">
        {{ footer }}
    </div>
    {% endif %}
</div>
{% endmacro %}

{# 배지 컴포넌트 #}
{% macro badge(text, color='secondary') %}
<span class="badge badge-{{ color }}">{{ text }}</span>
{% endmacro %}

{# 입력 필드 컴포넌트 #}
{% macro input_field(name, label, type='text', value='', required=false, placeholder='') %}
<div class="form-group">
    <label for="{{ name }}">
        {{ label }}
        {% if required %}
            <span class="text-danger">*</span>
        {% endif %}
    </label>
    <input type="{{ type }}"
           class="form-control"
           id="{{ name }}"
           name="{{ name }}"
           value="{{ value }}"
           {% if placeholder %}placeholder="{{ placeholder }}"{% endif %}
           {% if required %}required{% endif %}>
</div>
{% endmacro %}

{# 페이지네이션 컴포넌트 #}
{% macro pagination(current_page, total_pages, base_url='/') %}
<nav aria-label="Page navigation">
    <ul class="pagination">
        {# 이전 버튼 #}
        <li class="page-item {% if current_page == 1 %}disabled{% endif %}">
            <a class="page-link" href="{{ base_url }}?page={{ current_page - 1 }}">
                이전
            </a>
        </li>

        {# 페이지 번호 #}
        {% for page in range(1, total_pages + 1) %}
            {% if page == current_page %}
                <li class="page-item active">
                    <span class="page-link">{{ page }}</span>
                </li>
            {% elif (page - current_page)|abs <= 2 or page == 1 or page == total_pages %}
                <li class="page-item">
                    <a class="page-link" href="{{ base_url }}?page={{ page }}">
                        {{ page }}
                    </a>
                </li>
            {% elif (page - current_page)|abs == 3 %}
                <li class="page-item disabled">
                    <span class="page-link">...</span>
                </li>
            {% endif %}
        {% endfor %}

        {# 다음 버튼 #}
        <li class="page-item {% if current_page == total_pages %}disabled{% endif %}">
            <a class="page-link" href="{{ base_url }}?page={{ current_page + 1 }}">
                다음
            </a>
        </li>
    </ul>
</nav>
{% endmacro %}

{# 테이블 컴포넌트 #}
{% macro table(headers, rows, striped=true, hover=true) %}
<table class="table
              {% if striped %}table-striped{% endif %}
              {% if hover %}table-hover{% endif %}">
    <thead>
        <tr>
            {% for header in headers %}
                <th>{{ header }}</th>
            {% endfor %}
        </tr>
    </thead>
    <tbody>
        {% for row in rows %}
            <tr>
                {{ caller(row) if caller else '' }}
            </tr>
        {% endfor %}
    </tbody>
</table>
{% endmacro %}
"""

# 사용 예제
usage = """

            {{ alert('경고 메시지입니다.', type='warning') }}

        {# 카드 #}
        <section>
            <h2>카드</h2>
            {% call card('상품 제목', image='/static/product.jpg', footer='₩29,000') %}
                <p>이것은 상품 설명입니다.</p>
                {{ button('구매하기', type='primary') }}
            {% endcall %}
        </section>

        {# 폼 #}
        <section>
            <h2>입력 폼</h2>
            <form>
                {{ input_field('username', '사용자명', required=true, placeholder='사용자명 입력') }}
                {{ input_field('email', '이메일', type='email', required=true) }}
                {{ input_field('password', '비밀번호', type='password', required=true) }}
                {{ button('가입하기', type='success') }}
            </form>
        </section>

        {# 페이지네이션 #}
        <section>
            <h2>페이지네이션</h2>

        </section>

        {# 테이블 #}
        <section>
            <h2>테이블</h2>
            {% call(user) table(['ID', '이름', '이메일', '상태'], users) %}
                <td>{{ user.id }}</td>
                <td>{{ user.name }}</td>
                <td>{{ user.email }}</td>
                <td>{{ badge('활성', color='success') if user.is_active else badge('비활성', color='secondary') }}</td>
            {% endcall %}
        </section>
    </div>
</body>
</html>
"""

# forms.html
forms = """
{% macro render_field(field) %}
<div class="form-group {% if field.errors %}has-error{% endif %}">
    <label for="{{ field.id }}">
        {{ field.label }}
        {% if field.required %}
            <span class="required">*</span>
        {% endif %}
    </label>

    {% if field.type == 'textarea' %}
        <textarea class="form-control"
                  id="{{ field.id }}"
                  name="{{ field.name }}"
                  {% if field.required %}required{% endif %}>{{ field.value }}</textarea>
    {% elif field.type == 'select' %}
        <select class="form-control"
                id="{{ field.id }}"
                name="{{ field.name }}"
                {% if field.required %}required{% endif %}>
            {% for option in field.options %}
                <option value="{{ option.value }}"
                        {% if option.value == field.value %}selected{% endif %}>
                    {{ option.label }}
                </option>
            {% endfor %}
        </select>
    {% else %}
        <input type="{{ field.type }}"
               class="form-control"
               id="{{ field.id }}"
               name="{{ field.name }}"
               value="{{ field.value }}"
               {% if field.placeholder %}placeholder="{{ field.placeholder }}"{% endif %}
               {% if field.required %}required{% endif %}>
    {% endif %}

    {% if field.help_text %}
        <small class="form-text text-muted">{{ field.help_text }}</small>
    {% endif %}

    {% if field.errors %}
        <div class="invalid-feedback">
            {% for error in field.errors %}
                <div>{{ error }}</div>
            {% endfor %}
        </div>
    {% endif %}
</div>
{% endmacro %}

{% macro render_form(form, action='', method='POST') %}
<form action="{{ action }}" method="{{ method }}">
    {% for field in form.fields %}
        {{ render_field(field) }}
    {% endfor %}

    <div class="form-actions">
        {{ caller() if caller else '<button type="submit" class="btn btn-primary">제출</button>' }}
    </div>
</form>
{% endmacro %}
"""

🔍 매크로 변수​

varargs - 위치 인자​

{% macro sum(values) %}
    {% set total = 0 %}
    {% for value in values %}
        {% set total = total + value %}
    {% endfor %}
    {{ total }}
{% endmacro %}

{{ sum([1, 2, 3, 4, 5]) }}  {# 15 #}

kwargs - 키워드 인자​

{% macro render_attrs(attrs) %}
    {% for key, value in attrs.items() %}
        {{ key }}="{{ value }}"
    {% endfor %}
{% endmacro %}

<div {{ render_attrs({'class': 'card', 'id': 'main', 'data-id': '123'}) }}>
</div>

    caller 존재: {{ caller is defined }}
{% endmacro %}
