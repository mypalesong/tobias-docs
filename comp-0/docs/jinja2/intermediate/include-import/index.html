Include와 Import

템플릿을 분할하고 재사용해봅시다!

🎯 개요​

템플릿을 작은 조각으로 나누고 조합하는 방법을 배웁니다.

핵심 개념​

  • include: 템플릿 조각을 현재 위치에 삽입
  • import: 매크로를 가져오기
  • from...import: 특정 매크로만 선택적으로 가져오기

📐 include​

기본 사용법​

{# header.html #}
<header>
    <h1>내 웹사이트</h1>
    <nav>
        <a href="/">홈</a>
        <a href="/about">소개</a>
    </nav>
</header>
{# page.html #}
<!DOCTYPE html>
<html>
<head>
    <title>페이지</title>
</head>
<body>
    {% include 'header.html' %}

    <main>
        페이지 내용
    </main>

    {% include 'footer.html' %}
</body>
</html>

변수 전달​

{# greeting.html #}
<p>안녕하세요, {{ name }}님!</p>
{# main.html #}
{% set name = '홍길동' %}
{% include 'greeting.html' %}
{# name 변수가 자동으로 전달됨 #}

명시적 변수 전달​

{% include 'user_card.html' with context %}
{# 모든 컨텍스트 변수 전달 #}

{% include 'user_card.html' without context %}
{# 변수 전달 안 함 #}

조건부 include​

{% if user.is_admin %}
    {% include 'admin_panel.html' %}
{% endif %}

{# 또는 #}
{% include 'admin_panel.html' if user.is_admin %}

ignore missing​

{# 파일이 없어도 에러 발생 안 함 #}
{% include 'optional_sidebar.html' ignore missing %}

📦 import​

기본 import​

{# macros.html #}
{% macro button(text, type='primary') %}
    <button class="btn btn-{{ type }}">{{ text }}</button>
{% endmacro %}

{% macro alert(message, level='info') %}
    <div class="alert alert-{{ level }}">{{ message }}</div>
{% endmacro %}
{# page.html #}
{% import 'macros.html' as ui %}

{{ ui.button('클릭하세요') }}
{{ ui.alert('알림 메시지') }}

선택적 import (from...import)​

{% from 'macros.html' import button, alert %}

{{ button('클릭') }}
{{ alert('메시지') }}

별칭 사용​

{% from 'macros.html' import button as btn, alert as msg %}

{{ btn('클릭') }}
{{ msg('메시지') }}

with context​

{# 현재 컨텍스트 변수 접근 가능 #}
{% from 'forms.html' import render_field with context %}

{# 컨텍스트 없이 import #}
{% from 'forms.html' import render_field without context %}

                    {{ item.text }}
                </a>
            {% endfor %}
        </nav>

            <div class="user-menu">
                <span>{{ user.name }}</span>

            </div>

                <a href="/login">로그인</a>

"""

    {% block extra_css %}{% endblock %}
</head>
<body>
    {% include '_header.html' %}

    <div class="container">
        <div class="row">
            <div class="col-md-8">
                <main>

컴포넌트 라이브러리​

# components/buttons.html
buttons = """
{% macro primary(text, href='#') %}
    <a href="{{ href }}" class="btn btn-primary">{{ text }}</a>
{% endmacro %}

{% macro secondary(text, href='#') %}
    <a href="{{ href }}" class="btn btn-secondary">{{ text }}</a>
{% endmacro %}

{% macro danger(text, href='#', confirm_message='') %}
    <a href="{{ href }}"
       class="btn btn-danger"
       {% if confirm_message %}
       onclick="return confirm('{{ confirm_message }}')"
       {% endif %}>
        {{ text }}
    </a>
{% endmacro %}
"""

# components/cards.html
cards = """
{% macro product_card(product) %}
<div class="card product-card">
    {% if product.image %}
        <img src="{{ product.image }}" alt="{{ product.name }}">
    {% endif %}
    <div class="card-body">
        <h5>{{ product.name }}</h5>
        <p class="price">{{ product.price|currency }}</p>
        {% if product.discount %}
            <span class="badge badge-sale">{{ product.discount }}% 할인</span>
        {% endif %}
        {{ caller() if caller }}
    </div>
</div>
{% endmacro %}

{% macro user_card(user) %}
<div class="card user-card">
    <img src="{{ user.avatar|default('/static/default-avatar.png') }}"
         alt="{{ user.name }}"
         class="avatar">
    <div class="card-body">
        <h5>{{ user.name }}</h5>
        <p>{{ user.bio|truncate(100) }}</p>
        {% from 'components/buttons.html' import primary %}
        {{ primary('프로필 보기', href='/user/' ~ user.id) }}
    </div>
</div>
{% endmacro %}
"""

# components/forms.html
forms = """
{% macro text_input(name, label, value='', required=false, placeholder='') %}
<div class="form-group">
    <label for="{{ name }}">
        {{ label }}
        {% if required %}<span class="required">*</span>{% endif %}
    </label>
    <input type="text"
           id="{{ name }}"
           name="{{ name }}"
           class="form-control"
           value="{{ value }}"
           {% if placeholder %}placeholder="{{ placeholder }}"{% endif %}
           {% if required %}required{% endif %}>
</div>
{% endmacro %}

{% macro email_input(name, label, value='', required=false) %}
<div class="form-group">
    <label for="{{ name }}">
        {{ label }}
        {% if required %}<span class="required">*</span>{% endif %}
    </label>
    <input type="email"
           id="{{ name }}"
           name="{{ name }}"
           class="form-control"
           value="{{ value }}"
           {% if required %}required{% endif %}>
</div>
{% endmacro %}

{% macro select_input(name, label, options, value='', required=false) %}
<div class="form-group">
    <label for="{{ name }}">
        {{ label }}
        {% if required %}<span class="required">*</span>{% endif %}
    </label>
    <select id="{{ name }}"
            name="{{ name }}"
            class="form-control"
            {% if required %}required{% endif %}>
        {% for option in options %}
            <option value="{{ option.value }}"
                    {% if option.value == value %}selected{% endif %}>
                {{ option.label }}
            </option>
        {% endfor %}
    </select>
</div>
{% endmacro %}
"""

# 사용 예제
usage = """
{% from 'components/buttons.html' import primary, danger %}
{% from 'components/cards.html' import product_card, user_card %}
{% from 'components/forms.html' import text_input, email_input, select_input %}

<div class="page">
    {# 상품 카드 #}
    <div class="products">
        {% for product in products %}
            {% call product_card(product) %}
                {{ primary('장바구니', href='/cart/add/' ~ product.id) }}
            {% endcall %}
        {% endfor %}

    {# 사용자 폼 #}

        {{ text_input('name', '이름', value=user.name, required=true) }}
        {{ email_input('email', '이메일', value=user.email, required=true) }}
        {{ select_input('country', '국가', options=countries, value=user.country) }}

        {{ primary('저장') }}
        {{ danger('계정 삭제', href='/account/delete', confirm_message='정말 삭제하시겠습니까?') }}
    </form>
</div>
"""

📊 include vs import vs extends​

비교표​

기능includeimportextends
용도템플릿 조각 삽입매크로 가져오기레이아웃 상속
변수 접근부모 컨텍스트독립적부모 컨텍스트
여러 번 사용가능가능한 번만
위치어디든어디든첫 줄만

사용 시기​

{# include: HTML 조각 재사용 #}
{% include '_navigation.html' %}
{% include '_footer.html' %}

{# import: 매크로/함수 재사용 #}
{% from 'ui.html' import button, card %}

{# extends: 레이아웃 상속 #}
{% extends 'base.html' %}

📋 모범 사례​

✅ 권장사항​

{# 부분 템플릿은 _ 접두사 사용 #}
{% include '_header.html' %}
{% include '_sidebar.html' %}

{# 명확한 디렉토리 구조 #}
{% from 'components/buttons.html' import button %}
{% from 'components/forms.html' import input %}

{# 필요한 것만 import #}
{% from 'ui.html' import button, alert %}  {# ✅ #}
{% import 'ui.html' as ui %}  {# 전체 import는 필요시만 #}

{# ignore missing 활용 #}
{% include 'custom_header.html' ignore missing %}

❌ 피해야 할 것​

{# 너무 많은 include (성능 저하) #}
{% include 'a.html' %}
{% include 'b.html' %}
... {# 수십 개 #}

{# 순환 참조 #}
{# a.html includes b.html, b.html includes a.html #}

{# 모호한 파일명 #}
{% include 'template1.html' %}
{% include 'temp.html' %}
