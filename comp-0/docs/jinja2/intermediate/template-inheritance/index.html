템플릿 상속

템플릿 상속으로 재사용 가능한 레이아웃을 만들어봅시다!

🎯 템플릿 상속이란?​

템플릿 상속을 사용하면 기본 레이아웃을 정의하고, 자식 템플릿에서 특정 부분만 재정의할 수 있습니다.

핵심 개념​

  • 베이스 템플릿: 기본 구조와 블록 정의
  • 자식 템플릿: 베이스를 확장하고 블록 재정의
  • 블록: 재정의 가능한 영역

📐 기본 구조​

베이스 템플릿​

{# templates/base.html #}
<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <title>{% block title %}기본 제목{% endblock %}</title>
    {% block styles %}
    <link rel="stylesheet" href="/static/main.css">
    {% endblock %}
</head>
<body>
    <header>
        {% block header %}
        <h1>내 웹사이트</h1>
        {% endblock %}
    </header>

    <main>
        {% block content %}
        {# 자식 템플릿에서 반드시 정의해야 함 #}
        {% endblock %}
    </main>

    <footer>
        {% block footer %}
        <p>&copy; 2024 내 웹사이트</p>
        {% endblock %}
    </footer>

    {% block scripts %}
    <script src="/static/main.js"></script>
    {% endblock %}
</body>
</html>

자식 템플릿​

{# templates/home.html #}
{% extends "base.html" %}

{% block title %}홈페이지{% endblock %}

{% block content %}
    <h2>환영합니다!</h2>
    <p>이것은 홈페이지입니다.</p>
{% endblock %}

🔧 extends 지시자​

기본 사용법​

{% extends "base.html" %}
from jinja2 import Environment, FileSystemLoader

env = Environment(loader=FileSystemLoader('templates'))
template = env.get_template('home.html')
html = template.render()

동적 상속​

{% extends layout_template %}
template = env.get_template('page.html')
html = template.render(layout_template='base.html')

조건부 상속​

{% extends 'mobile.html' if is_mobile else 'desktop.html' %}

📦 block 지시자​

블록 정의​

{# 베이스 템플릿 #}
{% block header %}
    기본 헤더 내용
{% endblock %}

블록 재정의​

{# 자식 템플릿 #}
{% block header %}
    새로운 헤더 내용
{% endblock %}

블록 이름 지정​

{% block navigation %}
    ...
{% endblock navigation %}

{# 또는 #}
{% block navigation %}
    ...
{% endblock %}

🔗 super() 함수​

부모 블록의 내용을 유지하면서 추가할 수 있습니다.

기본 사용​

{# base.html #}
{% block styles %}
<link rel="stylesheet" href="/static/base.css">
{% endblock %}
{# child.html #}
{% extends "base.html" %}

{% block styles %}
    {{ super() }}
    <link rel="stylesheet" href="/static/child.css">
{% endblock %}

출력:

<link rel="stylesheet" href="/static/base.css">
<link rel="stylesheet" href="/static/child.css">

부모 내용 + 추가 내용​

{% block title %}
    {{ super() }} - 추가 제목
{% endblock %}

🎨 실전 예제​

블로그 레이아웃​

from jinja2 import Environment, FileSystemLoader

env = Environment(loader=FileSystemLoader('templates'))

# base.html
base = """
<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>{% block title %}내 블로그{% endblock %}</title>

    {% block meta %}
    <meta name="description" content="개인 블로그">
    {% endblock %}

    {% block styles %}
    <link rel="stylesheet" href="/static/bootstrap.min.css">
    <link rel="stylesheet" href="/static/main.css">
    {% endblock %}
</head>
<body>
    <nav class="navbar">
        {% block navbar %}
        <div class="container">
            <a href="/">홈</a>
            <a href="/blog">블로그</a>
            <a href="/about">소개</a>
        </div>
        {% endblock %}
    </nav>

    <div class="container">
        {% block breadcrumb %}{% endblock %}

        <main>
            {% block content %}{% endblock %}
        </main>

        {% block sidebar %}
        <aside>
            <h3>카테고리</h3>
            <ul>
                <li><a href="/category/python">Python</a></li>
                <li><a href="/category/web">Web</a></li>
            </ul>
        </aside>
        {% endblock %}
    </div>

    <footer>
        {% block footer %}
        <p>&copy; 2024 내 블로그. All rights reserved.</p>
        {% endblock %}
    </footer>

    {% block scripts %}
    <script src="/static/jquery.min.js"></script>
    <script src="/static/bootstrap.min.js"></script>
    {% endblock %}
</body>
</html>
"""

# blog_post.html
blog_post = """
{% extends "base.html" %}

{% block title %}{{ post.title }} - {{ super() }}{% endblock %}

{% block meta %}
    {{ super() }}
    <meta name="author" content="{{ post.author }}">
    <meta property="og:title" content="{{ post.title }}">
{% endblock %}

{% block breadcrumb %}
<nav aria-label="breadcrumb">
    <ol class="breadcrumb">
        <li><a href="/">홈</a></li>
        <li><a href="/blog">블로그</a></li>
        <li class="active">{{ post.title }}</li>
    </ol>
</nav>
{% endblock %}

{% block content %}
<article class="blog-post">
    <header>
        <h1>{{ post.title }}</h1>
        <p class="meta">
            <span class="author">{{ post.author }}</span> |
            <time datetime="{{ post.created_at }}">
                {{ post.created_at|datetimeformat('%Y년 %m월 %d일') }}
            </time>
        </p>
    </header>

    <div class="content">
        {{ post.content|safe }}
    </div>

    <footer>
        <div class="tags">
            {% for tag in post.tags %}
                <a href="/tag/{{ tag|slugify }}" class="tag">
                    #{{ tag }}
                </a>
            {% endfor %}
        </div>

        <div class="share">
            <a href="https://twitter.com/share?url={{ request.url }}">
                트위터 공유
            </a>
        </div>
    </footer>
</article>

<section class="comments">
    {% block comments %}
    <h2>댓글 {{ post.comments|length }}개</h2>
    {% for comment in post.comments %}
        <div class="comment">
            <strong>{{ comment.author }}</strong>
            <p>{{ comment.content }}</p>
        </div>
    {% endfor %}
    {% endblock %}
</section>
{% endblock %}

{% block scripts %}
    {{ super() }}
    <script src="/static/syntax-highlighter.js"></script>
{% endblock %}
"""

# 렌더링
with open('templates/base.html', 'w', encoding='utf-8') as f:
    f.write(base)

with open('templates/blog_post.html', 'w', encoding='utf-8') as f:
    f.write(blog_post)

template = env.get_template('blog_post.html')
html = template.render(
    post={
        'title': 'Jinja2 템플릿 상속 가이드',
        'author': '홍길동',
        'created_at': datetime.now(),
        'content': '<p>템플릿 상속은...</p>',
        'tags': ['Python', 'Jinja2', 'Template'],
        'comments': [
            {'author': '김철수', 'content': '좋은 글입니다!'},
            {'author': '이영희', 'content': '도움이 되었어요.'}
        ]
    },
    request={'url': 'https://example.com/post/123'}
)

다단계 상속​

# base.html - 최상위
base = """
<!DOCTYPE html>
<html>
<head>
    <title>{% block title %}{% endblock %}</title>
    {% block styles %}{% endblock %}
</head>
<body>
    {% block body %}{% endblock %}
</body>
</html>
"""

# layout.html - 중간 레이아웃
layout = """
{% extends "base.html" %}

{% block styles %}
    {{ super() }}
    <link rel="stylesheet" href="/static/layout.css">
{% endblock %}

{% block body %}
<div class="container">
    <header>
        {% block header %}
        <h1>사이트 제목</h1>
        {% endblock %}
    </header>

    <main>
        {% block content %}{% endblock %}
    </main>

    <footer>
        {% block footer %}
        <p>&copy; 2024</p>
        {% endblock %}
    </footer>
</div>
{% endblock %}
"""

# page.html - 최종 페이지
page = """
{% extends "layout.html" %}

{% block title %}페이지 제목{% endblock %}

{% block content %}
<h2>페이지 내용</h2>
<p>이것은 3단계 상속 예제입니다.</p>
{% endblock %}
"""

📊 블록 중첩​

중첩된 블록​

{# base.html #}
{% block outer %}
    <div class="outer">
        {% block inner %}
            기본 내용
        {% endblock %}
    </div>
{% endblock %}
{# child.html #}
{% extends "base.html" %}

{% block inner %}
    새로운 내용
{% endblock %}

출력:

<div class="outer">
    새로운 내용
</div>

전체 블록 재정의​

{% extends "base.html" %}

{% block outer %}
    <div class="completely-new">
        완전히 새로운 구조
    </div>
{% endblock %}

🎯 블록 스코프​

블록 내 변수​

{# base.html #}
{% block content %}
    {% set default_message = "기본 메시지" %}
    {{ default_message }}
{% endblock %}
{# child.html #}
{% extends "base.html" %}

{% block content %}
    {% set custom_message = "커스텀 메시지" %}
    {{ custom_message }}
    {# default_message는 접근 불가 #}
{% endblock %}

블록 외부 변수 접근​

{# base.html #}
{% set site_name = "내 사이트" %}

{% block header %}
    <h1>{{ site_name }}</h1>  {# 접근 가능 #}
{% endblock %}

📊 실습 과제​

과제 1: 쇼핑몰 레이아웃​

# 다음 구조의 템플릿을 만드세요:
# - base.html: 기본 레이아웃
# - product_list.html: 상품 목록 페이지
# - product_detail.html: 상품 상세 페이지
# - cart.html: 장바구니 페이지

# 요구사항:
# - 모든 페이지는 동일한 헤더/푸터
# - 상품 페이지는 사이드바 포함
# - 장바구니는 사이드바 없음

과제 2: 블로그 시스템​

# 3단계 상속 구조 만들기:
# - base.html: 기본 HTML 구조
# - blog_base.html: 블로그 레이아웃 (내비게이션, 사이드바)
# - post.html: 블로그 포스트
# - archive.html: 아카이브 페이지

# 요구사항:
# - super() 사용하여 스타일 누적
# - 블록 단위로 적절히 분리
# - 선택적 블록 활용
