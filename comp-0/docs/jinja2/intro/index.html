Jinja2 완벽 가이드

Python에서 가장 강력하고 인기 있는 템플릿 엔진 Jinja2를 마스터해봅시다!

🎨 Jinja2란?​

Jinja2는 Python으로 작성된 현대적이고 디자이너 친화적인 템플릿 엔진입니다.

주요 특징​

  • 🚀 빠른 성능: 컴파일된 바이트코드로 실행
  • 🔒 안전함: 자동 HTML 이스케이핑으로 XSS 방지
  • 🎯 강력함: 템플릿 상속, 매크로, 필터 등 풍부한 기능
  • 🧩 확장 가능: 커스텀 필터, 테스트, 함수 추가 가능
  • 📝 직관적 문법: Python과 유사한 문법으로 쉬운 학습
  • 🌐 국제화 지원: i18n 내장 지원

🎯 누가 사용하나요?​

  • Flask - 기본 템플릿 엔진
  • Django (선택사항) - Jinja2 지원
  • Ansible - 설정 파일 템플릿
  • SaltStack - 시스템 설정 관리
  • Pelican - 정적 사이트 생성기

💡 Jinja2를 사용하는 이유​

# ❌ 문자열 연결 (나쁜 방법)
html = "<h1>Hello, " + name + "!</h1>"

# ✅ Jinja2 템플릿 (좋은 방법)
template = Template("<h1>Hello, {{ name }}!</h1>")
html = template.render(name=name)

장점​

  1. 코드와 프레젠테이션 분리: 로직과 UI를 명확하게 구분
  2. 재사용성: 템플릿 상속과 인클루드로 중복 제거
  3. 유지보수성: 템플릿만 수정하면 전체 사이트 변경 가능
  4. 보안: 자동 이스케이핑으로 안전한 출력
  5. 팀워크: 디자이너와 개발자가 독립적으로 작업 가능

🚀 빠른 시작​

설치​

pip install jinja2

첫 번째 템플릿​

from jinja2 import Template

# 템플릿 정의
template = Template("Hello, {{ name }}!")

# 렌더링
output = template.render(name="World")
print(output)  # Hello, World!

파일 기반 템플릿​

from jinja2 import Environment, FileSystemLoader

# 환경 설정
env = Environment(loader=FileSystemLoader('templates'))

# 템플릿 로드
template = env.get_template('index.html')

# 렌더링
html = template.render(
    title="홈페이지",
    user={"name": "홍길동", "age": 30}
)

📖 기본 문법 미리보기​

변수 출력​

{{ variable }}
{{ user.name }}
{{ items[0] }}

필터 적용​

{{ name|upper }}
{{ price|round(2) }}
{{ text|truncate(100) }}

제어 구조​

{% if user.is_active %}
    환영합니다!
{% endif %}

{% for item in items %}
    - {{ item }}
{% endfor %}

템플릿 상속​

{# base.html #}
<!DOCTYPE html>
<html>
<head>
    <title>{% block title %}{% endblock %}</title>
</head>
<body>
    {% block content %}{% endblock %}
</body>
</html>

{# page.html #}
{% extends "base.html" %}

{% block title %}페이지 제목{% endblock %}

{% block content %}
    <h1>페이지 내용</h1>
{% endblock %}
