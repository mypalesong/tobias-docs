설치 및 환경 설정

Jinja2를 설치하고 첫 번째 템플릿을 만들어봅시다!

📦 설치​

pip를 사용한 설치​

pip install jinja2

특정 버전 설치​

# 최신 안정 버전
pip install jinja2==3.1.2

# 최소 버전 지정
pip install "jinja2>=3.0.0"

가상 환경에 설치 (권장)​

# 가상 환경 생성
python -m venv venv

# 활성화 (Windows)
venv\Scripts\activate

# 활성화 (Mac/Linux)
source venv/bin/activate

# Jinja2 설치
pip install jinja2

✅ 설치 확인​

import jinja2
print(jinja2.__version__)
# 출력: 3.1.2

🚀 첫 번째 템플릿​

문자열 템플릿​

from jinja2 import Template

# 템플릿 생성
template = Template("Hello, {{ name }}!")

# 렌더링
output = template.render(name="World")
print(output)
# 출력: Hello, World!

여러 변수 사용​

from jinja2 import Template

template = Template("""
이름: {{ name }}
나이: {{ age }}
직업: {{ job }}
""")

result = template.render(
    name="홍길동",
    age=30,
    job="개발자"
)
print(result)

출력:

이름: 홍길동
나이: 30
직업: 개발자

📁 파일 기반 템플릿​

디렉토리 구조​

project/
├── app.py
└── templates/
    ├── index.html
    └── about.html

templates/index.html​

<!DOCTYPE html>
<html>
<head>
    <title>{{ title }}</title>
</head>
<body>
    <h1>Welcome, {{ username }}!</h1>
    <p>Today is {{ date }}</p>
</body>
</html>

app.py​

from jinja2 import Environment, FileSystemLoader
from datetime import datetime

# 환경 설정
env = Environment(loader=FileSystemLoader('templates'))

# 템플릿 로드
template = env.get_template('index.html')

# 렌더링
html = template.render(
    title="홈페이지",
    username="홍길동",
    date=datetime.now().strftime("%Y-%m-%d")
)

print(html)

🔧 Environment 설정​

기본 설정​

from jinja2 import Environment, FileSystemLoader

env = Environment(
    loader=FileSystemLoader('templates'),
    autoescape=True,  # 자동 HTML 이스케이핑
    trim_blocks=True,  # 블록 뒤 공백 제거
    lstrip_blocks=True,  # 블록 앞 공백 제거
)

다양한 로더​

FileSystemLoader (파일 시스템)​

from jinja2 import FileSystemLoader

loader = FileSystemLoader('templates')

PackageLoader (패키지)​

from jinja2 import PackageLoader

loader = PackageLoader('myapp', 'templates')

DictLoader (딕셔너리)​

from jinja2 import DictLoader

loader = DictLoader({
    'index.html': '<h1>{{ title }}</h1>',
    'about.html': '<p>{{ content }}</p>'
})

🎨 실전 예제​

이메일 템플릿​

from jinja2 import Template

email_template = Template("""
안녕하세요 {{ user.name }}님,

{{ product }}에 대한 주문이 접수되었습니다.

주문 정보:
- 주문번호: {{ order_id }}
- 수량: {{ quantity }}
- 총액: {{ total|round(2) }}원

배송 예정일: {{ delivery_date }}

감사합니다.
""")

result = email_template.render(
    user={'name': '김철수'},
    product='노트북',
    order_id='ORD-12345',
    quantity=1,
    total=1500000,
    delivery_date='2024-01-20'
)

print(result)

HTML 페이지 생성​

from jinja2 import Environment, FileSystemLoader

env = Environment(loader=FileSystemLoader('templates'))

# templates/blog_post.html
"""
<!DOCTYPE html>
<html>
<head>
    <title>{{ post.title }}</title>
</head>
<body>
    <article>
        <h1>{{ post.title }}</h1>
        <p class="meta">
            작성자: {{ post.author }} |
            작성일: {{ post.date }}
        </p>
        <div class="content">
            {{ post.content }}
        </div>
    </article>
</body>
</html>
"""

template = env.get_template('blog_post.html')

html = template.render(
    post={
        'title': 'Jinja2 시작하기',
        'author': '홍길동',
        'date': '2024-01-15',
        'content': 'Jinja2는 강력한 템플릿 엔진입니다...'
    }
)

# 파일로 저장
with open('output.html', 'w', encoding='utf-8') as f:
    f.write(html)

🔍 디버깅​

템플릿 내용 출력​

from jinja2 import Environment, FileSystemLoader, meta

env = Environment(loader=FileSystemLoader('templates'))

# 템플릿 소스 가져오기
source, filename, uptodate = env.get_loader().get_source(
    env, 'index.html'
)

print("Template source:")
print(source)

# 템플릿에서 사용된 변수 찾기
ast = env.parse(source)
variables = meta.find_undeclared_variables(ast)
print(f"Variables: {variables}")

에러 처리​

from jinja2 import Environment, TemplateNotFound, TemplateSyntaxError

env = Environment(loader=FileSystemLoader('templates'))

try:
    template = env.get_template('nonexistent.html')
except TemplateNotFound as e:
    print(f"템플릿을 찾을 수 없습니다: {e}")

try:
    template = Template("{{ missing }")  # 문법 오류
except TemplateSyntaxError as e:
    print(f"템플릿 문법 오류: {e}")
