🔤 구분자 (Delimiters)​

Jinja2는 세 가지 주요 구분자를 사용합니다:

구분자용도예제
{{ }}변수 출력{{ name }}
{% %}문장 (제어 구조){% if condition %}
{# #}주석{# 이것은 주석입니다 #}

변수 출력 {{ }}​

{{ username }}
{{ user.name }}
{{ items[0] }}
{{ dict['key'] }}

문장 {% %}​

{% if logged_in %}
    환영합니다!
{% endif %}

{% for item in items %}
    {{ item }}
{% endfor %}

주석 \​

{# 이 부분은 렌더링되지 않습니다 #}

{#
    여러 줄
    주석도
    가능합니다
#}

📝 템플릿 구조​

기본 구조​

<!DOCTYPE html>
<html>
<head>
    <title>{{ title }}</title>
</head>
<body>
    {% if user %}
        <h1>Welcome, {{ user.name }}!</h1>
    {% else %}
        <h1>Please login</h1>
    {% endif %}

    {% for post in posts %}
        <article>
            <h2>{{ post.title }}</h2>
            <p>{{ post.content }}</p>
        </article>
    {% endfor %}
</body>
</html>

🔢 표현식​

산술 연산​

{{ 5 + 3 }}        {# 8 #}
{{ 10 - 2 }}       {# 8 #}
{{ 4 * 2 }}        {# 8 #}
{{ 16 / 2 }}       {# 8.0 #}
{{ 17 // 2 }}      {# 8 (정수 나눗셈) #}
{{ 17 % 2 }}       {# 1 (나머지) #}
{{ 2 ** 3 }}       {# 8 (거듭제곱) #}

비교 연산​

{{ age == 18 }}
{{ age != 20 }}
{{ age > 18 }}
{{ age < 30 }}
{{ age >= 18 }}
{{ age <= 65 }}

논리 연산​

{{ True and False }}    {# False #}
{{ True or False }}     {# True #}
{{ not True }}          {# False #}

멤버십 테스트​

{{ 'apple' in ['apple', 'banana'] }}  {# True #}
{{ 'orange' not in fruits }}          {# True/False #}

🎯 리터럴​

문자열​

{{ "Hello World" }}
{{ 'Single quotes' }}
{{ "문자열 " ~ "연결" }}  {# 문자열 연결 #}

숫자​

{{ 42 }}
{{ 3.14 }}
{{ 1e6 }}  {# 1000000.0 #}

리스트​

{{ [1, 2, 3, 4, 5] }}
{{ ['apple', 'banana', 'orange'] }}
{{ [1, 'mixed', True, 3.14] }}

딕셔너리​

{{ {'name': '홍길동', 'age': 30} }}
{{ {'key': value} }}

튜플​

{{ (1, 2, 3) }}
{{ ('a', 'b', 'c') }}

🔗 속성 접근​

점 표기법​

{{ user.name }}
{{ post.author.email }}
{{ company.address.city }}

대괄호 표기법​

{{ user['name'] }}
{{ items[0] }}
{{ dict['key'] }}

메서드 호출​

{{ user.get_full_name() }}
{{ "hello".upper() }}
{{ items.append('new') }}

🧮 실전 예제​

장바구니 합계 계산​

{% set total = 0 %}
<ul>
{% for item in cart %}
    <li>
        {{ item.name }}: {{ item.price }}원
        (수량: {{ item.quantity }})
    </li>
    {% set total = total + (item.price * item.quantity) %}
{% endfor %}
</ul>
<p>총 금액: {{ total|round(2) }}원</p>

조건부 스타일링​

<div class="status {{ 'active' if user.is_active else 'inactive' }}">
    {{ user.status }}
</div>

{# 출력: <div class="status active">온라인</div> #}

복잡한 표현식​

{{ (price * quantity * (1 - discount / 100))|round(2) }}

{% if (age >= 18) and (is_verified or is_admin) %}
    접근 허용
{% endif %}

📐 화이트스페이스 제어​

기본 동작​

<div>
    {% for item in items %}
        {{ item }}
    {% endfor %}
</div>

{# 출력에 공백과 줄바꿈이 포함됨 #}

공백 제거​

<div>
    {%- for item in items %}
        {{ item }}
    {%- endfor %}
</div>

{# - 기호로 앞뒤 공백 제거 #}

자세한 내용은 화이트스페이스 제어 참조​

🔄 변수 할당​

{% set name = "홍길동" %}
{% set age = 30 %}
{% set is_admin = True %}

{{ name }}  {# 홍길동 #}

여러 변수 동시 할당​

{% set x, y, z = 1, 2, 3 %}
{{ x }}, {{ y }}, {{ z }}  {# 1, 2, 3 #}

블록 할당​

{% set navigation %}
    <ul>
        <li>Home</li>
        <li>About</li>
    </ul>
{% endset %}

{{ navigation }}

💡 팁​

  1. 구분자 기억하기

    • 출력: {{ }} (2개)
    • 문장: {% %} (1개 + 공백)
    • 주석: {# #} (해시)

  2. 표현식은 간단하게

    • 복잡한 로직은 Python 코드에서 처리
    • 템플릿은 프레젠테이션에 집중

  3. 가독성 우선

    • 들여쓰기 사용
    • 긴 표현식은 여러 줄로 분리
