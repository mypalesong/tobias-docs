조건문과 반복문으로 템플릿 로직을 제어해봅시다!

🔀 if 조건문​

기본 if​

{% if user.is_active %}
    <p>활성 사용자입니다.</p>
{% endif %}
from jinja2 import Template

template = Template("""
{% if logged_in %}
    환영합니다!
{% endif %}
""")

print(template.render(logged_in=True))

if-else​

{% if age >= 18 %}
    <p>성인입니다.</p>
{% else %}
    <p>미성년자입니다.</p>
{% endif %}

if-elif-else​

{% if score >= 90 %}
    <span class="grade-a">A</span>
{% elif score >= 80 %}
    <span class="grade-b">B</span>
{% elif score >= 70 %}
    <span class="grade-c">C</span>
{% else %}
    <span class="grade-f">F</span>
{% endif %}
template = Template("""
성적: {{ score }}점 -
{% if score >= 90 %}
    우수
{% elif score >= 70 %}
    양호
{% elif score >= 50 %}
    보통
{% else %}
    미흡
{% endif %}
""")

print(template.render(score=85))
# 출력: 성적: 85점 - 양호

복잡한 조건​

{% if (age >= 18) and (has_license or is_admin) %}
    운전 가능
{% endif %}

{% if user.role in ['admin', 'moderator'] %}
    관리 권한 있음
{% endif %}

{% if not user.is_banned %}
    접근 허용
{% endif %}

🔁 for 반복문​

기본 for​

<ul>
{% for item in items %}
    <li>{{ item }}</li>
{% endfor %}
</ul>
template = Template("""
<ul>
{% for fruit in fruits %}
    <li>{{ fruit }}</li>
{% endfor %}
</ul>
""")

print(template.render(fruits=['사과', '바나나', '오렌지']))

출력:

<ul>
    <li>사과</li>
    <li>바나나</li>
    <li>오렌지</li>
</ul>

for-else​

<ul>
{% for item in items %}
    <li>{{ item }}</li>
{% else %}
    <li>항목이 없습니다.</li>
{% endfor %}
</ul>
template = Template("""
{% for item in items %}
    - {{ item }}
{% else %}
    목록이 비어있습니다.
{% endfor %}
""")

print(template.render(items=[]))
# 출력: 목록이 비어있습니다.

딕셔너리 반복​

{% for key, value in dict.items() %}
    {{ key }}: {{ value }}
{% endfor %}
template = Template("""
{% for key, value in user.items() %}
    {{ key }}: {{ value }}
{% endfor %}
""")

print(template.render(user={'name': '홍길동', 'age': 30, 'city': '서울'}))

🔢 loop 변수​

기본 loop 변수​

변수설명
loop.index현재 반복 (1부터 시작)
loop.index0현재 반복 (0부터 시작)
loop.revindex남은 반복 (1부터 시작)
loop.revindex0남은 반복 (0부터 시작)
loop.first첫 번째 반복인지
loop.last마지막 반복인지
loop.length전체 항목 수
loop.cycle순환 헬퍼
loop.depth중첩 깊이
loop.depth0중첩 깊이 (0부터)

인덱스 사용​

{% for item in items %}
    {{ loop.index }}. {{ item }}
{% endfor %}
template = Template("""
{% for name in names %}
    {{ loop.index }}. {{ name }}
{% endfor %}
""")

print(template.render(names=['Alice', 'Bob', 'Charlie']))

출력:

1. Alice
2. Bob
3. Charlie

first와 last​

<ul>
{% for item in items %}
    <li class="{{ 'first' if loop.first }}{{ 'last' if loop.last }}">
        {{ item }}
    </li>
{% endfor %}
</ul>
template = Template("""
{% for item in items %}
{% if loop.first %}
[시작]
{% endif %}
{{ loop.index }}. {{ item }}
{% if loop.last %}
[끝]
{% endif %}
{% endfor %}
""")

print(template.render(items=['A', 'B', 'C']))

loop.cycle​

{% for row in rows %}
    <tr class="{{ loop.cycle('odd', 'even') }}">
        <td>{{ row }}</td>
    </tr>
{% endfor %}
template = Template("""
<table>
{% for item in items %}
    <tr class="{{ loop.cycle('bg-gray', 'bg-white') }}">
        <td>{{ item }}</td>
    </tr>
{% endfor %}
</table>
""")

🎯 중첩 반복문​

{% for category in categories %}
    <h2>{{ category.name }}</h2>
    <ul>
    {% for product in category.products %}
        <li>{{ product.name }} - {{ product.price }}원</li>
    {% endfor %}
    </ul>
{% endfor %}
template = Template("""
{% for category in categories %}
<h3>{{ category.name }}</h3>
<ul>
{% for item in category.items %}
    <li>{{ item }}</li>
{% endfor %}
</ul>
{% endfor %}
""")

categories = [
    {'name': '과일', 'items': ['사과', '바나나']},
    {'name': '채소', 'items': ['당근', '양파']}
]

print(template.render(categories=categories))

🔍 필터링​

if 조건과 함께​

{% for user in users %}
    {% if user.is_active %}
        <li>{{ user.name }}</li>
    {% endif %}
{% endfor %}

필터 사용​

{% for item in items|select('even') %}
    {{ item }}
{% endfor %}

{% for user in users|selectattr('is_active') %}
    {{ user.name }}
{% endfor %}

{% for user in users|rejectattr('is_banned') %}
    {{ user.name }}
{% endfor %}

📊 실전 예제​

테이블 생성​

from jinja2 import Template

template = Template("""
<table>
    <thead>
        <tr>
            <th>#</th>
            <th>이름</th>
            <th>나이</th>
            <th>상태</th>
        </tr>
    </thead>
    <tbody>
    {% for user in users %}
        <tr class="{{ loop.cycle('odd', 'even') }}">
            <td>{{ loop.index }}</td>
            <td>{{ user.name }}</td>
            <td>{{ user.age }}</td>
            <td>
                {% if user.is_active %}
                    <span class="badge-success">활성</span>
                {% else %}
                    <span class="badge-danger">비활성</span>
                {% endif %}
            </td>
        </tr>
    {% else %}
        <tr>
            <td colspan="4">사용자가 없습니다.</td>
        </tr>
    {% endfor %}
    </tbody>
</table>
""")

users = [
    {'name': '홍길동', 'age': 30, 'is_active': True},
    {'name': '김철수', 'age': 25, 'is_active': False},
    {'name': '이영희', 'age': 28, 'is_active': True}
]

print(template.render(users=users))

네비게이션 메뉴​

template = Template("""
<nav>
    <ul>
    {% for item in menu %}
        <li class="{{ 'active' if item.url == current_url }}">
            <a href="{{ item.url }}">
                {{ item.title }}
                {% if item.badge %}
                    <span class="badge">{{ item.badge }}</span>
                {% endif %}
            </a>
            {% if item.children %}
                <ul class="submenu">
                {% for child in item.children %}
                    <li><a href="{{ child.url }}">{{ child.title }}</a></li>
                {% endfor %}
                </ul>
            {% endif %}
        </li>
    {% endfor %}
    </ul>
</nav>
""")

menu = [
    {'title': 'Home', 'url': '/', 'badge': None, 'children': []},
    {'title': 'Products', 'url': '/products', 'badge': 'New', 'children': [
        {'title': 'Electronics', 'url': '/products/electronics'},
        {'title': 'Books', 'url': '/products/books'}
    ]},
    {'title': 'About', 'url': '/about', 'badge': None, 'children': []}
]

html = template.render(menu=menu, current_url='/products')

카드 그리드​

template = Template("""
<div class="row">
{% for product in products %}
    {% if loop.index0 % 3 == 0 and not loop.first %}
    </div><div class="row">
    {% endif %}

    <div class="col-md-4">
        <div class="card">
            <h3>{{ product.name }}</h3>
            <p>{{ product.price|round(2) }}원</p>

            {% if product.discount > 0 %}
                <span class="badge-sale">{{ product.discount }}% 할인</span>
            {% endif %}

            {% if product.stock > 0 %}
                <button>구매하기</button>
            {% else %}
                <button disabled>품절</button>
            {% endif %}
        </div>
    </div>

    {% if loop.last %}
    </div>
    {% endif %}
{% endfor %}
""")

페이지네이션​

template = Template("""
<nav>
    <ul class="pagination">
        {% if current_page > 1 %}
            <li><a href="?page={{ current_page - 1 }}">이전</a></li>
        {% endif %}

        {% for page in range(1, total_pages + 1) %}
            <li class="{{ 'active' if page == current_page }}">
                <a href="?page={{ page }}">{{ page }}</a>
            </li>
        {% endfor %}

        {% if current_page < total_pages %}
            <li><a href="?page={{ current_page + 1 }}">다음</a></li>
        {% endif %}
    </ul>
</nav>
""")

html = template.render(current_page=3, total_pages=10)

💡 모범 사례​

✅ 좋은 예​

{# 명확한 조건 #}
{% if user.is_authenticated and user.is_active %}
    환영합니다!
{% endif %}

{# 간단한 반복 #}
{% for item in items %}
    {{ item }}
{% endfor %}

{# else 활용 #}
{% for item in items %}
    {{ item }}
{% else %}
    항목 없음
{% endfor %}

❌ 피해야 할 예​

{# 너무 복잡한 조건 - Python에서 처리하세요 #}
{% if (user.age >= 18 and user.country == 'KR') or (user.age >= 21 and user.country == 'US') or user.is_admin %}
    ...
{% endif %}

{# 깊은 중첩 - 리팩토링 필요 #}
{% for cat in categories %}
    {% for product in cat.products %}
        {% for review in product.reviews %}
            {% for comment in review.comments %}
                ...
            {% endfor %}
        {% endfor %}
    {% endfor %}
{% endfor %}
