필터

필터를 사용하여 변수를 변환하고 포맷팅해봅시다!

🔧 필터 기본​

기본 문법​

{{ variable|filter }}
{{ variable|filter(argument) }}
{{ variable|filter1|filter2 }}  {# 체이닝 #}
from jinja2 import Template

template = Template("{{ name|upper }}")
print(template.render(name="hello"))
# 출력: HELLO

📝 문자열 필터​

upper / lower​

{{ "hello"|upper }}             {# HELLO #}
{{ "WORLD"|lower }}             {# world #}

capitalize / title​

{{ "hello world"|capitalize }}  {# Hello world #}
{{ "hello world"|title }}       {# Hello World #}
template = Template("{{ text|title }}")
print(template.render(text="python programming"))
# 출력: Python Programming

trim / strip​

{{ "  hello  "|trim }}          {# "hello" #}
{{ "  world  "|strip }}         {# "world" #}

truncate​

{{ long_text|truncate(20) }}
{{ long_text|truncate(20, True) }}      {# 단어 경계에서 자르기 #}
{{ long_text|truncate(20, False, '...') }}  {# 접미사 변경 #}
template = Template("{{ text|truncate(30) }}")
print(template.render(
    text="이것은 매우 긴 텍스트입니다. 잘려야 합니다."
))
# 출력: 이것은 매우 긴 텍스트입니다. 잘려야 합니...

replace​

{{ "Hello World"|replace("World", "Jinja") }}
{# Hello Jinja #}

wordcount​

{{ "Hello World Python"|wordcount }}
{# 3 #}

striptags​

{{ "<p>Hello <b>World</b></p>"|striptags }}
{# Hello World #}

🔢 숫자 필터​

round​

{{ 42.5|round }}                {# 43.0 #}
{{ 42.5|round(0, 'floor') }}    {# 42.0 #}
{{ 42.5|round(0, 'ceil') }}     {# 43.0 #}
{{ 3.141592|round(2) }}         {# 3.14 #}
template = Template("가격: {{ price|round(2) }}원")
print(template.render(price=19.999))
# 출력: 가격: 20.0원

abs​

{{ -42|abs }}                   {# 42 #}
{{ -3.14|abs }}                 {# 3.14 #}

int / float​

{{ "42"|int }}                  {# 42 #}
{{ "3.14"|float }}              {# 3.14 #}

📚 리스트 필터​

length​

{{ items|length }}
{{ [1, 2, 3, 4, 5]|length }}    {# 5 #}

first / last​

{{ items|first }}
{{ items|last }}
template = Template("""
첫 번째: {{ numbers|first }}
마지막: {{ numbers|last }}
""")

print(template.render(numbers=[1, 2, 3, 4, 5]))

join​

{{ items|join(', ') }}
{{ ['a', 'b', 'c']|join(' and ') }}
{# a and b and c #}
template = Template("태그: {{ tags|join(', ') }}")
print(template.render(tags=['Python', 'Jinja2', 'Web']))
# 출력: 태그: Python, Jinja2, Web

sort / reverse​

{{ [3, 1, 4, 1, 5]|sort }}
{# [1, 1, 3, 4, 5] #}

{{ [1, 2, 3, 4, 5]|reverse }}
{# [5, 4, 3, 2, 1] #}

unique​

{{ [1, 2, 2, 3, 3, 3]|unique }}
{# [1, 2, 3] #}

sum​

{{ [1, 2, 3, 4, 5]|sum }}
{# 15 #}
template = Template("총합: {{ prices|sum|round(2) }}원")
print(template.render(prices=[1000, 2000, 3000]))
# 출력: 총합: 6000원

max / min​

{{ [1, 5, 3, 9, 2]|max }}       {# 9 #}
{{ [1, 5, 3, 9, 2]|min }}       {# 1 #}

batch​

{% for row in items|batch(3) %}
    {{ row }}
{% endfor %}

{# [1,2,3], [4,5,6], [7,8,9] 형태로 그룹화 #}

slice​

{% for group in items|slice(3) %}
    {{ group }}
{% endfor %}

🔄 변환 필터​

default​

{{ undefined_var|default('기본값') }}
{{ none_value|default('N/A') }}
{{ ''|default('빈값', true) }}  {# 빈 문자열도 False로 처리 #}
template = Template("{{ user.nickname|default(user.name) }}")
print(template.render(user={'name': '홍길동'}))
# 출력: 홍길동

safe​

{{ html_content|safe }}
{# HTML 이스케이핑 하지 않음 #}
from jinja2 import Template

template = Template("{{ content|safe }}")
print(template.render(content="<b>Bold</b>"))
# 출력: <b>Bold</b> (HTML 그대로 출력)

escape / e​

{{ user_input|escape }}
{{ "<script>alert('xss')</script>"|e }}
{# &lt;script&gt;alert(&#39;xss&#39;)&lt;/script&gt; #}

tojson​

<script>
var data = {{ data|tojson }};
</script>
template = Template("<script>var user = {{ user|tojson }};</script>")
print(template.render(user={'name': '홍길동', 'age': 30}))
# 출력: <script>var user = {"name": "홍길동", "age": 30};</script>

list / dict​

{{ range(5)|list }}             {# [0, 1, 2, 3, 4] #}
{{ [['a', 1], ['b', 2]]|dict }} {# {'a': 1, 'b': 2} #}

📅 날짜/시간 필터 (커스텀)​

from jinja2 import Environment
from datetime import datetime

def datetimeformat(value, format='%Y-%m-%d'):
    return value.strftime(format)

env = Environment()
env.filters['datetimeformat'] = datetimeformat

template = env.from_string("{{ now|datetimeformat('%Y년 %m월 %d일') }}")
print(template.render(now=datetime.now()))
# 출력: 2024년 01월 15일

🎯 실전 예제​

가격 포맷팅​

from jinja2 import Environment

def format_currency(value):
    return f"{value:,.0f}원"

env = Environment()
env.filters['currency'] = format_currency

template = env.from_string("""
상품가: {{ price|currency }}
할인가: {{ (price * 0.9)|round(0)|currency }}
""")

print(template.render(price=1500000))

출력:

상품가: 1,500,000원
할인가: 1,350,000원

사용자 목록​

from jinja2 import Template

template = Template("""
<h2>활성 사용자 ({{ users|selectattr('is_active')|list|length }}명)</h2>
<ul>
{% for user in users|selectattr('is_active')|sort(attribute='name') %}
    <li>
        {{ user.name|title }}
        ({{ user.email|lower }})
    </li>
{% endfor %}
</ul>
""")

users = [
    {'name': 'hong gildong', 'email': 'HONG@EXAMPLE.COM', 'is_active': True},
    {'name': 'kim chulsoo', 'email': 'KIM@EXAMPLE.COM', 'is_active': False},
    {'name': 'lee younghee', 'email': 'LEE@EXAMPLE.COM', 'is_active': True}
]

print(template.render(users=users))

텍스트 요약​

template = Template("""
<article>
    <h2>{{ post.title|title }}</h2>
    <p class="meta">
        작성: {{ post.author|title }} |
        태그: {{ post.tags|join(', ') }} |
        단어 수: {{ post.content|wordcount }}
    </p>
    <div class="summary">
        {{ post.content|truncate(200) }}
    </div>
</article>
""")

post = {
    'title': 'jinja2 template guide',
    'author': 'hong gildong',
    'tags': ['Python', 'Jinja2', 'Template'],
    'content': '이것은 매우 긴 글입니다... ' * 50
}

html = template.render(post=post)

📊 필터 조합 패턴​

안전한 HTML 출력​

{{ user_content|striptags|truncate(100)|safe }}

정렬된 고유 목록​

{{ items|unique|sort|join(', ') }}

조건부 포맷팅​

{{ value|default('없음')|upper }}
{{ email|lower|default('이메일 없음') }}

💡 팁과 트릭​

조건부 필터 적용​

{{ text if text else default_text|upper }}
{{ (text|upper) if should_uppercase else text }}

속성으로 정렬​

{% for item in items|sort(attribute='date', reverse=True) %}
    {{ item.title }}
{% endfor %}

필터와 메서드 혼용​

{{ text.split()|join(' - ')|upper }}

과제 1: 상품 카탈로그​

products = [
    {'name': 'laptop', 'price': 1500000, 'stock': 5, 'tags': ['Electronics', 'Computer']},
    {'name': 'mouse', 'price': 30000, 'stock': 0, 'tags': ['Electronics', 'Accessory']},
    {'name': 'keyboard', 'price': 80000, 'stock': 3, 'tags': ['Electronics', 'Computer']}
]

# 다음을 포함하는 HTML 생성:
# - 상품명을 Title Case로
# - 가격을 통화 형식으로 (1,500,000원)
# - 재고가 있는 상품만 표시
# - 태그를 쉼표로 구분
# - 가격순 정렬

과제 2: 블로그 포스트 목록​

posts = [
    {'title': 'python basics', 'content': '...' * 100, 'tags': ['python', 'tutorial'], 'views': 1200},
    {'title': 'jinja2 guide', 'content': '...' * 80, 'tags': ['jinja2', 'template'], 'views': 850}
]

# 다음을 적용:
# - 제목을 Title Case로
# - 내용을 150자로 제한
# - 태그를 대문자로 변환 후 조인
# - 조회수가 높은 순으로 정렬
