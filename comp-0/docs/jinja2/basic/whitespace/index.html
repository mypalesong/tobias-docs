화이트스페이스 제어

템플릿의 공백과 줄바꿈을 제어하여 깔끔한 출력을 만들어봅시다!

🎯 화이트스페이스 문제​

문제 상황​

<div>
    {% for item in items %}
        <span>{{ item }}</span>
    {% endfor %}
</div>
from jinja2 import Template

template = Template("""
<div>
    {% for item in items %}
        <span>{{ item }}</span>
    {% endfor %}
</div>
""")

print(template.render(items=['A', 'B', 'C']))

출력 (불필요한 공백 포함):

<div>

        <span>A</span>

        <span>B</span>

        <span>C</span>

</div>

➖ 마이너스 기호 -​

앞쪽 공백 제거 -​

<div>
    {%- for item in items %}
        <span>{{ item }}</span>
    {% endfor %}
</div>

뒤쪽 공백 제거 -​

<div>
    {% for item in items -%}
        <span>{{ item }}</span>
    {% endfor %}
</div>

양쪽 공백 제거​

<div>
    {%- for item in items -%}
        <span>{{ item }}</span>
    {%- endfor -%}
</div>
template = Template("""
<div>
    {%- for item in items -%}
        <span>{{ item }}</span>
    {%- endfor -%}
</div>
""")

print(template.render(items=['A', 'B', 'C']))

출력:

<div><span>A</span><span>B</span><span>C</span></div>

⚙️ Environment 설정​

trim_blocks​

블록 태그 뒤의 첫 번째 줄바꿈 제거

from jinja2 import Environment

env = Environment(trim_blocks=True)

template = env.from_string("""
<ul>
{% for item in items %}
    <li>{{ item }}</li>
{% endfor %}
</ul>
""")

lstrip_blocks​

블록 태그 앞의 공백 제거

env = Environment(lstrip_blocks=True)

template = env.from_string("""
<ul>
    {% for item in items %}
    <li>{{ item }}</li>
    {% endfor %}
</ul>
""")

둘 다 사용 (권장)​

from jinja2 import Environment, FileSystemLoader

env = Environment(
    loader=FileSystemLoader('templates'),
    trim_blocks=True,
    lstrip_blocks=True
)

📐 실전 예제​

인라인 목록​

from jinja2 import Template

# ❌ 공백 많음
template1 = Template("""
{{ items[0] }}
{% for item in items[1:] %}
, {{ item }}
{% endfor %}
""")

print(template1.render(items=['A', 'B', 'C']))
# 출력:
# A
#
# , B
#
# , C

# ✅ 공백 제거
template2 = Template("""
{{- items[0] -}}
{%- for item in items[1:] -%}
, {{ item }}
{%- endfor -%}
""")

print(template2.render(items=['A', 'B', 'C']))
# 출력: A, B, C

JSON 출력​

from jinja2 import Environment

env = Environment(
    trim_blocks=True,
    lstrip_blocks=True
)

template = env.from_string("""
{
    "users": [
        {% for user in users %}
        {
            "name": "{{ user.name }}",
            "age": {{ user.age }}
        }{{ "," if not loop.last }}
        {% endfor %}
    ]
}
""")

users = [
    {'name': '홍길동', 'age': 30},
    {'name': '김철수', 'age': 25}
]

print(template.render(users=users))

출력:

{

        {
            "name": "홍길동",
            "age": 30
        },
        {
            "name": "김철수",
            "age": 25
        }
    ]
}

HTML 최적화​

from jinja2 import Environment

env = Environment(
    trim_blocks=True,
    lstrip_blocks=True
)

template = env.from_string("""
<!DOCTYPE html>
<html>
<head>
    <title>{{ title }}</title>
</head>
<body>
    <ul>
    {% for item in items %}
        <li>{{ item }}</li>
    {% endfor %}
    </ul>
</body>
</html>
""")

html = template.render(title='목록', items=['A', 'B', 'C'])

🎨 상황별 패턴​

패턴 1: 인라인 출력​

{# 한 줄로 출력 #}
{{- items|join(', ') -}}

패턴 2: CSV 생성​

{% for row in rows -%}
{{ row.name }},{{ row.age }},{{ row.email }}
{% endfor %}
template = Template("""
{%- for row in rows -%}
{{ row.name }},{{ row.age }},{{ row.email }}
{% endfor -%}
""")

rows = [
    {'name': '홍길동', 'age': 30, 'email': 'hong@example.com'},
    {'name': '김철수', 'age': 25, 'email': 'kim@example.com'}
]

print(template.render(rows=rows))

출력:

홍길동,30,hong@example.com
김철수,25,kim@example.com

패턴 3: 태그 사이 공백 제거​

<div>
    {%- if user -%}
        <span>{{ user.name }}</span>
    {%- else -%}
        <span>Guest</span>
    {%- endif -%}
</div>

패턴 4: 조건부 줄바꿈​

{% for item in items -%}
{{ item }}{{ "\n" if not loop.last }}
{%- endfor %}

📊 비교표​

마이너스 기호 위치​

패턴효과
{%-태그 앞 공백 제거
-%}태그 뒤 공백 제거
{{-변수 앞 공백 제거
-}}변수 뒤 공백 제거
{#-주석 앞 공백 제거
-#}주석 뒤 공백 제거

Environment 옵션​

옵션설명효과
trim_blocks블록 태그 뒤 줄바꿈 제거{% ... %}\n → {% ... %}
lstrip_blocks블록 태그 앞 공백 제거 {% ... %} → {% ... %}
둘 다양쪽 정리깔끔한 출력

🔍 실전 케이스​

SQL 쿼리 생성​

from jinja2 import Environment

env = Environment(
    trim_blocks=True,
    lstrip_blocks=True
)

template = env.from_string("""
SELECT
    {% for column in columns %}
    {{ column }}{{ "," if not loop.last }}
    {% endfor %}
FROM {{ table }}
{% if conditions %}
WHERE
    {% for key, value in conditions.items() %}
    {{ key }} = '{{ value }}'{{ " AND" if not loop.last }}
    {% endfor %}
{% endif %}
""")

sql = template.render(
    columns=['id', 'name', 'email'],
    table='users',
    conditions={'status': 'active', 'role': 'admin'}
)

print(sql)

출력:

SELECT
    id,
    name,
    email
FROM users
WHERE
    status = 'active' AND
    role = 'admin'

env = Environment(trim_blocks=True, lstrip_blocks=True)

template = env.from_string("""
안녕하세요 {{ user.name }}님,

주문이 완료되었습니다.

주문 내역:
{% for item in order.items %}
- {{ item.name }} ({{ item.quantity }}개) - {{ item.price }}원
{% endfor %}

총 금액: {{ order.total }}원

감사합니다.
""")

email = template.render(

        'total': 40000
    }
)

💡 모범 사례​

✅ 권장​

# 전역 설정 사용
env = Environment(
    trim_blocks=True,
    lstrip_blocks=True
)

# 필요한 경우에만 마이너스 사용
template = """
<ul>
{% for item in items %}
    <li>{{- item -}}</li>
{% endfor %}
</ul>
"""

❌ 비권장​

{# 과도한 마이너스 사용 #}
{%- for item in items -%}
    {{- item -}}
{%- endfor -%}

{# 모든 곳에 마이너스 (가독성 저하) #}
{{- user.name -}}{{- " " -}}{{- user.age -}}
