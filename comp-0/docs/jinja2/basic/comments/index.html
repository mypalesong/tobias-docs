주석

템플릿에 주석을 작성하여 코드를 문서화해봅시다!

💬 기본 주석​

단일 라인 주석​

{# 이것은 주석입니다 #}
{# 주석은 렌더링되지 않습니다 #}
from jinja2 import Template

template = Template("""
<h1>제목</h1>
{# 이 부분은 출력되지 않습니다 #}
<p>내용</p>
""")

print(template.render())

출력:

<h1>제목</h1>

<p>내용</p>

    이것은

📝 주석 사용 예제​

코드 설명​

{# 사용자 정보 섹션 #}
<div class="user-info">
    <h2>{{ user.name }}</h2>
    {# 이메일은 인증된 사용자만 표시 #}
    {% if user.is_verified %}
        <p>{{ user.email }}</p>
    {% endif %}
</div>

섹션 구분​

{# ==================== 헤더 영역 ==================== #}
<header>
    <nav>{{ navigation }}</nav>
</header>

{# ==================== 메인 컨텐츠 ==================== #}
<main>
    {{ content }}
</main>

{# ==================== 푸터 영역 ==================== #}
<footer>
    {{ footer }}
</footer>

🚫 코드 주석 처리​

임시 비활성화​

{# 디버깅을 위해 임시로 주석 처리
{% if debug_mode %}
    <div class="debug">
        {{ debug_info }}
    </div>
{% endif %}
#}

A/B 테스트​

{# Version A #}
<button class="btn-primary">클릭하세요</button>

{# Version B - 테스트 중
<button class="btn-success">지금 시작하기</button>
#}

📋 문서화 패턴​

템플릿 헤더​

{#
=============================================================================
템플릿: user_profile.html
작성자: 홍길동
작성일: 2024-01-15
설명: 사용자 프로필 페이지

필수 변수:
    - user: 사용자 객체 (name, email, avatar)
    - posts: 포스트 목록

선택 변수:
    - show_email: 이메일 표시 여부 (기본값: False)
=============================================================================
#}

블록 문서화​

{#
    상품 카드 컴포넌트

    입력:
        - product: 상품 객체
          - name: 상품명 (필수)
          - price: 가격 (필수)
          - image: 이미지 URL (선택)
          - discount: 할인율 (선택, 0-100)

    출력: HTML 카드 엘리먼트
#}
<div class="product-card">
    <h3>{{ product.name }}</h3>
    <p>{{ product.price }}원</p>
    {% if product.discount %}
        <span class="badge">{{ product.discount }}% 할인</span>
    {% endif %}
</div>

💡 주석 vs HTML 주석​

Jinja2 주석 \​

{# 이 주석은 소스코드에서도 보이지 않음 #}
<div>내용</div>

출력 (HTML 소스):

<div>내용</div>

HTML 주석 <!-- -->​

<!-- 이 주석은 HTML 소스에 포함됨 -->
<div>내용</div>

출력 (HTML 소스):

<!-- 이 주석은 HTML 소스에 포함됨 -->
<div>내용</div>

비교​

from jinja2 import Template

template = Template("""
{# Jinja2 주석 - 렌더링 결과에 없음 #}
<!-- HTML 주석 - 렌더링 결과에 있음 -->
<div>내용</div>
""")

print(template.render())

🎯 실전 활용​

API 응답 템플릿​

{#
    API: /api/users
    Method: GET
    Response Format: JSON

    Returns:
        - users: 사용자 목록
        - total: 전체 개수
        - page: 현재 페이지
#}
{
    "users": [
        {% for user in users %}
        {
            "id": {{ user.id }},
            "name": "{{ user.name }}"
            {# "email": "{{ user.email }}" - 보안상 제외 #}
        }{{ "," if not loop.last }}
        {% endfor %}
    ],
    "total": {{ total }},
    "page": {{ page }}
}

설정 파일 템플릿​

{#
    Nginx Configuration Template
    Generated: {{ now }}
    Environment: {{ environment }}
#}

server {
    listen {{ port }};
    server_name {{ server_name }};

    {# SSL 설정 - 프로덕션 환경에서만 #}
    {% if environment == 'production' %}
    ssl_certificate {{ ssl_cert }};
    ssl_certificate_key {{ ssl_key }};
    {% endif %}

    location / {
        {# 정적 파일 서빙 #}
        root {{ root_path }};
        index index.html;
    }
}

이메일 템플릿​

{#
    이메일: 주문 확인
    트리거: 주문 완료 시
    수신자: 구매자
#}
<!DOCTYPE html>
<html>
<head>
    <title>주문 확인</title>
</head>
<body>
    {# 헤더 #}
    <h1>주문이 완료되었습니다!</h1>

    {# 주문 정보 #}
    <div class="order-info">
        <p>주문번호: {{ order.id }}</p>
        <p>주문일시: {{ order.created_at }}</p>

        {# 상품 목록 #}
        <h2>주문 상품</h2>
        <ul>
        {% for item in order.items %}
            <li>
                {{ item.name }} - {{ item.quantity }}개
                {# 가격 표시 #}
                ({{ (item.price * item.quantity)|round(2) }}원)
            </li>
        {% endfor %}
        </ul>

        {# 총액 #}
        <p><strong>총 금액: {{ order.total }}원</strong></p>
    </div>

    {# 푸터 #}
    <footer>
        <p>{{ company_name }}</p>
        {# 고객센터: {{ support_email }} #}
    </footer>
</body>
</html>

📋 실습 과제​

과제 1: 템플릿 문서화​

다음 템플릿에 적절한 주석을 추가하세요:

{% extends "base.html" %}

{% block content %}
<div class="container">
    {% for post in posts %}
    <article>
        <h2>{{ post.title }}</h2>
        <p>{{ post.content|truncate(200) }}</p>
        {% if post.author %}
        <span>작성자: {{ post.author.name }}</span>
        {% endif %}
    </article>
    {% endfor %}
</div>
{% endblock %}

과제 2: 설정 파일 주석​

환경별 설정 템플릿에 주석을 추가하세요:

{
    "debug": {{ debug }},
    "database": {
        "host": "{{ db_host }}",
        "port": {{ db_port }}
    },
    {% if environment == 'production' %}
    "cache": {
        "enabled": true,
        "ttl": 3600
    }
    {% endif %}
}
