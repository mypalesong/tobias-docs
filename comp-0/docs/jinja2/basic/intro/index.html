기초편 소개

Jinja2의 기본 개념과 핵심 문법을 배워봅시다!

📚 이 섹션에서 배울 내용​

1. 설치 및 환경 설정​

  • Jinja2 설치 방법
  • 기본 환경 설정
  • 첫 번째 템플릿 작성

2. 기본 문법​

  • 구분자 (Delimiters)
  • 변수, 표현식, 문장
  • 템플릿 구조

3. 변수와 표현식​

  • 변수 출력
  • 속성 접근
  • 인덱싱과 슬라이싱
  • 표현식 작성

4. 필터​

  • 필터 기본 사용법
  • 체이닝
  • 주요 내장 필터
  • 필터 인자

5. 제어 구조​

  • if-elif-else 조건문
  • for 반복문
  • 루프 변수
  • 필터링과 정렬

6. 주석​

  • 단일 라인 주석
  • 다중 라인 주석
  • 주석 처리 팁

7. 화이트스페이스 제어​

  • 공백 제거
  • 라인 제어
  • 최적화

🎯 학습 목표​

이 섹션을 마치면 다음을 할 수 있습니다:

  • ✅ Jinja2를 설치하고 기본 환경을 구성할 수 있습니다
  • ✅ 변수를 출력하고 필터를 적용할 수 있습니다
  • ✅ 조건문과 반복문을 사용할 수 있습니다
  • ✅ 간단한 템플릿을 작성할 수 있습니다
  • ✅ 템플릿의 화이트스페이스를 제어할 수 있습니다

💡 학습 방법​

  1. 순서대로 학습: 각 문서를 순서대로 읽어주세요
  2. 코드 실행: 예제 코드를 직접 실행해보세요
  3. 실습 문제: 각 섹션의 실습 문제를 풀어보세요
  4. 프로젝트 적용: 간단한 프로젝트를 만들어보세요

🚀 준비물​

필수​

  • Python 3.6 이상
  • 텍스트 에디터 (VS Code, PyCharm 등)
  • 터미널/명령 프롬프트

선택사항​

  • Git (버전 관리)
  • 가상 환경 (venv, conda)

📖 예제 미리보기​

변수 출력​

from jinja2 import Template

template = Template("안녕하세요, {{ name }}님!")
print(template.render(name="홍길동"))
# 출력: 안녕하세요, 홍길동님!

조건문​

{% if age >= 18 %}
    성인입니다.
{% else %}
    미성년자입니다.
{% endif %}

반복문​

<ul>
{% for item in items %}
    <li>{{ item }}</li>
{% endfor %}
</ul>

필터​

{{ "hello world"|upper }}
{# 출력: HELLO WORLD #}

{{ price|round(2) }}
{# 출력: 19.99 #}
