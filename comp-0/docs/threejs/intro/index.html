Three.js 완벽 가이드

웹에서 멋진 3D 그래픽을 만들어봅시다!

🎯 Three.js란?​

Three.js는 WebGL을 쉽게 사용할 수 있게 해주는 JavaScript 3D 라이브러리입니다.

핵심 특징​

  • 간단한 API: 복잡한 WebGL을 쉽게 사용
  • 풍부한 기능: 3D 그래픽에 필요한 모든 것 제공
  • 크로스 브라우저: 모든 최신 브라우저 지원
  • 활발한 커뮤니티: 수천 개의 예제와 튜토리얼
  • 오픈소스: MIT 라이선스

Three.js로 할 수 있는 것​

  • 🎮 게임 개발: 브라우저 기반 3D 게임
  • 📊 데이터 시각화: 3D 차트와 그래프
  • 🏗️ 건축 시각화: 3D 건물 모델 뷰어
  • 🎨 인터랙티브 아트: 창의적인 3D 경험
  • 🛒 제품 프리뷰: 상품 3D 뷰어
  • 🌌 VR/AR: 가상/증강 현실 경험

🚀 빠른 시작​

설치​

# npm
npm install three

# CDN
<script type="importmap">
{
  "imports": {
    "three": "https://cdn.jsdelivr.net/npm/three@0.160.0/build/three.module.js"
  }
}
</script>

첫 번째 Scene​

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>My First Three.js Scene</title>
    <style>
        body { margin: 0; }
        canvas { display: block; }
    </style>
</head>
<body>
    <script type="module">
        import * as THREE from 'three';

        // Scene 생성
        const scene = new THREE.Scene();

        // Camera 생성
        const camera = new THREE.PerspectiveCamera(
            75,                                   // FOV
            window.innerWidth / window.innerHeight, // Aspect ratio
            0.1,                                  // Near plane
            1000                                  // Far plane
        );
        camera.position.z = 5;

        // Renderer 생성
        const renderer = new THREE.WebGLRenderer();
        renderer.setSize(window.innerWidth, window.innerHeight);
        document.body.appendChild(renderer.domElement);

        // Cube 생성
        const geometry = new THREE.BoxGeometry(1, 1, 1);
        const material = new THREE.MeshBasicMaterial({ color: 0x00ff00 });
        const cube = new THREE.Mesh(geometry, material);
        scene.add(cube);

        // 애니메이션
        function animate() {
            requestAnimationFrame(animate);

            cube.rotation.x += 0.01;
            cube.rotation.y += 0.01;

            renderer.render(scene, camera);
        }

        animate();
    </script>
</body>
</html>

🎨 주요 개념​

Scene Graph​

Scene
├── Camera
├── Lights
│   ├── AmbientLight
│   └── DirectionalLight
└── Objects
    ├── Mesh (Geometry + Material)
    ├── Group
    │   ├── Mesh
    │   └── Mesh
    └── Mesh

렌더링 파이프라인​

1. Scene 설정
   ↓
2. Object 추가 (Mesh, Lights, Camera)
   ↓
3. Animation Loop
   ↓
4. 상태 업데이트 (position, rotation, etc.)
   ↓
5. Renderer.render(scene, camera)
   ↓
6. 반복 (requestAnimationFrame)

핵심 컴포넌트​

Scene (장면)​

const scene = new THREE.Scene();
scene.background = new THREE.Color(0x000000);
scene.fog = new THREE.Fog(0xcccccc, 10, 15);

Camera (카메라)​

// 원근 카메라
const camera = new THREE.PerspectiveCamera(75, aspect, 0.1, 1000);

// 직교 카메라
const camera = new THREE.OrthographicCamera(
    left, right, top, bottom, near, far
);

Renderer (렌더러)​

const renderer = new THREE.WebGLRenderer({ antialias: true });
renderer.setSize(window.innerWidth, window.innerHeight);
renderer.setPixelRatio(window.devicePixelRatio);

Geometry (기하학)​

// 기본 도형
const box = new THREE.BoxGeometry(1, 1, 1);
const sphere = new THREE.SphereGeometry(1, 32, 32);
const plane = new THREE.PlaneGeometry(10, 10);

Material (재질)​

// 기본 재질
const basic = new THREE.MeshBasicMaterial({ color: 0xff0000 });

// 표준 재질 (조명 반응)
const standard = new THREE.MeshStandardMaterial({
    color: 0xff0000,
    roughness: 0.5,
    metalness: 0.5
});

Light (조명)​

// 주변광
const ambient = new THREE.AmbientLight(0xffffff, 0.5);

// 방향광
const directional = new THREE.DirectionalLight(0xffffff, 1);
directional.position.set(5, 5, 5);

// 점광
const point = new THREE.PointLight(0xffffff, 1, 100);

모듈화된 코드​

// src/scene.js
export function createScene() {
    const scene = new THREE.Scene();
    scene.background = new THREE.Color(0x000000);
    return scene;
}

// src/camera.js
export function createCamera(aspect) {
    const camera = new THREE.PerspectiveCamera(75, aspect, 0.1, 1000);
    camera.position.z = 5;
    return camera;
}

// src/renderer.js
export function createRenderer() {
    const renderer = new THREE.WebGLRenderer({ antialias: true });
    renderer.setSize(window.innerWidth, window.innerHeight);
    return renderer;
}

// src/main.js
import { createScene } from './scene.js';
import { createCamera } from './camera.js';
import { createRenderer } from './renderer.js';

const scene = createScene();
const camera = createCamera(window.innerWidth / window.innerHeight);
const renderer = createRenderer();

document.body.appendChild(renderer.domElement);

function animate() {
    requestAnimationFrame(animate);
    renderer.render(scene, camera);
}

animate();
