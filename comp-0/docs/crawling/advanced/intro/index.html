고급편 소개

실전 크롤링에 필요한 고급 기술을 마스터해봅시다!

🎯 학습 목표​

고급편을 마치면 다음을 할 수 있습니다:

  • ✅ 봇 탐지 우회 및 안티 크롤링 대응
  • ✅ 프록시 설정 및 IP 로테이션
  • ✅ 속도 제한 및 예의있는 크롤링
  • ✅ 견고한 에러 처리 및 재시도 로직
  • ✅ 성능 최적화 및 병렬 처리

📋 목차​

  1. 봇 탐지 우회 - User-Agent, Stealth 모드
  2. 프록시 설정 - IP 로테이션, 프록시 체인
  3. 속도 제한 - 예의있는 크롤링
  4. 에러 처리 & 재시도 - 안정적인 스크래핑
  5. 성능 최적화 - 병렬 처리, 캐싱

⚠️ 윤리적 크롤링​

반드시 지켜야 할 원칙​

  1. robots.txt 준수

    # robots.txt 확인
    # https://example.com/robots.txt

  2. 속도 제한

    • 서버에 부담을 주지 않도록 요청 간격 조절
    • 일반적으로 1-3초 간격 권장

  3. User-Agent 설정

    • 봇임을 명시하고 연락처 포함
    • 예: MyBot/1.0 (+http://mysite.com/bot)

  4. 이용 약관 준수

    • 사이트의 ToS(Terms of Service) 확인
    • 로그인이 필요한 콘텐츠는 신중히 접근

법적 고려사항​

  • ⚖️ 저작권 침해 주의
  • 🔒 개인정보 수집 시 GDPR/개인정보보호법 준수
  • 📜 사이트 이용약관 확인

🛡️ 안티 크롤링 대응​

일반적인 탐지 방법​

  1. User-Agent 확인
  2. 행동 패턴 분석 (클릭 속도, 마우스 움직임)
  3. IP 추적
  4. CAPTCHA
  5. JavaScript 챌린지

대응 전략​

# 1. Stealth 모드
from playwright_stealth import stealth_sync

with sync_playwright() as p:
    browser = p.chromium.launch()
    page = browser.new_page()
    stealth_sync(page)  # 스텔스 적용
    page.goto('https://bot-protected-site.com')
# 2. 사람처럼 행동
import random
import time

# 랜덤 대기
time.sleep(random.uniform(2, 5))

# 마우스 움직임 시뮬레이션
page.mouse.move(random.randint(0, 800), random.randint(0, 600))

🚀 고급 패턴​

1. 세션 관리​

# 쿠키 저장
context = browser.new_context()
page = context.new_page()

# 로그인 후
page.goto('https://example.com/login')
page.fill('input[name="username"]', 'user')
page.fill('input[name="password"]', 'pass')
page.click('button[type="submit"]')

# 쿠키 저장
cookies = context.cookies()
import json
with open('cookies.json', 'w') as f:
    json.dump(cookies, f)

# 다음 실행 시 복원
with open('cookies.json', 'r') as f:
    cookies = json.load(f)
context.add_cookies(cookies)

2. 네트워크 모니터링​

# 네트워크 요청 가로채기
def handle_route(route):
    # API 호출만 로깅
    if '/api/' in route.request.url:
        print(f"API 호출: {route.request.url}")
    route.continue_()

page.route('**/*', handle_route)

3. 분산 크롤링​

# 여러 브라우저 인스턴스로 병렬 처리
from concurrent.futures import ThreadPoolExecutor

def scrape_page(url):
    with sync_playwright() as p:
        browser = p.chromium.launch()
        page = browser.new_page()
        page.goto(url)
        # 데이터 수집
        browser.close()

urls = ['url1', 'url2', 'url3']
with ThreadPoolExecutor(max_workers=3) as executor:
    executor.map(scrape_page, urls)
