선택자 마스터하기

Playwright의 강력한 선택자로 정확하게 요소를 찾아봅시다!

🎯 선택자 종류​

Playwright는 다양한 선택자를 지원합니다:

선택자설명예제
CSSCSS 선택자.class, #id, div > p
XPathXPath 표현식//div[@class='name']
Text텍스트 내용으로 검색text=로그인
RoleARIA role로 검색role=button[name="제출"]
Test IDdata-testid 속성data-testid=submit-btn

📝 CSS 선택자​

기본 선택자​

# ID로 선택
page.query_selector('#login-button')

# 클래스로 선택
page.query_selector('.user-name')

# 태그로 선택
page.query_selector('button')

# 속성으로 선택
page.query_selector('[type="submit"]')
page.query_selector('[data-id="123"]')

조합 선택자​

# 자식 선택자
page.query_selector('div > p')

# 후손 선택자
page.query_selector('div p')

# 인접 형제
page.query_selector('h1 + p')

# 일반 형제
page.query_selector('h1 ~ p')

가상 클래스​

# 첫 번째 요소
page.query_selector('li:first-child')

# 마지막 요소
page.query_selector('li:last-child')

# n번째 요소
page.query_selector('li:nth-child(3)')

# 짝수/홀수
page.query_selector('tr:nth-child(even)')
page.query_selector('tr:nth-child(odd)')

# 포함하지 않는
page.query_selector('div:not(.exclude)')

실전 예제​

from playwright.sync_api import sync_playwright

def css_selector_examples():
    with sync_playwright() as p:
        browser = p.chromium.launch()
        page = browser.new_page()
        page.goto('https://books.toscrape.com')

        # 모든 책 제목
        titles = page.query_selector_all('.product_pod h3 > a')
        for title in titles:
            print(title.get_attribute('title'))

        # 가격이 있는 요소
        prices = page.query_selector_all('.price_color')
        for price in prices:
            print(price.inner_text())

        # 별점 4개 이상인 책
        four_star_books = page.query_selector_all('.star-rating.Four, .star-rating.Five')
        print(f"4성 이상 책: {len(four_star_books)}권")

        browser.close()

🔍 XPath 선택자​

XPath는 더 복잡한 조건에 유용합니다.

기본 XPath​

# 태그로 선택
page.query_selector('xpath=//div')

# 속성으로 선택
page.query_selector('xpath=//button[@type="submit"]')

# 텍스트로 선택
page.query_selector('xpath=//h1[text()="제목"]')

# 텍스트 포함
page.query_selector('xpath=//p[contains(text(), "검색어")]')

고급 XPath​

# 조상 선택
page.query_selector('xpath=//td[text()="값"]/ancestor::tr')

# 형제 선택
page.query_selector('xpath=//label[text()="이름"]/following-sibling::input')

# 부모 선택
page.query_selector('xpath=//button[@class="btn"]/parent::div')

# 여러 조건
page.query_selector('xpath=//div[@class="product" and @data-available="true"]')

XPath vs CSS 비교​

# 텍스트로 버튼 찾기
# XPath (가능)
page.query_selector('xpath=//button[text()="로그인"]')

# CSS (불가능 - Text 선택자 사용)
page.query_selector('button:has-text("로그인")')  # Playwright 확장

📝 Text 선택자​

텍스트 내용으로 요소를 찾는 가장 직관적인 방법:

# 정확한 텍스트 매칭
page.click('text=로그인')

# 부분 매칭
page.click('text="로그"')  # "로그인", "로그아웃" 모두 매칭

# 대소문자 무시
page.click('text=/login/i')

# 정규표현식
page.click('text=/^로그[인아웃]+$/')

실전 예제​

# 링크 클릭
page.click('text="다음 페이지"')

# 버튼 클릭
page.click('text="제출하기"')

# 특정 텍스트를 포함하는 div
content = page.query_selector('text="Copyright"')
print(content.inner_text())

🎭 Role 선택자 (권장)​

접근성을 고려한 선택자 - 가장 안정적!

# 버튼
page.click('role=button[name="제출"]')

# 링크
page.click('role=link[name="홈으로"]')

# 텍스트 입력
page.fill('role=textbox[name="이메일"]', 'user@example.com')

# 체크박스
page.check('role=checkbox[name="동의"]')

# 헤딩
heading = page.query_selector('role=heading[level=1]')

주요 Role 목록​

Role설명예제
button버튼role=button
link링크role=link
textbox텍스트 입력role=textbox
checkbox체크박스role=checkbox
heading제목role=heading[level=2]
list리스트role=list
listitem리스트 아이템role=listitem

🏷️ Test ID 선택자​

테스트를 위한 data-testid 속성 활용:

<button data-testid="submit-btn">제출</button>
# data-testid로 선택
page.click('[data-testid="submit-btn"]')

# 또는
page.click('data-testid=submit-btn')

🔗 선택자 체이닝​

여러 선택자를 조합:

# CSS + Text
page.click('.nav-menu >> text=로그인')

# Role + Text
page.click('role=button >> text=확인')

# 중첩 선택
page.query_selector('.modal >> .footer >> button.submit')

💡 선택자 우선순위 (권장 순서)​

  1. Role 선택자 - 접근성 고려, 안정적
  2. Test ID - 테스트 전용, 변경 적음
  3. Text 선택자 - 직관적
  4. CSS ID - 유니크한 경우
  5. CSS Class - 구조에 의존
  6. XPath - 복잡한 조건 필요 시

🎯 실전 예제: 복잡한 페이지​

from playwright.sync_api import sync_playwright

def complex_selectors():
    with sync_playwright() as p:
        browser = p.chromium.launch()
        page = browser.new_page()
        page.goto('https://example-ecommerce.com')

        # 1. Role 선택자로 검색
        page.fill('role=textbox[name="Search"]', 'laptop')
        page.click('role=button[name="Search"]')

        # 2. 가격 범위로 필터링
        products = page.query_selector_all('.product-card')

        for product in products:
            # 제품명 (CSS)
            name = product.query_selector('h3.product-name').inner_text()

            # 가격 (XPath - $기호 제거)
            price_text = product.query_selector('xpath=.//span[@class="price"]').inner_text()
            price = float(price_text.replace('$', ''))

            # 재고 있는 제품만
            in_stock = product.query_selector('text="In Stock"')

            if in_stock and price < 1000:
                print(f"{name}: ${price}")

                # 장바구니 추가 (Text 선택자)
                add_button = product.query_selector('text="Add to Cart"')
                if add_button:
                    add_button.click()

        browser.close()

if __name__ == "__main__":
    complex_selectors()

element = page.query_selector('.some-class')
if element:
    print("요소를 찾았습니다!")
else:
    print("요소를 찾을 수 없습니다.")
