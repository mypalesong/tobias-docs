설치 및 환경 설정

Playwright를 설치하고 크롤링 환경을 구성해봅시다!

📦 요구사항​

  • Python 3.8 이상
  • pip (Python 패키지 관리자)
  • 4GB 이상의 디스크 공간 (브라우저 다운로드용)

🔧 설치 과정​

1. 가상 환경 생성 (권장)​

# Windows
python -m venv venv
venv\Scripts\activate

# macOS/Linux
python3 -m venv venv
source venv/bin/activate

2. Playwright 설치​

pip install playwright

3. 브라우저 설치​

# 모든 브라우저 설치 (Chromium, Firefox, WebKit)
playwright install

# 특정 브라우저만 설치
playwright install chromium
playwright install firefox
playwright install webkit

4. 설치 확인​

# test_installation.py
from playwright.sync_api import sync_playwright

def test_installation():
    with sync_playwright() as p:
        # Chromium 브라우저 실행
        browser = p.chromium.launch(headless=False)  # headless=False로 브라우저 창 표시
        page = browser.new_page()

        # 예제 페이지 방문
        page.goto('https://playwright.dev')

        # 페이지 제목 출력
        print(f"Page title: {page.title()}")

        # 3초 대기 (브라우저 확인용)
        page.wait_for_timeout(3000)

        # 브라우저 종료
        browser.close()
        print("✅ Playwright 설치 성공!")

if __name__ == "__main__":
    test_installation()
python test_installation.py

🌐 브라우저 옵션​

Chromium (권장)​

  • 가장 빠르고 안정적
  • Chrome과 동일한 엔진
browser = p.chromium.launch()

Firefox​

  • Gecko 엔진 사용
  • 일부 사이트에서 필요
browser = p.firefox.launch()

WebKit​

  • Safari와 동일한 엔진
  • macOS/iOS 테스트에 유용
browser = p.webkit.launch()

⚙️ 실행 옵션​

헤드리스 모드 (기본값)​

# 브라우저 창 없이 백그라운드 실행
browser = p.chromium.launch(headless=True)

헤드풀 모드 (개발 시 유용)​

# 브라우저 창 표시
browser = p.chromium.launch(headless=False)

느린 실행 (디버깅용)​

# 각 동작 사이에 지연 추가 (밀리초)
browser = p.chromium.launch(
    headless=False,
    slow_mo=1000  # 1초 지연
)

브라우저 크기 설정​

browser = p.chromium.launch()
context = browser.new_context(
    viewport={'width': 1920, 'height': 1080}
)
page = context.new_page()

🐳 Docker에서 설치​

FROM python:3.11-slim

# 시스템 의존성 설치
RUN apt-get update && apt-get install -y \
    wget \
    gnupg \
    && rm -rf /var/lib/apt/lists/*

# Playwright 설치
RUN pip install playwright

# 브라우저 설치
RUN playwright install --with-deps chromium

WORKDIR /app
COPY . .

CMD ["python", "scraper.py"]

📚 추가 패키지 (선택)​

데이터 처리​

pip install pandas beautifulsoup4

스크래핑 유틸리티​

pip install requests lxml

데이터 저장​

pip install pymongo psycopg2-binary sqlalchemy

전체 설치​

# requirements.txt
playwright==1.40.0
pandas==2.1.0
beautifulsoup4==4.12.0
requests==2.31.0
python-dotenv==1.0.0
pip install -r requirements.txt

🔍 트러블슈팅​

"playwright: command not found"​

# PATH 확인
python -m playwright install

브라우저 실행 실패​

# 의존성 다시 설치
playwright install-deps chromium

WSL (Windows Subsystem for Linux)에서 실행​

# X11 포워딩 설정
export DISPLAY=:0

macOS에서 권한 오류​

# Playwright에 권한 부여
xattr -cr /path/to/chromium

✅ 설치 완료!​
