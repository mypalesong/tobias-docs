첫 번째 스크립트

첫 번째 웹 크롤링 스크립트를 작성해봅시다!

🎯 목표​

간단한 뉴스 사이트에서 제목과 링크를 추출합니다.

📝 기본 구조​

from playwright.sync_api import sync_playwright

def scrape_website():
    with sync_playwright() as p:
        # 1. 브라우저 실행
        browser = p.chromium.launch(headless=False)

        # 2. 새 페이지 생성
        page = browser.new_page()

        # 3. URL 방문
        page.goto('https://news.ycombinator.com')

        # 4. 데이터 추출
        titles = page.query_selector_all('.titleline > a')

        # 5. 결과 출력
        for title in titles:
            text = title.inner_text()
            link = title.get_attribute('href')
            print(f"{text}: {link}")

        # 6. 브라우저 종료
        browser.close()

if __name__ == "__main__":
    scrape_website()

🔍 코드 설명​

1. Playwright 컨텍스트 생성​

with sync_playwright() as p:
    # Playwright 인스턴스 자동 관리

sync_playwright()는 컨텍스트 관리자로, 자동으로 리소스를 정리합니다.

2. 브라우저 실행​

browser = p.chromium.launch(headless=False)
  • headless=False: 브라우저 창 표시 (디버깅 시 유용)
  • headless=True: 백그라운드 실행 (프로덕션)

3. 페이지 생성​

page = browser.new_page()

새로운 탭(페이지)을 생성합니다.

4. URL 이동​

page.goto('https://news.ycombinator.com')

지정된 URL로 이동하고 페이지 로드를 기다립니다.

5. 요소 선택​

titles = page.query_selector_all('.titleline > a')
  • query_selector(): 첫 번째 매칭 요소
  • query_selector_all(): 모든 매칭 요소

6. 데이터 추출​

text = title.inner_text()  # 텍스트 내용
link = title.get_attribute('href')  # 속성 값

💡 개선된 버전​

from playwright.sync_api import sync_playwright
import csv
from datetime import datetime

def scrape_hacker_news():
    """Hacker News 상위 30개 기사 스크래핑"""

    with sync_playwright() as p:
        # 브라우저 실행
        browser = p.chromium.launch(headless=True)
        page = browser.new_page()

        # 페이지 방문
        print("Hacker News 방문 중...")
        page.goto('https://news.ycombinator.com')

        # 제목 추출
        stories = []
        title_elements = page.query_selector_all('.titleline > a')

        for idx, title in enumerate(title_elements, 1):
            story = {
                'rank': idx,
                'title': title.inner_text(),
                'url': title.get_attribute('href'),
                'scraped_at': datetime.now().isoformat()
            }
            stories.append(story)
            print(f"{idx}. {story['title']}")

        # CSV 저장
        with open('hacker_news.csv', 'w', newline='', encoding='utf-8') as f:
            writer = csv.DictWriter(f, fieldnames=['rank', 'title', 'url', 'scraped_at'])
            writer.writeheader()
            writer.writerows(stories)

        print(f"\n✅ {len(stories)}개 기사를 hacker_news.csv에 저장했습니다!")

        browser.close()

if __name__ == "__main__":
    scrape_hacker_news()

🎨 비동기 버전​

더 빠른 실행을 위한 비동기 버전:

import asyncio
from playwright.async_api import async_playwright

async def scrape_async():
    async with async_playwright() as p:
        browser = await p.chromium.launch()
        page = await browser.new_page()

        await page.goto('https://news.ycombinator.com')

        titles = await page.query_selector_all('.titleline > a')

        for title in titles:
            text = await title.inner_text()
            link = await title.get_attribute('href')
            print(f"{text}: {link}")

        await browser.close()

if __name__ == "__main__":
    asyncio.run(scrape_async())

📊 실전 예제: 날씨 정보 스크래핑​

from playwright.sync_api import sync_playwright

def get_weather():
    with sync_playwright() as p:
        browser = p.chromium.launch()
        page = browser.new_page()

        # 구글 날씨 검색
        page.goto('https://www.google.com/search?q=날씨')

        # 날씨 정보 추출
        location = page.query_selector('#wob_loc')
        temperature = page.query_selector('#wob_tm')
        condition = page.query_selector('#wob_dc')

        if location and temperature and condition:
            print(f"지역: {location.inner_text()}")
            print(f"온도: {temperature.inner_text()}°C")
            print(f"날씨: {condition.inner_text()}")

        browser.close()

if __name__ == "__main__":
    get_weather()

💡 팁​

  1. 개발자 도구 활용

    • F12를 눌러 요소 구조 확인
    • 선택자를 미리 테스트

  2. 작은 것부터 시작

    • 먼저 한 개 요소만 추출
    • 성공하면 전체로 확장

  3. 에러 처리

    • 요소가 없을 수 있음을 고려
    • try-except로 안전하게 처리
try:
    title = page.query_selector('h1').inner_text()
except:
    title = "제목 없음"
