React 중급

기초를 마스터했다면, 이제 React의 더 고급 기능과 실무 패턴을 학습할 차례입니다.

중급 과정 개요​

이 과정에서는 React의 핵심 고급 기능들을 배우고, 실제 애플리케이션을 구축하는 데 필요한 패턴과 기법을 익힙니다.

학습 내용​

1. Hooks 소개​

React Hooks의 개념과 기본 사용법을 배웁니다.

  • useState 심화
  • useReducer
  • useRef
  • 기타 내장 Hooks

2. useEffect Hook​

부수 효과(side effects)를 처리하는 방법을 학습합니다.

  • 데이터 가져오기
  • 구독 관리
  • 정리(cleanup) 함수
  • 의존성 배열

3. Custom Hooks​

재사용 가능한 로직을 만드는 방법을 배웁니다.

  • Custom Hook 작성법
  • 공통 패턴
  • 베스트 프랙티스

4. Context API​

전역 상태를 관리하는 방법을 학습합니다.

  • Context 생성과 사용
  • Provider와 Consumer
  • useContext Hook
  • Props Drilling 해결

5. 폼 처리​

복잡한 폼을 효율적으로 다루는 방법을 배웁니다.

  • Controlled vs Uncontrolled
  • 폼 유효성 검사
  • 폼 라이브러리 (React Hook Form)

6. 컴포넌트 라이프사이클​

컴포넌트의 생명주기를 이해합니다.

  • 마운트, 업데이트, 언마운트
  • 클래스 vs 함수 컴포넌트
  • 라이프사이클 메서드

7. Error Boundaries​

오류를 우아하게 처리하는 방법을 학습합니다.

  • Error Boundary 구현
  • 오류 로깅
  • Fallback UI

사전 요구사항​

중급 과정을 시작하기 전에 다음을 이해하고 있어야 합니다:

  • JSX 문법
  • 컴포넌트 작성
  • Props와 State
  • 이벤트 처리
  • 조건부 렌더링
  • 리스트와 Key

학습 목표​

이 과정을 완료하면 다음을 할 수 있습니다:

  • ✓ React Hooks를 효과적으로 사용
  • ✓ 재사용 가능한 Custom Hooks 작성
  • ✓ Context API로 전역 상태 관리
  • ✓ 복잡한 폼 구현
  • ✓ API 데이터 가져오기 및 관리
  • ✓ 오류 처리 및 로깅
  • ✓ 성능 최적화 기초

실전 프로젝트​

중급 과정에서는 다음과 같은 프로젝트를 만들어볼 것을 권장합니다:

  1. Todo 앱: CRUD 기능이 있는 완전한 Todo 애플리케이션
  2. 날씨 앱: API를 사용하여 날씨 정보를 가져오는 앱
  3. 블로그 플랫폼: 게시글 작성, 편집, 삭제 기능
  4. 인증 시스템: 로그인/로그아웃 기능이 있는 앱
