고급편 소개

프로덕션 환경에서 RabbitMQ를 운영하는 방법을 배워봅시다!

🎯 이번 섹션에서 배울 내용​

고급편에서는 엔터프라이즈 환경에서 필수적인 운영 기술을 다룹니다.

주요 학습 내용​

  1. 클러스터링

    • 고가용성 구성
    • 노드 관리
    • 미러링 큐
    • 장애 복구

  2. 모니터링

    • 메트릭 수집
    • 알림 설정
    • Grafana 연동
    • 성능 대시보드

  3. 성능 최적화

    • 처리량 향상
    • 메모리 관리
    • 디스크 I/O 최적화
    • 병목 현상 해결

  4. 보안

    • TLS/SSL 설정
    • 사용자 권한 관리
    • vHost 분리
    • 네트워크 보안

✅ 사전 요구사항​

  • 기초편 완료
  • 중급편 완료
  • Docker 사용 경험
  • 리눅스 기본 명령어

🎓 학습 목표​

이 섹션을 완료하면 다음을 할 수 있습니다:

  • 프로덕션 환경 RabbitMQ 클러스터 구축
  • 실시간 모니터링 및 알림 시스템 구축
  • 성능 병목 현상 진단 및 해결
  • 보안 취약점 점검 및 강화
