설치 및 첫 API

FastAPI를 설치하고 첫 번째 API를 만들어봅시다!

🎯 학습 목표​

  • FastAPI 및 Uvicorn 설치
  • 첫 번째 API 애플리케이션 작성
  • 자동 문서 확인 및 테스트

📦 설치​

기본 설치​

# FastAPI 설치
pip install fastapi

# ASGI 서버 (Uvicorn) 설치
pip install "uvicorn[standard]"

개발 환경 설정 (권장)​

# 가상 환경 생성
python -m venv venv

# 가상 환경 활성화 (Windows)
venv\Scripts\activate

# 가상 환경 활성화 (Mac/Linux)
source venv/bin/activate

# 패키지 설치
pip install fastapi uvicorn[standard]
pip install python-multipart  # Form 데이터용
pip install pydantic[email]    # 이메일 검증용

requirements.txt​

# requirements.txt
fastapi==0.109.0
uvicorn[standard]==0.27.0
python-multipart==0.0.6
pydantic[email]==2.5.3
pytest==7.4.3
httpx==0.26.0
pip install -r requirements.txt

🚀 첫 번째 API​

main.py​

# main.py
from fastapi import FastAPI

app = FastAPI()

@app.get("/")
def read_root():
    """루트 엔드포인트"""
    return {"message": "Hello World"}

@app.get("/hello/{name}")
def say_hello(name: str):
    """인사 엔드포인트"""
    return {"message": f"Hello, {name}!"}

실행​

# 개발 모드 (자동 리로드)
uvicorn main:app --reload

# 포트 변경
uvicorn main:app --reload --port 8080

# 호스트 변경 (외부 접근 허용)
uvicorn main:app --reload --host 0.0.0.0

📝 자동 문서​

FastAPI는 자동으로 API 문서를 생성합니다!

Swagger UI​

  • URL: http://127.0.0.1:8000/docs
  • 대화형 API 문서
  • 직접 테스트 가능

ReDoc​

  • URL: http://127.0.0.1:8000/redoc
  • 깔끔한 읽기용 문서

OpenAPI Schema​

  • URL: http://127.0.0.1:8000/openapi.json
  • JSON 형식의 API 스펙

🧪 테스트​

curl로 테스트​

# 루트 엔드포인트
curl http://127.0.0.1:8000/
# {"message":"Hello World"}

# 인사 엔드포인트
curl http://127.0.0.1:8000/hello/FastAPI
# {"message":"Hello, FastAPI!"}

Python requests로 테스트​

# test_api.py
import requests

# 루트 엔드포인트
response = requests.get("http://127.0.0.1:8000/")
print(response.json())
# {'message': 'Hello World'}

# 인사 엔드포인트
response = requests.get("http://127.0.0.1:8000/hello/Alice")
print(response.json())
# {'message': 'Hello, Alice!'}

💡 FastAPI 앱 구조​

from fastapi import FastAPI

# 앱 인스턴스 생성
app = FastAPI(
    title="My API",           # API 제목
    description="API 설명",   # 설명
    version="1.0.0",          # 버전
    docs_url="/docs",         # Swagger UI 경로
    redoc_url="/redoc"        # ReDoc 경로
)

# 경로 동작 (Path Operation)
@app.get("/")               # HTTP GET 메서드
def read_root():            # 경로 동작 함수
    return {"message": "Hello"}

# 다양한 HTTP 메서드
@app.post("/items")
def create_item():
    return {"message": "Item created"}

@app.put("/items/{item_id}")
def update_item(item_id: int):
    return {"item_id": item_id}

@app.delete("/items/{item_id}")
def delete_item(item_id: int):
    return {"item_id": item_id}

🎨 실전 예제​

간단한 카운터 API​

# counter_api.py
from fastapi import FastAPI

app = FastAPI()

# 상태 저장 (실제로는 DB 사용)
counter = {"value": 0}

@app.get("/counter")
def get_counter():
    """현재 카운터 값 조회"""
    return counter

@app.post("/counter/increment")
def increment_counter():
    """카운터 증가"""
    counter["value"] += 1
    return counter

@app.post("/counter/decrement")
def decrement_counter():
    """카운터 감소"""
    counter["value"] -= 1
    return counter

@app.post("/counter/reset")
def reset_counter():
    """카운터 리셋"""
    counter["value"] = 0
    return counter

실행 및 테스트​

# 실행
uvicorn counter_api:app --reload

# 테스트
curl http://127.0.0.1:8000/counter
# {"value": 0}

curl -X POST http://127.0.0.1:8000/counter/increment
# {"value": 1}

curl -X POST http://127.0.0.1:8000/counter/increment
# {"value": 2}

curl -X POST http://127.0.0.1:8000/counter/decrement
# {"value": 1}

curl -X POST http://127.0.0.1:8000/counter/reset
# {"value": 0}

⚠️ 일반적인 문제​

1. 포트 이미 사용 중​

# 다른 포트 사용
uvicorn main:app --reload --port 8001

2. 모듈을 찾을 수 없음​

# 현재 디렉토리 확인
python -c "import sys; print(sys.path)"

# PYTHONPATH 설정
export PYTHONPATH="${PYTHONPATH}:${PWD}"

3. 자동 리로드 안됨​

# --reload 플래그 확인
uvicorn main:app --reload

# 파일 저장 확인
# Ctrl+S로 저장 후 변경 사항 확인

📚 핵심 정리​

  • fastapi: 웹 프레임워크
  • uvicorn: ASGI 서버
  • @app.get("/"): 경로 동작 데코레이터
  • /docs: 자동 API 문서
  • --reload: 자동 리로드 모드
