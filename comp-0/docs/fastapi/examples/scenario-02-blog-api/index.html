시나리오 2: 블로그 API

JWT 인증과 권한 관리가 포함된 블로그 API를 만들어봅시다!

📌 기능​

  • ✅ 사용자 회원가입/로그인 (JWT)
  • ✅ 포스트 CRUD (작성자만 수정/삭제 가능)
  • ✅ 댓글 작성
  • ✅ 좋아요 기능

📁 프로젝트 구조​

blog-api/
├── main.py              # FastAPI 애플리케이션 & API 엔드포인트
├── models.py            # SQLAlchemy 모델 (User, Post, Comment)
├── schemas.py           # Pydantic 스키마
├── auth.py              # JWT 인증 로직
├── database.py          # 데이터베이스 연결
├── utils.py             # 유틸리티 함수 (비밀번호 해싱 등)
├── requirements.txt     # 의존성 패키지
└── blog.db             # SQLite 데이터베이스 파일

📊 시퀀스 다이어그램​

sequenceDiagram
    participant C as Client
    participant API as FastAPI
    participant Auth as JWT Auth
    participant DB as Database

    Note over C,DB: 1. 회원가입
    C->>API: POST /register<br/>{username, password}
    API->>API: Hash password
    API->>DB: INSERT INTO users
    DB-->>API: User created
    API-->>C: 201 Created

    Note over C,DB: 2. 로그인
    C->>API: POST /login<br/>{username, password}
    API->>DB: SELECT user
    DB-->>API: User data
    API->>API: Verify password
    API->>Auth: Create JWT token
    Auth-->>API: Access token
    API-->>C: 200 OK<br/>{access_token}

    Note over C,DB: 3. 포스트 작성 (인증 필요)
    C->>API: POST /posts<br/>Authorization: Bearer {token}
    API->>Auth: Verify token
    Auth-->>API: User info
    API->>DB: INSERT INTO posts
    DB-->>API: Post created
    API-->>C: 201 Created

    Note over C,DB: 4. 포스트 삭제 (작성자만)
    C->>API: DELETE /posts/{id}<br/>Authorization: Bearer {token}
    API->>Auth: Verify token
    Auth-->>API: User info
    API->>DB: SELECT post
    DB-->>API: Post data
    API->>API: Check author_id == user_id
    alt Authorized
        API->>DB: DELETE post
        DB-->>API: Deleted
        API-->>C: 204 No Content
    else Unauthorized
        API-->>C: 403 Forbidden
    end

    Note over C,DB: 5. 좋아요 토글
    C->>API: POST /posts/{id}/like<br/>Authorization: Bearer {token}
    API->>Auth: Verify token
    Auth-->>API: User info
    API->>DB: Check if already liked
    DB-->>API: Like status
    alt Already liked
        API->>DB: Remove like
    else Not liked
        API->>DB: Add like
    end
    DB-->>API: Updated
    API-->>C: 200 OK

📝 핵심 코드​

models.py​

from sqlalchemy import Column, Integer, String, ForeignKey, Table
from sqlalchemy.orm import relationship
from database import Base

post_likes = Table(
    "post_likes",
    Base.metadata,
    Column("user_id", Integer, ForeignKey("users.id")),
    Column("post_id", Integer, ForeignKey("posts.id"))
)

class User(Base):
    __tablename__ = "users"
    id = Column(Integer, primary_key=True)
    username = Column(String, unique=True, index=True)
    email = Column(String, unique=True, index=True)
    hashed_password = Column(String)
    posts = relationship("Post", back_populates="author")

class Post(Base):
    __tablename__ = "posts"
    id = Column(Integer, primary_key=True)
    title = Column(String, index=True)
    content = Column(String)
    author_id = Column(Integer, ForeignKey("users.id"))
    author = relationship("User", back_populates="posts")
    likes = relationship("User", secondary=post_likes)

JWT 인증​

from fastapi import Depends, HTTPException, status
from fastapi.security import OAuth2PasswordBearer
from jose import jwt, JWTError

oauth2_scheme = OAuth2PasswordBearer(tokenUrl="token")

async def get_current_user(token: str = Depends(oauth2_scheme)):
    try:
        payload = jwt.decode(token, SECRET_KEY, algorithms=[ALGORITHM])
        username = payload.get("sub")
        if username is None:
            raise HTTPException(status_code=401)
    except JWTError:
        raise HTTPException(status_code=401)

    user = get_user_by_username(username)
    return user

보호된 엔드포인트​

@app.post("/posts", response_model=PostResponse)
def create_post(
    post: PostCreate,
    current_user: User = Depends(get_current_user),
    db: Session = Depends(get_db)
):
    """포스트 작성 (인증 필요)"""
    db_post = Post(**post.dict(), author_id=current_user.id)
    db.add(db_post)
    db.commit()
    return db_post

@app.delete("/posts/{post_id}")
def delete_post(
    post_id: int,
    current_user: User = Depends(get_current_user),
    db: Session = Depends(get_db)
):
    """포스트 삭제 (작성자만 가능)"""
    post = db.query(Post).filter(Post.id == post_id).first()

    if post.author_id != current_user.id:
        raise HTTPException(status_code=403, detail="Not authorized")

    db.delete(post)
    db.commit()
