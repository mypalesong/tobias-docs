설치 및 설정

PostgreSQL을 설치하고 초기 설정하는 방법을 알아봅니다.

설치 방법​

Windows​

1. 공식 설치 프로그램​

PostgreSQL 공식 사이트에서 설치 프로그램을 다운로드합니다.

1. 설치 프로그램 실행
2. 설치 경로 선택
3. 구성 요소 선택 (PostgreSQL Server, pgAdmin 4, Command Line Tools)
4. 데이터 디렉토리 선택
5. 관리자(postgres) 비밀번호 설정
6. 포트 설정 (기본: 5432)
7. 로케일 설정 (기본값 사용 권장)
8. 설치 완료

2. Chocolatey 사용​

choco install postgresql

macOS​

Homebrew 사용​

# PostgreSQL 설치
brew install postgresql@16

# 서비스 시작
brew services start postgresql@16

Postgres.app 사용​

Postgres.app을 다운로드하여 설치합니다.

Linux (Ubuntu/Debian)​

# APT 저장소 업데이트
sudo apt update

# PostgreSQL 설치
sudo apt install postgresql postgresql-contrib

# 서비스 시작
sudo systemctl start postgresql

# 부팅 시 자동 시작 설정
sudo systemctl enable postgresql

Linux (CentOS/RHEL)​

# PostgreSQL 저장소 추가
sudo yum install -y https://download.postgresql.org/pub/repos/yum/reporpms/EL-8-x86_64/pgdg-redhat-repo-latest.noarch.rpm

# PostgreSQL 설치
sudo yum install -y postgresql16-server

# 데이터베이스 초기화
sudo /usr/pgsql-16/bin/postgresql-16-setup initdb

# 서비스 시작
sudo systemctl start postgresql-16
sudo systemctl enable postgresql-16

설치 확인​

버전 확인​

psql --version

출력 예시:

psql (PostgreSQL) 16.1

PostgreSQL 접속​

# Linux/macOS
sudo -u postgres psql

# Windows (Command Prompt)
psql -U postgres

기본 명령어​

-- 현재 사용자 확인
SELECT current_user;

-- 데이터베이스 목록 보기
\l

-- 종료
\q

초기 설정​

1. 새 사용자 생성​

CREATE USER myuser WITH PASSWORD 'mypassword';

2. 데이터베이스 생성​

CREATE DATABASE mydb OWNER myuser;

3. 권한 부여​

GRANT ALL PRIVILEGES ON DATABASE mydb TO myuser;

4. 원격 접속 허용 (옵션)​

pg_hba.conf 수정​

# 설정 파일 위치 찾기
sudo -u postgres psql -c "SHOW hba_file;"

# 파일 편집
sudo nano /etc/postgresql/16/main/pg_hba.conf

다음 줄 추가:

# IPv4 local connections:
host    all             all             0.0.0.0/0               md5

postgresql.conf 수정​

# 설정 파일 위치 찾기
sudo -u postgres psql -c "SHOW config_file;"

# 파일 편집
sudo nano /etc/postgresql/16/main/postgresql.conf

다음 줄 수정:

listen_addresses = '*'

서비스 재시작​

# Linux
sudo systemctl restart postgresql

# macOS (Homebrew)
brew services restart postgresql@16

# Windows
net stop postgresql-x64-16
net start postgresql-x64-16

GUI 도구​

pgAdmin 4​

가장 인기 있는 PostgreSQL 관리 도구입니다.

  • 다운로드
  • 웹 기반 인터페이스
  • 데이터베이스 관리, 쿼리 실행, 시각화

DBeaver​

범용 데이터베이스 도구입니다.

  • 다운로드
  • 여러 데이터베이스 지원
  • 무료 커뮤니티 에디션 제공

DataGrip​

JetBrains의 전문 데이터베이스 IDE입니다.

  • 다운로드
  • 강력한 SQL 편집기
  • 유료 (30일 무료 체험)

환경 변수 설정 (옵션)​

Linux/macOS​

.bashrc 또는 .zshrc에 추가:

export PATH=/usr/lib/postgresql/16/bin:$PATH
export PGDATA=/var/lib/postgresql/16/main

Windows​

시스템 환경 변수에 추가:

  • Path에 C:\Program Files\PostgreSQL\16\bin 추가

문제 해결​

포트 충돌​

# 사용 중인 포트 확인 (Linux)
sudo lsof -i :5432

# 사용 중인 포트 확인 (Windows)
netstat -ano | findstr :5432

서비스 시작 실패​

# 로그 확인 (Linux)
sudo journalctl -u postgresql

# 로그 위치 (Windows)
C:\Program Files\PostgreSQL\16\data\log

비밀번호 분실​

# pg_hba.conf에서 임시로 trust 인증으로 변경
# 비밀번호 재설정 후 다시 md5로 변경
ALTER USER postgres WITH PASSWORD 'new_password';
