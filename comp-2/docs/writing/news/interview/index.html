3. 인터뷰 기법

좋은 인터뷰는 좋은 기사의 시작입니다. 효과적인 질문과 경청을 통해 핵심 정보를 얻는 방법을 배워보세요.

3.1 인터뷰 준비​

1. 사전 조사​

✅ 해야 할 것:
- 인터뷰이 배경 조사
- 관련 이슈 파악
- 이전 인터뷰 검토
- 전문 용어 학습

❌ 하지 말아야 할 것:
- 준비 없이 만나기
- 뻔한 질문만 준비

2. 질문지 작성​

핵심 질문 5-7개:
1. 반드시 물어야 할 것
2. 논란이 되는 부분
3. 독자가 궁금해하는 것

보조 질문 10-15개:
- 세부 사항
- 구체적인 예시
- 개인적 경험

3. 인터뷰 설정​

✅ 좋은 인터뷰 장소:
- 조용한 카페
- 사무실 (공식적)
- 현장 (맥락 이해에 도움)

❌ 피해야 할 장소:
- 시끄러운 곳
- 인터뷰이가 불편한 곳

3.2 질문 기법​

열린 질문 (Open-ended)​

✅ 좋은 예:
"이 정책의 장단점은 무엇이라고 보시나요?"
"그 결정을 내리게 된 과정을 설명해주시겠어요?"

❌ 나쁜 예:
"이 정책이 좋나요?" (예/아니오로 끝남)

구체적 질문​

✅ 좋은 예:
"정확히 몇 명이 영향을 받나요?"
"언제 시작하실 계획인가요?"

❌ 나쁜 예:
"많은 사람들이 영향을 받나요?"
"곧 시작하시나요?"

팔로우업 질문​

인터뷰이: "우리는 새로운 접근을 시도했습니다."

✅ 팔로우업:
"구체적으로 어떤 점이 새로운가요?"
"이전 방식과 어떻게 다른가요?"
"왜 그 방식을 선택하셨나요?"

어려운 질문하기​

✅ 예의 바르게:
"비판적인 의견도 있는데, 이에 대해서는
어떻게 생각하시나요?"

❌ 공격적으로:
"왜 그렇게 잘못된 결정을 하셨나요?"

3.3 인터뷰 진행​

도입부​

1. 인사 및 소개 (2분)
   - 자기소개
   - 기사 목적 설명
   - 소요 시간 안내

2. 아이스 브레이킹 (3분)
   - 가벼운 대화
   - 편안한 분위기 조성

3. 본론 진입 (5분)
   - 쉬운 질문부터 시작
   - 점차 핵심으로

중간 진행​

✅ 효과적인 기법:

- 적극적 경청
  고개를 끄덕이며 들기
  메모하면서도 눈 맞춤 유지

- 침묵 활용
  답변 후 2-3초 기다리기
  추가 발언 유도

- 재질문
  "조금 더 자세히 설명해주시겠어요?"
  "예를 들어주실 수 있나요?"

- 요약 확인
  "제가 이해한 게 맞나요?"
  "정리하자면..."

마무리​

1. 추가 질문 기회
   "마지막으로 더 하실 말씀이 있으신가요?"

2. 확인 사항
   - 직함, 이름 정확한지 확인
   - 인용 가능한 부분 확인
   - 연락처 재확인

3. 감사 인사
   "귀한 시간 내주셔서 감사합니다"

3.4 인용문 사용​

직접 인용​

✅ 좋은 예:
김철수 장관은 "이번 정책은 국민을 위한 것이며,
반드시 성공시킬 것"이라고 강조했다.

❌ 나쁜 예:
김 장관은 정책이 잘될 거라고 했다. (간접 인용)

부분 인용​

✅ 좋은 예:
그는 이번 조치가 "역사적 전환점"이 될 것이라고
평가했다.

인용문 편집​

✅ 허용:
원본: "음... 그러니까... 이 정책은 좋다고
      생각합니다만..."
편집: "이 정책은 좋다고 생각합니다"

❌ 금지:
의미 왜곡하는 편집
맥락 무시하고 일부만 인용

인용문 출처 표기​

✅ 명확한 출처:
서울대 경제학과 김철수 교수는 "..."라고 말했다.
고용노동부 관계자는 "..."라고 밝혔다.

❌ 모호한 출처:
관계자는 "..."라고 했다.
소식통에 따르면 "..."

3.5 다양한 인터뷰 형식​

1. 대면 인터뷰​

장점:
- 비언어적 표현 관찰
- 라포 형성 용이
- 심도 있는 대화

단점:
- 시간과 장소 제약
- 비용 발생 가능

팁:
- 녹음기 2대 준비
- 백업 배터리
- 메모 병행

2. 전화 인터뷰​

장점:
- 시간 절약
- 즉시 확인 가능
- 비용 절감

단점:
- 비언어적 단서 없음
- 연결 끊김 위험

팁:
- 녹음 동의 받기
- 조용한 곳에서
- 중요 부분 재확인

3. 이메일 인터뷰​

장점:
- 시간 제약 없음
- 정확한 답변
- 문서로 남음

단점:
- 형식적인 답변
- 팔로우업 어려움
- 시간 소요

팁:
- 명확한 질문
- 답변 기한 설정
- 추가 질문 여지

4. 화상 인터뷰​

장점:
- 얼굴 보며 대화
- 원거리 가능
- 녹화 용이

단점:
- 기술적 문제
- 화면 피로

팁:
- 사전 테스트
- 안정적인 인터넷
- 조명 확인

3.6 인터뷰 후 작업​

1. 즉시 정리​

인터뷰 직후 (1시간 이내):
- 전체 흐름 메모
- 인상적인 부분 표시
- 추가 확인 사항 정리

2. 녹취록 작성​

✅ 중요한 부분만:
- 핵심 인용문
- 통계, 숫자
- 논란의 여지 있는 부분

전체 녹취는 시간 낭비일 수 있음

3. 팩트 체크​

확인할 것:
- [ ] 직함, 소속
- [ ] 언급된 숫자
- [ ] 날짜, 장소
- [ ] 인명, 지명

3.7 인터뷰 윤리​

반드시 지킬 것​

✅ 녹음 동의:
"인터뷰 내용을 녹음해도 될까요?"

✅ Off-the-record 존중:
인터뷰이가 "이건 기사에 쓰지 말아주세요"라고
하면 반드시 지키기

✅ 사전 검토 요청 대응:
- 인용문 확인: 가능
- 기사 전체 검토: 원칙적 불가

하지 말아야 할 것​

❌ 유도 질문:
"그렇게 하면 문제가 생기지 않을까요?"
→ "예상되는 효과는 무엇인가요?"

❌ 답변 강요:
"반드시 답해주셔야 합니다"
→ "답변하기 어려우시면 이해합니다"

❌ 약속 위반:
"30분이면 된다" → 1시간
"가벼운 인터뷰" → 어려운 질문

3.8 실전 팁​

항상 백업하세요

녹음기 2대, 메모도 병행하세요. 녹음 실패는 재앙입니다.

침묵을 두려워하지 마세요

질문 후 기다리면 더 좋은 답변을 얻을 수 있습니다.

마지막 질문을 활용하세요

"마지막으로 하실 말씀은?"에서 가장 좋은 인용문이 나올 수 있습니다.
