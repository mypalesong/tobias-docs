n8n 워크플로우 자동화 가이드

n8n으로 업무를 자동화하고 다양한 서비스를 연결하는 방법을 배워보세요.

n8n이란?​

n8n은 오픈소스 워크플로우 자동화 도구입니다. 코드 없이 다양한 앱과 서비스를 연결하여 자동화할 수 있습니다.

✅ 오픈소스 & 자체 호스팅
✅ 350개 이상의 통합 (노드)
✅ 시각적 워크플로우 편집기
✅ JavaScript/Python 표현식 지원
✅ API를 통한 프로그래밍 제어

주요 특징​

1. Fair-code 라이선스​

**개인 & 소규모 팀**: 무료
**대규모 조직**: Enterprise 라이선스

자체 서버에 설치 가능 → 데이터 통제권

2. 노드 기반 자동화​

[Trigger]  →  [Process]  →  [Action]
   ↓             ↓            ↓
 Cron         Filter       Slack
 Webhook      Transform    Email
 Manual       API Call     Database

3. 코드와 No-code의 조화​

왜 n8n인가?​

vs Zapier​

**Zapier**
- SaaS only
- 월 구독료
- 제한된 커스터마이징

**n8n**
- 자체 호스팅 가능
- 무료 (셀프 호스트)
- 완전한 커스터마이징
- 표현식으로 복잡한 로직 구현

vs Make (Integromat)​

**Make**
- 비주얼 중심
- 복잡한 시나리오

**n8n**
- 개발자 친화적
- Git 버전 관리 가능
- API로 워크플로우 관리
- Python/Node.js SDK

사용 사례​

✅ **데이터 동기화**
   - CRM ↔ 마케팅 도구
   - 데이터베이스 ↔ Spreadsheet
   - API ↔ Webhook

✅ **알림 자동화**
   - 에러 발생 → Slack 알림
   - 새 주문 → 이메일 발송
   - 서버 모니터링 → PagerDuty

✅ **데이터 처리**
   - CSV 파싱 및 변환
   - 이미지 리사이징
   - PDF 생성

✅ **비즈니스 프로세스**
   - 승인 워크플로우
   - 고객 온보딩
   - 리드 관리

기본 개념​

워크플로우 (Workflow)​

자동화 프로세스의 전체 흐름입니다.

Example: "신규 고객 환영 이메일"

[새 고객 등록] → [고객 정보 확인] → [환영 이메일 발송] → [CRM 업데이트]

노드 (Node)​

워크플로우의 개별 단계입니다.

**Trigger 노드**: 워크플로우 시작
- Webhook
- Cron (스케줄)
- Manual Trigger

**Action 노드**: 작업 수행
- HTTP Request
- Send Email
- Database Insert

**Logic 노드**: 데이터 처리
- IF
- Switch
- Merge

커넥션 (Connection)​

노드 간 데이터 흐름입니다.

[Node A] ──data──> [Node B]
         ──success──> [Node C]
         ──error──> [Node D]

Python으로 n8n 제어​

n8n API 사용​

import requests

class N8nClient:
    """n8n API 클라이언트"""

    def __init__(self, base_url, api_key):
        self.base_url = base_url
        self.headers = {
            'X-N8N-API-KEY': api_key,
            'Content-Type': 'application/json'
        }

    def get_workflows(self):
        """모든 워크플로우 조회"""
        response = requests.get(
            f'{self.base_url}/workflows',
            headers=self.headers
        )
        return response.json()

    def execute_workflow(self, workflow_id, data=None):
        """워크플로우 실행"""
        response = requests.post(
            f'{self.base_url}/workflows/{workflow_id}/execute',
            headers=self.headers,
            json={'data': data or {}}
        )
        return response.json()

# 사용
client = N8nClient(
    base_url='http://localhost:5678/api/v1',
    api_key='your_api_key'
)

# 워크플로우 실행
result = client.execute_workflow(
    workflow_id='123',
    data={'name': 'John', 'email': 'john@example.com'}
)
