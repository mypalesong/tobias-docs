5. 주요 통합

n8n의 가장 큰 강점은 다양한 서비스와의 통합입니다. 이 섹션에서는 실무에서 자주 사용되는 주요 통합 방법과 실전 예제를 다룹니다.

Slack은 팀 커뮤니케이션과 알림에 가장 많이 사용되는 통합입니다.

Slack 통합 Best Practices
  • Rate Limiting: Slack API는 분당 요청 횟수 제한이 있습니다. 대량 메시지는 배치 처리하세요.
  • Thread 사용: 관련 메시지는 thread로 묶어 채널을 깔끔하게 유지하세요.
  • Webhook vs OAuth2: 간단한 메시지는 Webhook, 파일 업로드나 채널 관리는 OAuth2를 사용하세요.
  • Error Handling: 네트워크 오류에 대비해 재시도 로직을 추가하세요.

Email 통합 Best Practices
  • HTML 템플릿: 반응형 디자인을 사용하여 모바일에서도 잘 보이게 하세요.
  • 첨부파일 크기: 25MB 이하로 유지하세요. 큰 파일은 클라우드 링크를 사용하세요.
  • 발송 속도: Gmail은 하루 500통 제한이 있습니다. 대량 발송은 SendGrid나 AWS SES를 사용하세요.
  • 스팸 방지: SPF, DKIM, DMARC를 설정하여 신뢰도를 높이세요.
