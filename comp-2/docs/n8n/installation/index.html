1. 설치 및 설정

n8n은 다양한 방법으로 설치할 수 있으며, 각 방법은 서로 다른 사용 사례와 환경에 적합합니다. 이 문서는 프로덕션 환경에서 n8n을 안전하고 효율적으로 설치하고 구성하는 방법을 다룹니다.

1.1 설치 방법​

Docker로 설치 (권장)​

Docker는 n8n을 가장 빠르고 안정적으로 실행할 수 있는 방법입니다.

기본 Docker 설치​

# 최신 버전 실행
docker run -it --rm \
  --name n8n \
  -p 5678:5678 \
  -v ~/.n8n:/home/node/.n8n \
  n8nio/n8n

# 특정 버전 실행
docker run -it --rm \
  --name n8n \
  -p 5678:5678 \
  -v ~/.n8n:/home/node/.n8n \
  n8nio/n8n:1.0.0

영구 실행 (백그라운드)​

docker run -d \
  --name n8n \
  --restart unless-stopped \
  -p 5678:5678 \
  -v ~/.n8n:/home/node/.n8n \
  -e WEBHOOK_URL="https://n8n.yourdomain.com/" \
  -e GENERIC_TIMEZONE="Asia/Seoul" \
  n8nio/n8n
팁

Docker 볼륨을 사용하면 데이터 관리가 더 용이합니다:

docker volume create n8n_data
docker run -d --name n8n -p 5678:5678 -v n8n_data:/home/node/.n8n n8nio/n8n

npm으로 설치​

Node.js 환경에서 직접 설치하려면 npm을 사용할 수 있습니다.

# 전역 설치
npm install n8n -g

# n8n 실행
n8n

# 특정 포트에서 실행
n8n start --port 5679

# 터널 모드로 실행 (개발용)
n8n start --tunnel

로컬 개발 환경​

# 프로젝트 디렉토리 생성
mkdir my-n8n-project
cd my-n8n-project

# package.json 생성
npm init -y

# n8n 로컬 설치
npm install n8n

# package.json에 스크립트 추가
{
  "scripts": {
    "start": "n8n",
    "start:tunnel": "n8n start --tunnel"
  }
}
경고

npm 설치 시 Node.js 18.10 이상, npm 8.0 이상이 필요합니다.

node --version  # v18.10.0 이상
npm --version   # 8.0.0 이상

Desktop App 설치​

n8n Desktop은 Windows, macOS, Linux용 독립 실행형 애플리케이션입니다.

  1. n8n 다운로드 페이지에서 OS에 맞는 설치 파일 다운로드
  2. 설치 파일 실행
  3. 설치 마법사 따라가기

장점:

  • 설정이 필요 없음
  • 자동 업데이트
  • 로컬 개발 및 테스트에 이상적

단점:

  • 프로덕션 환경에 부적합
  • 제한된 커스터마이징

n8n Cloud​

관리형 서비스를 원한다면 n8n Cloud를 사용할 수 있습니다.

  1. n8n.cloud에서 계정 생성
  2. 워크스페이스 생성
  3. 즉시 사용 가능

장점:

  • 인프라 관리 불필요
  • 자동 백업 및 업데이트
  • 99.9% 가동 시간 보장

단점:

  • 구독 비용 발생
  • 자체 호스팅 대비 제한된 제어

1.2 환경 변수 설정​

n8n의 동작은 환경 변수를 통해 세밀하게 제어할 수 있습니다.

기본 환경 변수​

# .env 파일 생성
cat > .env << 'EOF'
# 기본 설정
N8N_BASIC_AUTH_ACTIVE=true
N8N_BASIC_AUTH_USER=admin
N8N_BASIC_AUTH_PASSWORD=your_secure_password

# 호스트 설정
N8N_HOST=0.0.0.0
N8N_PORT=5678
N8N_PROTOCOL=https
N8N_PATH=/

# Webhook 설정
WEBHOOK_URL=https://n8n.yourdomain.com/

# 타임존
GENERIC_TIMEZONE=Asia/Seoul

# 실행 설정
EXECUTIONS_TIMEOUT=300
EXECUTIONS_TIMEOUT_MAX=3600
EXECUTIONS_DATA_SAVE_ON_ERROR=all
EXECUTIONS_DATA_SAVE_ON_SUCCESS=all
EXECUTIONS_DATA_SAVE_MANUAL_EXECUTIONS=true

# 워크플로우 설정
WORKFLOWS_DEFAULT_NAME=My Workflow
N8N_DEFAULT_LOCALE=ko

# 로그 설정
N8N_LOG_LEVEL=info
N8N_LOG_OUTPUT=console,file
N8N_LOG_FILE_LOCATION=/home/node/.n8n/logs/

# 외부 훅 설정
EXTERNAL_HOOK_FILES=/home/node/.n8n/custom-hooks.js
EOF

환경 변수 우선순위

환경 변수는 다음 순서로 로드됩니다:

  1. 시스템 환경 변수

Docker에서는 -e 플래그나 --env-file을 사용하여 환경 변수를 전달할 수 있습니다.

API 키 인증​

# Python에서 n8n API 사용 예제
import requests
import os

class N8nClient:
    def __init__(self, base_url, api_key):
        self.base_url = base_url.rstrip('/')
        self.api_key = api_key
        self.headers = {
            'X-N8N-API-KEY': api_key,
            'Content-Type': 'application/json'
        }

    def get_workflows(self):
        """모든 워크플로우 조회"""
        response = requests.get(
            f"{self.base_url}/api/v1/workflows",
            headers=self.headers
        )
        response.raise_for_status()
        return response.json()

    def get_workflow(self, workflow_id):
        """특정 워크플로우 조회"""
        response = requests.get(
            f"{self.base_url}/api/v1/workflows/{workflow_id}",
            headers=self.headers
        )
        response.raise_for_status()
        return response.json()

    def activate_workflow(self, workflow_id, active=True):
        """워크플로우 활성화/비활성화"""
        response = requests.patch(
            f"{self.base_url}/api/v1/workflows/{workflow_id}",
            headers=self.headers,
            json={'active': active}
        )
        response.raise_for_status()
        return response.json()

    def execute_workflow(self, workflow_id, data=None):
        """워크플로우 수동 실행"""
        response = requests.post(
            f"{self.base_url}/api/v1/workflows/{workflow_id}/execute",
            headers=self.headers,
            json=data or {}
        )
        response.raise_for_status()
        return response.json()

# 사용 예제
if __name__ == "__main__":
    client = N8nClient(
        base_url="https://n8n.yourdomain.com",
        api_key=os.getenv("N8N_API_KEY")
    )

    # 워크플로우 목록 조회
    workflows = client.get_workflows()
    print(f"Total workflows: {len(workflows['data'])}")

    # 첫 번째 워크플로우 활성화
    if workflows['data']:
        workflow_id = workflows['data'][0]['id']
        result = client.activate_workflow(workflow_id, active=True)
        print(f"Workflow {workflow_id} activated: {result['active']}")

1.5 Reverse Proxy 설정​

Nginx 설정​

# /etc/nginx/sites-available/n8n
upstream n8n {
    server 127.0.0.1:5678;
}

server {
    listen 80;
    server_name n8n.yourdomain.com;

    # HTTP를 HTTPS로 리다이렉트
    return 301 https://$server_name$request_uri;
}

server {
    listen 443 ssl http2;
    server_name n8n.yourdomain.com;

    # SSL 인증서
    ssl_certificate /etc/letsencrypt/live/n8n.yourdomain.com/fullchain.pem;
    ssl_certificate_key /etc/letsencrypt/live/n8n.yourdomain.com/privkey.pem;

    # SSL 설정
    ssl_protocols TLSv1.2 TLSv1.3;
    ssl_ciphers HIGH:!aNULL:!MD5;
    ssl_prefer_server_ciphers on;
    ssl_session_cache shared:SSL:10m;
    ssl_session_timeout 10m;

    # HSTS 헤더
    add_header Strict-Transport-Security "max-age=31536000; includeSubDomains" always;

    # 보안 헤더
    add_header X-Frame-Options "SAMEORIGIN" always;
    add_header X-Content-Type-Options "nosniff" always;
    add_header X-XSS-Protection "1; mode=block" always;
    add_header Referrer-Policy "no-referrer-when-downgrade" always;

    # 로그
    access_log /var/log/nginx/n8n_access.log;
    error_log /var/log/nginx/n8n_error.log;

    # 클라이언트 업로드 크기 제한
    client_max_body_size 50M;

    location / {
        proxy_pass http://n8n;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_set_header X-Forwarded-Host $host;
        proxy_set_header X-Forwarded-Port $server_port;

        # WebSocket 지원
        proxy_buffering off;
        proxy_read_timeout 86400;
        proxy_connect_timeout 86400;
        proxy_send_timeout 86400;
    }
}

Traefik 설정​

# docker-compose.yml
version: '3.8'

services:
  traefik:
    image: traefik:v2.10
    container_name: traefik
    restart: unless-stopped
    command:
      - "--api.insecure=false"
      - "--providers.docker=true"
      - "--providers.docker.exposedbydefault=false"
      - "--entrypoints.web.address=:80"
      - "--entrypoints.websecure.address=:443"
      - "--certificatesresolvers.letsencrypt.acme.httpchallenge=true"
      - "--certificatesresolvers.letsencrypt.acme.httpchallenge.entrypoint=web"
      - "--certificatesresolvers.letsencrypt.acme.email=admin@yourdomain.com"
      - "--certificatesresolvers.letsencrypt.acme.storage=/letsencrypt/acme.json"
    ports:
      - "80:80"
      - "443:443"
    volumes:
      - /var/run/docker.sock:/var/run/docker.sock:ro
      - traefik_letsencrypt:/letsencrypt
    labels:
      # 글로벌 HTTP to HTTPS 리다이렉트
      - "traefik.http.middlewares.redirect-to-https.redirectscheme.scheme=https"
      - "traefik.http.routers.redirs.rule=hostregexp(`{host:.+}`)"
      - "traefik.http.routers.redirs.entrypoints=web"
      - "traefik.http.routers.redirs.middlewares=redirect-to-https"

  n8n:
    image: n8nio/n8n:latest
    container_name: n8n
    restart: unless-stopped
    environment:
      - N8N_HOST=n8n.yourdomain.com
      - N8N_PORT=5678
      - N8N_PROTOCOL=https
      - WEBHOOK_URL=https://n8n.yourdomain.com/
      - GENERIC_TIMEZONE=Asia/Seoul
    volumes:
      - n8n_data:/home/node/.n8n
    labels:
      - "traefik.enable=true"
      # HTTP 라우터
      - "traefik.http.routers.n8n.rule=Host(`n8n.yourdomain.com`)"
      - "traefik.http.routers.n8n.entrypoints=websecure"
      - "traefik.http.routers.n8n.tls=true"
      - "traefik.http.routers.n8n.tls.certresolver=letsencrypt"
      # 서비스 설정

Python 백업 스크립트​

#!/usr/bin/env python3
# n8n_backup.py

import os
import sys
import subprocess
import shutil
from datetime import datetime, timedelta
from pathlib import Path

class N8nBackup:
    def __init__(self, backup_dir, n8n_dir=None, retention_days=7):
        self.backup_dir = Path(backup_dir)
        self.n8n_dir = Path(n8n_dir or os.path.expanduser("~/.n8n"))
        self.retention_days = retention_days
        self.timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")

        # 백업 디렉토리 생성
        self.backup_dir.mkdir(parents=True, exist_ok=True)

    def backup_sqlite(self):
        """SQLite 데이터베이스 백업"""
        backup_file = self.backup_dir / f"n8n_backup_{self.timestamp}.tar.gz"

        # n8n 컨테이너 중지
        subprocess.run(["docker", "stop", "n8n"], check=False)

        try:
            # tar로 압축
            subprocess.run([
                "tar", "-czf", str(backup_file),
                "-C", str(self.n8n_dir), "."
            ], check=True)

            print(f"백업 완료: {backup_file}")
            return backup_file
        finally:
            # n8n 컨테이너 재시작
            subprocess.run(["docker", "start", "n8n"], check=False)

    def backup_postgres(self, host, port, database, user, password):
        """PostgreSQL 데이터베이스 백업"""
        backup_file = self.backup_dir / f"n8n_db_{self.timestamp}.sql.gz"

        env = os.environ.copy()
        env['PGPASSWORD'] = password

        # pg_dump 실행
        with open(backup_file, 'wb') as f:
            pg_dump = subprocess.Popen([
                "pg_dump",
                "-h", host,
                "-p", str(port),
                "-U", user,
                database
            ], stdout=subprocess.PIPE, env=env)

            gzip = subprocess.Popen(
                ["gzip"],
                stdin=pg_dump.stdout,
                stdout=f
            )

            pg_dump.stdout.close()
            gzip.communicate()

        print(f"데이터베이스 백업 완료: {backup_file}")

        # 파일 백업
        files_backup = self.backup_dir / f"n8n_files_{self.timestamp}.tar.gz"
        subprocess.run([
            "tar", "-czf", str(files_backup),
            "-C", str(self.n8n_dir),
            "custom/", "nodes/",
            "--exclude=database.sqlite"
        ], check=True)

        print(f"파일 백업 완료: {files_backup}")

        return backup_file, files_backup

    def cleanup_old_backups(self):
        """오래된 백업 파일 삭제"""
        cutoff_date = datetime.now() - timedelta(days=self.retention_days)

        for backup_file in self.backup_dir.glob("n8n_*"):
            if backup_file.stat().st_mtime < cutoff_date.timestamp():
                backup_file.unlink()
                print(f"삭제됨: {backup_file}")

    def restore_sqlite(self, backup_file):
        """SQLite 백업 복구"""
        backup_path = Path(backup_file)

        if not backup_path.exists():
            raise FileNotFoundError(f"백업 파일을 찾을 수 없음: {backup_file}")

        # n8n 중지
        subprocess.run(["docker", "stop", "n8n"], check=False)

        try:
            # 기존 데이터 백업
            if self.n8n_dir.exists():
                backup_old = self.n8n_dir.parent / f".n8n.old.{int(datetime.now().timestamp())}"
                shutil.move(str(self.n8n_dir), str(backup_old))
                print(f"기존 데이터 백업: {backup_old}")

            # 복구
            self.n8n_dir.mkdir(parents=True, exist_ok=True)
            subprocess.run([
                "tar", "-xzf", str(backup_path),
                "-C", str(self.n8n_dir)
            ], check=True)

            print(f"복구 완료: {backup_file}")
        finally:
            # n8n 재시작
            subprocess.run(["docker", "start", "n8n"], check=False)

# 사용 예제
if __name__ == "__main__":
    backup = N8nBackup(backup_dir="/backup/n8n")

    # SQLite 백업
    # backup.backup_sqlite()

    # PostgreSQL 백업
    backup.backup_postgres(
        host="localhost",
        port=5432,
        database="n8n",
        user="n8n_user",
        password="secure_password"
    )

    # 오래된 백업 정리
    backup.cleanup_old_backups()

진단 정보 수집​

#!/usr/bin/env python3
# n8n_diagnostics.py

import subprocess
import json
from datetime import datetime

def collect_diagnostics():
    """n8n 진단 정보 수집"""

    diagnostics = {
        "timestamp": datetime.now().isoformat(),
        "system": {},
        "docker": {},
        "n8n": {},
        "database": {},
        "logs": {}
    }

    # 시스템 정보
    try:
        diagnostics["system"]["os"] = subprocess.check_output(
            ["uname", "-a"], text=True
        ).strip()

        diagnostics["system"]["memory"] = subprocess.check_output(
            ["free", "-h"], text=True
        )

        diagnostics["system"]["disk"] = subprocess.check_output(
            ["df", "-h"], text=True
        )
    except Exception as e:
        diagnostics["system"]["error"] = str(e)

    # Docker 정보
    try:
        diagnostics["docker"]["version"] = subprocess.check_output(
            ["docker", "--version"], text=True
        ).strip()

        diagnostics["docker"]["containers"] = subprocess.check_output(
            ["docker", "ps", "-a"], text=True
        )

        diagnostics["docker"]["stats"] = subprocess.check_output(
            ["docker", "stats", "--no-stream"], text=True
        )
    except Exception as e:
        diagnostics["docker"]["error"] = str(e)

    # n8n 정보
    try:
        diagnostics["n8n"]["version"] = subprocess.check_output(
            ["docker", "exec", "n8n", "n8n", "--version"], text=True
        ).strip()

        diagnostics["n8n"]["env"] = subprocess.check_output(
            ["docker", "exec", "n8n", "env"], text=True
        )
    except Exception as e:
        diagnostics["n8n"]["error"] = str(e)

    # 데이터베이스 정보
    try:
        db_info = subprocess.check_output([
            "docker", "exec", "n8n-postgres",
            "psql", "-U", "n8n_user", "-d", "n8n",
            "-c", "SELECT version();"
        ], text=True)
        diagnostics["database"]["version"] = db_info

        db_size = subprocess.check_output([
            "docker", "exec", "n8n-postgres",
            "psql", "-U", "n8n_user", "-d", "n8n",
            "-c", "SELECT pg_size_pretty(pg_database_size('n8n'));"
        ], text=True)
        diagnostics["database"]["size"] = db_size
    except Exception as e:
        diagnostics["database"]["error"] = str(e)

    # 로그 (최근 100줄)
    try:
        diagnostics["logs"]["n8n"] = subprocess.check_output(
            ["docker", "logs", "--tail", "100", "n8n"], text=True
        )
    except Exception as e:
        diagnostics["logs"]["error"] = str(e)

    # 결과 저장
    output_file = f"n8n_diagnostics_{datetime.now().strftime('%Y%m%d_%H%M%S')}.json"
    with open(output_file, 'w') as f:
        json.dump(diagnostics, f, indent=2)

    print(f"진단 정보 저장됨: {output_file}")
    return output_file

if __name__ == "__main__":
    collect_diagnostics()

과제 3: 자동 백업 시스템 구축​

매일 자동으로 n8n 데이터를 백업하는 시스템을 만드세요.

요구사항:

  • PostgreSQL 데이터베이스 백업
  • 파일 시스템 백업
  • 7일 이상 된 백업 자동 삭제
  • Cron으로 매일 실행
솔루션 보기
#!/usr/bin/env python3
# /usr/local/bin/n8n_backup.py

import os
import subprocess
from datetime import datetime, timedelta
from pathlib import Path

BACKUP_DIR = Path("/backup/n8n")
RETENTION_DAYS = 7

def backup_database():
    """PostgreSQL 백업"""
    timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
    backup_file = BACKUP_DIR / f"db_{timestamp}.sql.gz"

    BACKUP_DIR.mkdir(parents=True, exist_ok=True)

    env = os.environ.copy()
    env['PGPASSWORD'] = 'secure_password_123'

    with open(backup_file, 'wb') as f:
        pg_dump = subprocess.Popen([
            "docker", "exec", "n8n_postgres",
            "pg_dump", "-U", "n8n_user", "n8n"
        ], stdout=subprocess.PIPE, env=env)

        subprocess.run(["gzip"], stdin=pg_dump.stdout, stdout=f)

    print(f"데이터베이스 백업 완료: {backup_file}")
    return backup_file

def backup_files():
    """파일 시스템 백업"""
    timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
    backup_file = BACKUP_DIR / f"files_{timestamp}.tar.gz"

    subprocess.run([
        "docker", "run", "--rm",
        "-v", "n8n_data:/data",
        "-v", f"{BACKUP_DIR}:/backup",
        "alpine",
        "tar", "-czf", f"/backup/files_{timestamp}.tar.gz",
        "-C", "/data", "."
    ])

    print(f"파일 백업 완료: {backup_file}")
    return backup_file

def cleanup_old_backups():
    """오래된 백업 삭제"""
    cutoff = datetime.now() - timedelta(days=RETENTION_DAYS)

    for backup_file in BACKUP_DIR.glob("*"):
        if backup_file.stat().st_mtime < cutoff.timestamp():
            backup_file.unlink()
            print(f"삭제됨: {backup_file}")

if __name__ == "__main__":
    backup_database()
    backup_files()
    cleanup_old_backups()
# 실행 권한 부여
chmod +x /usr/local/bin/n8n_backup.py

# Cron 설정
crontab -e

# 매일 새벽 2시 실행
0 2 * * * /usr/local/bin/n8n_backup.py >> /var/log/n8n_backup.log 2>&1
