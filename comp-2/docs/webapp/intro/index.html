웹 애플리케이션 개발 가이드

Vue 3와 FastAPI로 현대적인 웹 애플리케이션을 구축하는 방법을 배워보세요.

웹 애플리케이션이란?​

웹 애플리케이션은 사용자가 웹 브라우저를 통해 접근하고 사용할 수 있는 소프트웨어 애플리케이션입니다. 설치가 필요 없고, 어디서나 접근 가능하며, 실시간 업데이트가 가능합니다.

✅ 크로스 플랫폼 - 브라우저만 있으면 사용 가능
✅ 즉시 배포 - 사용자가 새로 다운로드할 필요 없음
✅ 중앙 집중식 관리 - 한 곳에서 업데이트
✅ 확장성 - 사용자 증가에 따라 스케일링
✅ 협업 가능 - 여러 사용자가 동시 사용

기술 스택​

Frontend: Vue 3​

Vue 3는 사용자 인터페이스를 구축하기 위한 프로그레시브 JavaScript 프레임워크입니다.

**주요 특징:**
- Composition API - 더 나은 로직 재사용
- TypeScript 완전 지원
- Vite - 빠른 개발 서버
- 반응형 시스템 - 자동 UI 업데이트
- 단일 파일 컴포넌트 (SFC)

사용 사례:

  • SPA (Single Page Application)
  • 관리자 대시보드
  • 실시간 데이터 시각화
  • E-commerce 프론트엔드

Backend: FastAPI​

FastAPI는 Python 3.7+의 타입 힌트를 기반으로 한 현대적이고 빠른 웹 프레임워크입니다.

**주요 특징:**
- 빠른 성능 - Node.js 및 Go에 필적
- 자동 API 문서 생성 (Swagger UI)
- Pydantic을 통한 데이터 검증
- async/await 지원
- 타입 안정성

사용 사례:

  • RESTful API
  • GraphQL API
  • 마이크로서비스
  • ML 모델 서빙

Backend: Hello World API​

# app/main.py
from fastapi import FastAPI
from fastapi.middleware.cors import CORSMiddleware

app = FastAPI(title="My Web App API")

# CORS 설정
app.add_middleware(
    CORSMiddleware,
    allow_origins=["http://localhost:5173"],  # Vue 개발 서버
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)

@app.get("/")
def read_root():
    return {"message": "Hello from FastAPI"}

@app.get("/api/items/{item_id}")
def read_item(item_id: int, q: str = None):
    return {"item_id": item_id, "q": q}

Frontend: API 호출​

<!-- src/App.vue -->
<script setup lang="ts">
import { ref, onMounted } from 'vue'
import axios from 'axios'

const message = ref('')
const loading = ref(false)

const fetchData = async () => {
  loading.value = true
  try {
    const response = await axios.get('http://localhost:8000/')
    message.value = response.data.message
  } catch (error) {
    console.error('Error:', error)
  } finally {
    loading.value = false
  }
}

onMounted(() => {
  fetchData()
})
</script>

<template>
  <div>
    <h1>Vue 3 + FastAPI</h1>
    <p v-if="loading">Loading...</p>
    <p v-else>{{ message }}</p>
    <button @click="fetchData">Refresh</button>
  </div>
</template>

일반적인 함정​

1. CORS 문제​

# ❌ 나쁜 예: 모든 origin 허용 (프로덕션)
app.add_middleware(
    CORSMiddleware,
    allow_origins=["*"],  # 보안 위험!
)

# ✅ 좋은 예: 특정 origin만 허용
app.add_middleware(
    CORSMiddleware,
    allow_origins=["https://myapp.com"],
)

2. 비밀번호 저장​

# ❌ 나쁜 예: 평문 저장
user.password = password

# ✅ 좋은 예: 해시 저장
from passlib.context import CryptContext
pwd_context = CryptContext(schemes=["bcrypt"])
user.password = pwd_context.hash(password)

3. SQL Injection​

# ❌ 나쁜 예: 문자열 포매팅
query = f"SELECT * FROM users WHERE email = '{email}'"

# ✅ 좋은 예: ORM 사용
user = db.query(User).filter(User.email == email).first()

    logger.info(f"Response: {response.status_code}")
