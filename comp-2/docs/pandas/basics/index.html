1. 기본 개념

Pandas는 Python에서 데이터 분석을 위한 가장 강력하고 널리 사용되는 라이브러리입니다. 이 문서에서는 Pandas의 핵심 개념과 기본적인 사용법을 다룹니다.

1.1 Pandas 시작하기​

먼저 Pandas를 설치하고 임포트하는 방법을 알아봅시다.

# Pandas 설치
pip install pandas
import pandas as pd
import numpy as np

# 버전 확인
print(f"Pandas 버전: {pd.__version__}")
팁

pd는 Pandas의 표준 별칭입니다. 대부분의 Pandas 코드에서 이 별칭을 사용하므로, 이 관례를 따르는 것이 좋습니다.

1.2 Series - 1차원 데이터 구조​

Series는 Pandas의 기본 데이터 구조 중 하나로, 인덱스가 있는 1차원 배열입니다.

Series 생성​

# 리스트로 Series 생성
s1 = pd.Series([10, 20, 30, 40, 50])
print(s1)
# 출력:
# 0    10
# 1    20
# 2    30
# 3    40
# 4    50
# dtype: int64

# 인덱스를 지정하여 Series 생성
s2 = pd.Series([10, 20, 30, 40, 50],
               index=['a', 'b', 'c', 'd', 'e'])
print(s2)
# 출력:
# a    10
# b    20
# c    30
# d    40
# e    50
# dtype: int64

# 딕셔너리로 Series 생성
data_dict = {'서울': 9720846, '부산': 3404423, '인천': 2947217, '대구': 2427954}
s3 = pd.Series(data_dict)
print(s3)
# 출력:
# 서울    9720846
# 부산    3404423
# 인천    2947217
# 대구    2427954
# dtype: int64

# NumPy 배열로 Series 생성
s4 = pd.Series(np.random.randn(5))
print(s4)

Series 조작​

# Series 값과 인덱스 접근
print(s2.values)  # array([10, 20, 30, 40, 50])
print(s2.index)   # Index(['a', 'b', 'c', 'd', 'e'], dtype='object')

# 인덱스로 값 선택
print(s2['a'])    # 10
print(s2[['a', 'c', 'e']])  # 여러 값 선택

# 위치 기반 선택
print(s2[0])      # 10
print(s2[1:4])    # b, c, d 선택

# 조건부 선택
print(s2[s2 > 25])  # 25보다 큰 값만 선택

# Series 연산
print(s2 + 10)    # 모든 값에 10 더하기
print(s2 * 2)     # 모든 값에 2 곱하기
print(s2 ** 2)    # 모든 값 제곱
팁

Series는 NumPy 배열과 비슷하지만, 인덱스를 가지고 있어 더 강력합니다. 인덱스는 숫자뿐만 아니라 문자열, 날짜 등 다양한 타입이 될 수 있습니다.

1.3 DataFrame - 2차원 데이터 구조​

DataFrame은 행과 열로 구성된 2차원 테이블 형태의 데이터 구조입니다. 엑셀 스프레드시트나 SQL 테이블과 유사합니다.

딕셔너리로 DataFrame 생성​

    '이름': ['김철수', '이영희', '박민수', '정수진', '최동욱'],

    '점수': [85, 92, 78, 88, 95]

df = pd.DataFrame(data)
print(df)
# 출력:
#     이름  나이  도시  점수
# 0  김철수  25  서울  85

# 3  정수진  28  대구  88

# 인덱스 지정
df_with_index = pd.DataFrame(data, index=['A', 'B', 'C', 'D', 'E'])

1.5 인덱싱과 슬라이싱​

DataFrame에서 데이터를 선택하는 방법은 크게 loc(라벨 기반)과 iloc(위치 기반)가 있습니다.

컬럼 선택​

# 단일 컬럼 선택 (Series 반환)
print(df['이름'])
print(df.이름)  # 점(.) 표기법 (컬럼명에 공백이 없을 때만 사용 가능)

# 여러 컬럼 선택 (DataFrame 반환)
print(df[['이름', '점수']])

# 첫 번째 컬럼 선택
print(df.iloc[:, 0])

loc - 라벨 기반 인덱싱​

# 인덱스 라벨로 행 선택
print(df.loc[0])        # 0번 행 선택 (Series 반환)
print(df.loc[[0, 2, 4]])  # 여러 행 선택 (DataFrame 반환)

# 행과 열 동시 선택
print(df.loc[0, '이름'])           # 0번 행의 '이름' 값
print(df.loc[0:2, '이름':'점수'])  # 0~2번 행, '이름'~'점수' 컬럼 (끝 포함!)
print(df.loc[:, ['이름', '점수']])  # 모든 행, 특정 컬럼들

# 조건으로 행 선택
print(df.loc[df['나이'] > 30])
print(df.loc[df['도시'] == '서울'])
print(df.loc[(df['나이'] > 28) & (df['점수'] >= 85)])

iloc - 위치 기반 인덱싱​

# 위치(정수)로 행 선택
print(df.iloc[0])       # 첫 번째 행
print(df.iloc[-1])      # 마지막 행
print(df.iloc[0:3])     # 0~2번 행 (끝 제외!)

# 행과 열 동시 선택
print(df.iloc[0, 1])           # 첫 번째 행, 두 번째 컬럼
print(df.iloc[0:3, 0:2])       # 0~2번 행, 0~1번 컬럼
print(df.iloc[:, [0, 3]])      # 모든 행, 첫 번째와 네 번째 컬럼
print(df.iloc[1:5, 1:4])       # 1~4번 행, 1~3번 컬럼

# 리스트로 선택
print(df.iloc[[0, 2, 4], [0, 3]])  # 특정 행과 열만 선택
팁

loc vs iloc 차이점:

  • loc: 라벨(이름) 기반, 슬라이싱 시 끝 포함
  • iloc: 위치(정수) 기반, 슬라이싱 시 끝 제외

일반적으로 조건 필터링은 loc, 순수 위치 기반 선택은 iloc를 사용합니다.

1.6 데이터 선택 및 필터링​

다양한 조건으로 데이터를 필터링하는 방법을 알아봅시다.

# 단일 조건 필터링
high_scorers = df[df['점수'] > 85]
print(high_scorers)

seoul_people = df[df['도시'] == '서울']
print(seoul_people)

# 여러 조건 필터링 (AND: &)
result = df[(df['나이'] > 28) & (df['점수'] >= 85)]
print(result)

# 여러 조건 필터링 (OR: |)
result = df[(df['도시'] == '서울') | (df['도시'] == '부산')]
print(result)

# NOT 조건 (~)
not_seoul = df[~(df['도시'] == '서울')]
print(not_seoul)

# isin() 메서드 사용
cities = ['서울', '부산', '인천']
result = df[df['도시'].isin(cities)]
print(result)

# 문자열 메서드 사용
# 이름에 '김'이 포함된 사람
kim_people = df[df['이름'].str.contains('김')]
print(kim_people)

# 이름이 '이'로 시작하는 사람
lee_people = df[df['이름'].str.startswith('이')]
print(lee_people)

# 범위 조건
age_range = df[df['나이'].between(27, 30)]
print(age_range)

# query() 메서드 (SQL 스타일)
result = df.query('나이 > 28 and 점수 >= 85')
print(result)

result = df.query('도시 in ["서울", "부산"]')
print(result)
팁

복잡한 조건을 사용할 때는 각 조건을 괄호 ()로 묶어야 합니다. 그렇지 않으면 연산자 우선순위 때문에 예상과 다른 결과가 나올 수 있습니다.

1.7 컬럼 추가, 삭제, 수정​

DataFrame의 구조를 변경하는 방법을 알아봅시다.

컬럼 추가​

# 새로운 컬럼 추가 (고정값)
df['국적'] = '한국'
print(df)

# 계산된 값으로 컬럼 추가
df['점수_100점'] = df['점수']  # 복사
df['점수_200점'] = df['점수'] * 2  # 계산

# 조건부 컬럼 추가
df['등급'] = df['점수'].apply(lambda x: 'A' if x >= 90 else ('B' if x >= 80 else 'C'))
print(df)

# numpy.where() 사용
df['성인'] = np.where(df['나이'] >= 30, '성인', '청년')
print(df)

# 여러 조건 (np.select 사용)
conditions = [
    df['점수'] >= 90,
    df['점수'] >= 80,
    df['점수'] >= 70
]
choices = ['A', 'B', 'C']
df['학점'] = np.select(conditions, choices, default='D')
print(df)

# insert() 메서드로 특정 위치에 추가
df.insert(2, '성별', ['남', '여', '남', '여', '남', '남', '여'])
print(df)

컬럼 수정​

# 기존 컬럼 값 수정
df['점수'] = df['점수'] + 5  # 모든 점수에 5점 추가

# 조건부 수정
df.loc[df['나이'] > 30, '점수'] = df['점수'] + 10

# apply() 함수로 수정
df['이름'] = df['이름'].apply(lambda x: x + '님')

# map() 함수로 수정 (딕셔너리 매핑)
city_code = {'서울': 'SEL', '부산': 'PUS', '인천': 'ICN', '대구': 'DAE', '광주': 'GWJ'}
df['도시코드'] = df['도시'].map(city_code)

# replace() 메서드
df['도시'] = df['도시'].replace({'서울': 'Seoul', '부산': 'Busan'})

컬럼 삭제​

# drop() 메서드 (원본 유지)
df_dropped = df.drop('국적', axis=1)
df_dropped = df.drop(['국적', '성별'], axis=1)

# drop() 메서드 (원본 수정)
df.drop('국적', axis=1, inplace=True)

# del 키워드 (원본 수정)
del df['성별']

# pop() 메서드 (컬럼 제거 후 반환)
removed_column = df.pop('도시코드')
print(removed_column)

컬럼명 변경​

# rename() 메서드
df = df.rename(columns={'이름': 'name', '나이': 'age'})

# 여러 컬럼 한 번에 변경
df = df.rename(columns={
    'name': '이름',
    'age': '나이',
    '도시': '거주지'
})

# 모든 컬럼명 변경
df.columns = ['col1', 'col2', 'col3', 'col4', 'col5']

# 컬럼명 소문자로 변경
df.columns = df.columns.str.lower()
팁

inplace=True 옵션은 원본 DataFrame을 직접 수정합니다. 이 옵션을 사용하지 않으면 새로운 DataFrame을 반환하므로, 결과를 변수에 할당해야 합니다.

1.8 기본 통계 함수​

Pandas는 데이터 분석을 위한 다양한 통계 함수를 제공합니다.

# 샘플 데이터
df = pd.DataFrame({
    '수학': [85, 92, 78, 88, 95, 82, 90],
    '영어': [88, 85, 92, 78, 90, 85, 88],
    '과학': [90, 88, 85, 92, 88, 90, 85]
})

# 기본 통계량
print(df.mean())        # 평균
print(df.median())      # 중앙값
print(df.mode())        # 최빈값
print(df.std())         # 표준편차
print(df.var())         # 분산
print(df.min())         # 최솟값
print(df.max())         # 최댓값
print(df.sum())         # 합계
print(df.count())       # 개수 (결측값 제외)

# 특정 컬럼 통계
print(df['수학'].mean())   # 수학 평균
print(df['영어'].sum())    # 영어 합계

# 행 방향 통계 (axis=1)
df['총점'] = df.sum(axis=1)
df['평균'] = df[['수학', '영어', '과학']].mean(axis=1)
print(df)

# 열 방향 통계 (axis=0, 기본값)
subject_avg = df[['수학', '영어', '과학']].mean(axis=0)
print(subject_avg)

# 누적 통계
print(df['수학'].cumsum())   # 누적 합
print(df['수학'].cumprod())  # 누적 곱
print(df['수학'].cummax())   # 누적 최댓값
print(df['수학'].cummin())   # 누적 최솟값

# 분위수
print(df['수학'].quantile(0.25))  # 1사분위수

import pandas as pd
import numpy as np

# 실습용 데이터 생성
np.random.seed(42)

students = pd.DataFrame({
    '학번': range(1001, 1021),
    '이름': [
        '김철수', '이영희', '박민수', '정수진', '최동욱',
        '강지훈', '윤서연', '장민재', '한소희', '오준석',
        '신예린', '조성훈', '배수지', '임재현', '홍지아',
        '서현우', '권나연', '노태양', '문가은', '황민호'
    ],
    '성별': ['남', '여', '남', '여', '남', '남', '여', '남', '여', '남',
             '여', '남', '여', '남', '여', '남', '여', '남', '여', '남'],
    '학년': np.random.choice([1, 2, 3, 4], size=20),
    '전공': np.random.choice(['컴퓨터공학', '경영학', '전자공학', '수학'], size=20),
    '중간고사': np.random.randint(60, 100, size=20),
    '기말고사': np.random.randint(60, 100, size=20),
    '출석점수': np.random.randint(15, 21, size=20)
})

# 1. 데이터 탐색
print("=== 데이터 기본 정보 ===")
print(students.head())
print("\n", students.info())
print("\n", students.describe())

# 2. 총점 및 평균 계산
students['총점'] = (students['중간고사'] * 0.4 +
                    students['기말고사'] * 0.4 +
                    students['출석점수'] * 0.2)
students['평균'] = students['총점']

# 3. 학점 부여
def assign_grade(score):
    if score >= 90:
        return 'A'
    elif score >= 80:
        return 'B'
    elif score >= 70:
        return 'C'
    elif score >= 60:
        return 'D'
    else:
        return 'F'

students['학점'] = students['총점'].apply(assign_grade)

# 4. 데이터 분석
print("\n=== 전공별 평균 점수 ===")
major_avg = students.groupby('전공')['총점'].mean().sort_values(ascending=False)
print(major_avg)

print("\n=== 학년별 평균 점수 ===")
year_avg = students.groupby('학년')['총점'].mean().sort_values()
print(year_avg)

print("\n=== 성별 평균 점수 ===")
gender_avg = students.groupby('성별')['총점'].mean()
print(gender_avg)

print("\n=== 학점 분포 ===")
print(students['학점'].value_counts().sort_index())

# 5. 우수 학생 선발 (총점 상위 30%)
top_30_percent = students.nlargest(int(len(students) * 0.3), '총점')
print("\n=== 우수 학생 (상위 30%) ===")
print(top_30_percent[['학번', '이름', '전공', '총점', '학점']])

# 6. 조건 필터링
print("\n=== 컴퓨터공학 전공 A학점 학생 ===")
cs_a_students = students[(students['전공'] == '컴퓨터공학') &
                         (students['학점'] == 'A')]
print(cs_a_students[['학번', '이름', '총점', '학점']])

# 7. 데이터 저장
students.to_csv('students_result.csv', index=False, encoding='utf-8-sig')
print("\n결과가 'students_result.csv'에 저장되었습니다.")
