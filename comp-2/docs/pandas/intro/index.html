pandas 데이터 분석 가이드

pandas로 데이터를 효율적으로 분석하고 조작하는 방법을 배워보세요.

pandas란?​

pandas는 Python의 강력한 데이터 분석 및 조작 라이브러리입니다. 표 형식 데이터를 쉽고 직관적으로 처리할 수 있습니다.

✅ DataFrame - 엑셀과 같은 표 형식 데이터 구조
✅ 다양한 파일 형식 지원 (CSV, Excel, SQL, JSON 등)
✅ 강력한 데이터 정제 및 변환 기능
✅ 그룹화, 피벗, 조인 등 SQL과 유사한 연산
✅ 시계열 데이터 처리

주요 특징​

1. DataFrame과 Series​

import pandas as pd

# Series: 1차원 데이터
temperatures = pd.Series([23, 25, 22, 24],
                        index=['Mon', 'Tue', 'Wed', 'Thu'])

# DataFrame: 2차원 표 형식 데이터
data = {
    'Name': ['Alice', 'Bob', 'Charlie'],
    'Age': [25, 30, 35],
    'City': ['Seoul', 'Busan', 'Incheon']
}
df = pd.DataFrame(data)

2. 간편한 데이터 조작​

# 데이터 필터링
young_people = df[df['Age'] < 30]

# 그룹화 및 집계
age_stats = df.groupby('City')['Age'].mean()

# 정렬
sorted_df = df.sort_values('Age', ascending=False)

3. 결측치 처리​

# 결측치 확인
df.isnull().sum()

# 결측치 제거
df.dropna()

# 결측치 채우기
df.fillna(0)

왜 pandas인가?​

vs 순수 Python​

**순수 Python (리스트, 딕셔너리)**
- 반복문으로 처리
- 느린 속도
- 복잡한 코드

**pandas**
- 벡터화 연산
- NumPy 기반 빠른 처리
- 간결한 코드
- 편리한 메서드 제공

vs Excel​

**Excel**
- GUI 기반
- 100만 행 제한
- 반복 작업 어려움
- 버전 관리 어려움

**pandas**
- 코드 기반 (재현 가능)
- 메모리가 허용하는 한 무제한
- 자동화 가능
- Git으로 버전 관리

사용 사례​

✅ **데이터 분석**
   - 매출 분석
   - 고객 행동 분석
   - A/B 테스트 결과 분석

✅ **데이터 전처리**
   - 머신러닝 전 데이터 정제
   - Feature Engineering
   - 데이터 정규화

✅ **리포트 생성**
   - 일일/주간/월간 리포트
   - 대시보드 데이터 준비
   - Excel 리포트 자동화

✅ **ETL 파이프라인**
   - 데이터 추출 (Extract)
   - 데이터 변환 (Transform)
   - 데이터 적재 (Load)

첫 DataFrame 만들기​

딕셔너리에서 생성​

import pandas as pd

# 데이터 준비
data = {
    '이름': ['김철수', '이영희', '박민수', '정지은'],
    '나이': [25, 30, 35, 28],
    '도시': ['서울', '부산', '대구', '인천'],
    '연봉': [3500, 4200, 4800, 3900]
}

# DataFrame 생성
df = pd.DataFrame(data)

# 데이터 확인
print(df)

3  정지은  28  인천  3900

pandas의 핵심 구조​

Series​

1차원 레이블 배열입니다.

# Series 생성
s = pd.Series([10, 20, 30, 40],
              index=['a', 'b', 'c', 'd'])

# 인덱스로 접근
print(s['b'])  # 20

# 연산
print(s * 2)   # 각 값이 2배

DataFrame​

2차원 레이블 데이터 구조입니다.

# DataFrame은 여러 Series의 모음
df = pd.DataFrame({
    'A': pd.Series([1, 2, 3]),
    'B': pd.Series([4, 5, 6])
})

일반적인 작업 흐름​

# 1. 데이터 로드
df = pd.read_csv('sales_data.csv')

# 2. 탐색
df.head()
df.info()
df.describe()

# 3. 정제
df = df.dropna()  # 결측치 제거
df = df.drop_duplicates()  # 중복 제거

# 4. 변환
df['날짜'] = pd.to_datetime(df['날짜'])
df['총액'] = df['수량'] * df['단가']

# 5. 분석
monthly_sales = df.groupby('월')['총액'].sum()

# 6. 시각화
monthly_sales.plot(kind='bar')

# 7. 저장
df.to_csv('cleaned_sales.csv', index=False)
