2. 기본 문법

Python의 기본 데이터 타입과 연산자를 배워봅시다.

2.1 데이터 타입​

숫자 (Numbers)​

# 정수 (int)
age = 25
population = 1000000
negative = -42

# 실수 (float)
height = 175.5
pi = 3.14159
scientific = 1.5e-4  # 0.00015

# 복소수 (complex)
complex_num = 3 + 4j

문자열 (Strings)​

# 작은따옴표 또는 큰따옴표
name = 'Python'
message = "Hello, World!"

# 여러 줄 문자열
multiline = """
첫 번째 줄
두 번째 줄
세 번째 줄
"""

# 문자열 연산
greeting = "Hello" + " " + "World"  # Hello World
repeat = "Hi! " * 3  # Hi! Hi! Hi!

# 인덱싱 (0부터 시작)
text = "Python"
print(text[0])   # P
print(text[-1])  # n (뒤에서 첫 번째)

# 슬라이싱
print(text[0:3])   # Pyt
print(text[2:])    # thon
print(text[:4])    # Pyth
print(text[::2])   # Pto (2칸씩)

불리언 (Boolean)​

# True 또는 False (첫 글자 대문자!)
is_student = True
is_working = False

# 비교 연산 결과
print(5 > 3)      # True
print(10 == 10)   # True
print(7 != 7)     # False

None​

# 값이 없음을 나타냄
result = None

if result is None:
    print("값이 없습니다")

2.2 타입 확인 및 변환​

타입 확인​

x = 42
print(type(x))  # <class 'int'>

y = 3.14
print(type(y))  # <class 'float'>

name = "Python"
print(type(name))  # <class 'str'>

타입 변환​

# 문자열 → 정수
age_str = "25"
age_int = int(age_str)
print(age_int + 5)  # 30

# 정수 → 문자열
num = 100
num_str = str(num)
print("Score: " + num_str)  # Score: 100

# 문자열 → 실수
price_str = "19.99"
price_float = float(price_str)
print(price_float * 2)  # 39.98

# 다양한 값 → 불리언
print(bool(0))      # False
print(bool(1))      # True
print(bool(""))     # False (빈 문자열)
print(bool("Hi"))   # True
print(bool(None))   # False

2.3 연산자​

산술 연산자​

a = 10
b = 3

print(a + b)   # 13 (덧셈)
print(a - b)   # 7  (뺄셈)
print(a * b)   # 30 (곱셈)
print(a / b)   # 3.3333... (나눗셈)
print(a // b)  # 3  (정수 나눗셈)
print(a % b)   # 1  (나머지)
print(a ** b)  # 1000 (거듭제곱)

비교 연산자​

x = 5
y = 10

print(x == y)   # False (같음)
print(x != y)   # True  (다름)
print(x > y)    # False (큼)
print(x < y)    # True  (작음)
print(x >= y)   # False (크거나 같음)
print(x <= y)   # True  (작거나 같음)

논리 연산자​

a = True
b = False

print(a and b)  # False (둘 다 참이어야 참)
print(a or b)   # True  (하나라도 참이면 참)
print(not a)    # False (반대)

# 실용 예제
age = 25
has_license = True

can_drive = age >= 18 and has_license
print(can_drive)  # True

대입 연산자​

x = 10

x += 5   # x = x + 5  → 15
x -= 3   # x = x - 3  → 12
x *= 2   # x = x * 2  → 24
x /= 4   # x = x / 4  → 6.0
x //= 2  # x = x // 2 → 3.0
x %= 2   # x = x % 2  → 1.0
x **= 3  # x = x ** 3 → 1.0

멤버십 연산자​

# in, not in
text = "Hello, Python!"

print("Python" in text)      # True
print("Java" in text)        # False
print("Java" not in text)    # True

numbers = [1, 2, 3, 4, 5]
print(3 in numbers)          # True
print(10 not in numbers)     # True

식별 연산자​

# is, is not (객체 동일성 비교)
x = [1, 2, 3]
y = [1, 2, 3]
z = x

print(x == y)   # True  (값이 같음)
print(x is y)   # False (다른 객체)
print(x is z)   # True  (같은 객체)

# None 비교 시 is 사용
result = None
print(result is None)      # ✅ 권장
print(result == None)      # ⚠️ 비권장

2.4 문자열 포매팅​

f-string (Python 3.6+, 권장)​

name = "김철수"
age = 25

# f-string
message = f"제 이름은 {name}이고, {age}세입니다."
print(message)

# 표현식 사용
x = 10
y = 20
print(f"{x} + {y} = {x + y}")  # 10 + 20 = 30

# 포맷 지정
pi = 3.14159
print(f"원주율: {pi:.2f}")  # 원주율: 3.14

price = 1234567
print(f"가격: {price:,}원")  # 가격: 1,234,567원

format() 메서드​

name = "Python"
version = 3.11

message = "언어: {}, 버전: {}".format(name, version)
print(message)  # 언어: Python, 버전: 3.11

# 인덱스 지정
message = "{1}의 버전은 {0}입니다".format(version, name)
print(message)  # Python의 버전은 3.11입니다

# 이름 지정
message = "{lang}의 버전은 {ver}입니다".format(lang=name, ver=version)

% 포매팅 (구식, 비권장)​

name = "Python"
age = 32

# 여전히 작동하지만 권장하지 않음
message = "언어: %s, 나이: %d년" % (name, age)

2.5 문자열 메서드​

대소문자 변환​

text = "Hello, Python!"

print(text.upper())      # HELLO, PYTHON!
print(text.lower())      # hello, python!
print(text.capitalize()) # Hello, python!
print(text.title())      # Hello, Python!

# 대소문자 바꾸기
print(text.swapcase())   # hELLO, pYTHON!

검색 및 확인​

text = "Python Programming"

# 검색
print(text.find("Pro"))      # 7 (시작 인덱스)
print(text.find("Java"))     # -1 (없음)
print(text.index("Pro"))     # 7 (find와 유사, 없으면 에러)

# 시작/끝 확인
print(text.startswith("Py")) # True
print(text.endswith("ing"))  # True

# 포함 확인
print("gram" in text)        # True

# 문자 타입 확인
print("123".isdigit())       # True
print("abc".isalpha())       # True
print("abc123".isalnum())    # True
print("   ".isspace())       # True

분리 및 결합​

# 분리
text = "apple,banana,orange"
fruits = text.split(",")
print(fruits)  # ['apple', 'banana', 'orange']

sentence = "Hello World Python"
words = sentence.split()  # 공백 기준
print(words)  # ['Hello', 'World', 'Python']

# 결합
fruits = ['apple', 'banana', 'orange']
text = ", ".join(fruits)
print(text)  # apple, banana, orange

# 줄 단위 분리
multiline = "첫째\n둘째\n셋째"
lines = multiline.splitlines()
print(lines)  # ['첫째', '둘째', '셋째']

제거 및 교체​

text = "  Hello, Python!  "

# 공백 제거
print(text.strip())   # "Hello, Python!"
print(text.lstrip())  # "Hello, Python!  " (왼쪽만)
print(text.rstrip())  # "  Hello, Python!" (오른쪽만)

# 문자 교체
message = "Hello, World!"
new_message = message.replace("World", "Python")
print(new_message)  # Hello, Python!

# 여러 번 교체
text = "one one one"
print(text.replace("one", "two", 2))  # two two one

2.6 연산자 우선순위​

# 괄호 > 거듭제곱 > 곱셈/나눗셈 > 덧셈/뺄셈

result = 2 + 3 * 4
print(result)  # 14 (곱셈 먼저)

result = (2 + 3) * 4
print(result)  # 20 (괄호 먼저)

result = 2 ** 3 ** 2
print(result)  # 512 (오른쪽부터: 3^2 = 9, 2^9 = 512)

result = (2 ** 3) ** 2
print(result)  # 64 (왼쪽부터: 2^3 = 8, 8^2 = 64)

2.7 입력과 출력​

입력​

# 기본 입력
name = input("이름을 입력하세요: ")

# 숫자 입력
age = int(input("나이를 입력하세요: "))
height = float(input("키를 입력하세요(cm): "))

# 여러 값 한 번에 입력
numbers = input("숫자들을 공백으로 구분하여 입력: ")
num_list = list(map(int, numbers.split()))
print(num_list)

출력​

# 기본 출력
print("Hello")

# 여러 값 출력
print("이름:", name, "나이:", age)

# 구분자 변경
print("apple", "banana", "orange", sep=", ")

# 줄바꿈 제거
print("Loading", end="...")
print("Done!")  # Loading...Done!

# 파일로 출력 (나중에 배움)
# print("로그", file=log_file)

2.8 실습 예제​

예제 1: BMI 계산기​

# BMI = 체중(kg) / (키(m) ** 2)

print("=== BMI 계산기 ===")

weight = float(input("체중(kg): "))
height_cm = float(input("키(cm): "))

height_m = height_cm / 100
bmi = weight / (height_m ** 2)

print(f"\n당신의 BMI: {bmi:.1f}")

if bmi < 18.5:
    print("저체중")
elif bmi < 23:
    print("정상")
elif bmi < 25:
    print("과체중")
else:
    print("비만")

예제 2: 문자열 분석기​

text = input("문장을 입력하세요: ")

print(f"\n=== 분석 결과 ===")
print(f"전체 길이: {len(text)}")
print(f"공백 제외: {len(text.replace(' ', ''))}")
print(f"단어 수: {len(text.split())}")
print(f"대문자: {text.upper()}")
print(f"소문자: {text.lower()}")
print(f"첫 글자: {text[0] if text else '없음'}")
print(f"마지막 글자: {text[-1] if text else '없음'}")

2.9 체크리스트​

데이터 타입:

  • 정수, 실수, 문자열, 불리언 이해
  • 타입 확인 (type())
  • 타입 변환 (int(), str(), float())

연산자:

  • 산술 연산자 (+, -, *, /, //, %, **)
  • 비교 연산자 (==, !=, <, >, <=, >=)
  • 논리 연산자 (and, or, not)
  • 멤버십 연산자 (in, not in)

문자열:

  • 인덱싱과 슬라이싱
  • f-string 포매팅
  • 주요 메서드 (upper, lower, split, join, replace)
