6. 모듈과 패키지

Python의 모듈과 패키지는 코드를 재사용 가능한 단위로 구성하는 핵심 메커니즘입니다. 이를 통해 대규모 프로젝트를 효율적으로 관리하고, 다른 개발자들이 만든 코드를 활용할 수 있습니다.

6.1 모듈이란?​

**모듈(Module)**은 Python 코드를 담고 있는 파일입니다. 함수, 클래스, 변수 등을 포함할 수 있으며, .py 확장자를 가집니다.

모듈의 장점​

  • 코드 재사용: 한 번 작성한 코드를 여러 프로그램에서 사용
  • 네임스페이스 관리: 변수명 충돌 방지
  • 코드 구조화: 논리적으로 관련된 코드를 그룹화
  • 유지보수 용이: 기능별로 분리하여 관리

모듈의 종류​

  1. 내장 모듈: Python과 함께 제공되는 표준 라이브러리
  2. 서드파티 모듈: 외부 개발자가 만든 패키지 (pip로 설치)
  3. 사용자 정의 모듈: 직접 작성한 .py 파일

6.2 모듈 가져오기 (Import)​

6.2.1 기본 import​

# 전체 모듈 가져오기
import math

result = math.sqrt(16)
print(result)  # 4.0

pi_value = math.pi
print(pi_value)  # 3.141592653589793

6.2.2 from...import 구문​

# 특정 항목만 가져오기
from math import sqrt, pi

result = sqrt(25)  # math. 접두어 없이 사용
print(result)  # 5.0
print(pi)  # 3.141592653589793
# 모든 항목 가져오기 (권장하지 않음)
from math import *

print(sqrt(9))  # 3.0
print(ceil(3.2))  # 4
왜 from module import *를 피해야 할까요?
  • 어떤 함수가 어느 모듈에서 왔는지 불분명해집니다
  • 네임스페이스 충돌 가능성이 높아집니다
  • 코드 가독성이 떨어집니다

6.2.3 별칭 사용 (as)​

# 모듈에 별칭 지정
import numpy as np
import pandas as pd

arr = np.array([1, 2, 3])
# 함수에 별칭 지정
from datetime import datetime as dt

now = dt.now()
print(now)

# 한 줄에 여러 모듈 (가독성 떨어짐)
import os, sys, json

6.3 사용자 정의 모듈 만들기​

6.3.1 간단한 모듈 생성​

mymath.py 파일 생성:

"""
간단한 수학 연산을 제공하는 모듈
"""

def add(a, b):
    """두 수를 더합니다."""
    return a + b

def subtract(a, b):
    """두 수를 뺍니다."""
    return a - b

def multiply(a, b):
    """두 수를 곱합니다."""
    return a * b

def divide(a, b):
    """두 수를 나눕니다."""
    if b == 0:
        raise ValueError("0으로 나눌 수 없습니다")
    return a / b

# 상수 정의
PI = 3.14159
E = 2.71828

main.py에서 사용:

import mymath

result1 = mymath.add(10, 5)
print(result1)  # 15

result2 = mymath.multiply(4, 3)
print(result2)  # 12

print(mymath.PI)  # 3.14159

6.3.2 모듈 실행 vs 임포트​

calculator.py:

def add(a, b):
    return a + b

def subtract(a, b):
    return a - b

# 이 블록은 직접 실행할 때만 실행됩니다
if __name__ == "__main__":
    print("Calculator 모듈 테스트")
    print(f"10 + 5 = {add(10, 5)}")
    print(f"10 - 5 = {subtract(10, 5)}")
# calculator.py를 직접 실행: 테스트 코드가 실행됨
# python calculator.py

# 다른 파일에서 import: 테스트 코드는 실행되지 않음
import calculator
result = calculator.add(3, 4)
__name__ 변수란?
  • 파일을 직접 실행하면: __name__ == "__main__"
  • 다른 파일에서 import하면: __name__ == "모듈명"
  • 이를 활용해 테스트 코드와 모듈 코드를 분리할 수 있습니다

6.3.3 모듈 검색 경로​

Python은 다음 순서로 모듈을 검색합니다:

  1. 현재 디렉토리
  2. PYTHONPATH 환경 변수에 지정된 디렉토리
  3. Python 설치 디렉토리
import sys

# 모듈 검색 경로 확인
for path in sys.path:
    print(path)
# 커스텀 경로 추가
import sys
sys.path.append('/path/to/my/modules')

import my_custom_module

6.4 표준 라이브러리 개요​

Python은 풍부한 표준 라이브러리를 제공합니다. 별도 설치 없이 바로 사용할 수 있습니다.

6.4.1 math - 수학 연산​

import math

# 기본 연산
print(math.sqrt(16))      # 4.0 (제곱근)
print(math.pow(2, 3))     # 8.0 (거듭제곱)
print(math.factorial(5))  # 120 (팩토리얼)

# 반올림 관련
print(math.ceil(3.2))     # 4 (올림)
print(math.floor(3.8))    # 3 (내림)
print(math.trunc(3.9))    # 3 (버림)

# 삼각함수
print(math.sin(math.pi / 2))  # 1.0
print(math.cos(0))            # 1.0
print(math.tan(math.pi / 4))  # 0.9999999999999999

# 로그
print(math.log(100, 10))  # 2.0 (log₁₀100)
print(math.log(8, 2))     # 3.0 (log₂8)
print(math.log(math.e))   # 1.0 (자연로그)

# 상수
print(math.pi)  # 3.141592653589793
print(math.e)   # 2.718281828459045
print(math.inf) # inf (무한대)

6.4.2 random - 난수 생성​

import random

# 기본 난수
print(random.random())  # 0.0 ~ 1.0 사이의 실수

# 범위 지정
print(random.randint(1, 10))    # 1 ~ 10 사이의 정수
print(random.uniform(1.0, 10.0))  # 1.0 ~ 10.0 사이의 실수
print(random.randrange(0, 100, 5))  # 0 ~ 100, 5씩 증가하는 값 중 선택

# 리스트에서 선택
colors = ['red', 'green', 'blue', 'yellow']
print(random.choice(colors))  # 랜덤으로 하나 선택

# 여러 개 선택
print(random.sample(colors, 2))  # 중복 없이 2개 선택
print(random.choices(colors, k=3))  # 중복 허용하여 3개 선택

# 섞기
numbers = [1, 2, 3, 4, 5]
random.shuffle(numbers)
print(numbers)  # [3, 1, 5, 2, 4] (순서 랜덤)

# 시드 설정 (재현 가능한 난수)
random.seed(42)
print(random.random())  # 항상 같은 값

6.4.3 datetime - 날짜와 시간​

from datetime import datetime, date, time, timedelta

# 현재 날짜와 시간
now = datetime.now()
print(now)  # 2024-01-15 14:30:45.123456

today = date.today()
print(today)  # 2024-01-15

# 특정 날짜/시간 생성
dt = datetime(2024, 12, 25, 18, 30, 0)
print(dt)  # 2024-12-25 18:30:00

d = date(2024, 6, 15)
t = time(14, 30, 45)

# 날짜/시간 형식화
print(now.strftime("%Y-%m-%d"))  # 2024-01-15
print(now.strftime("%Y년 %m월 %d일"))  # 2024년 01월 15일
print(now.strftime("%H:%M:%S"))  # 14:30:45
print(now.strftime("%A, %B %d, %Y"))  # Monday, January 15, 2024

# 문자열을 날짜로 변환
date_str = "2024-12-25"
dt = datetime.strptime(date_str, "%Y-%m-%d")
print(dt)

# 날짜 연산
tomorrow = today + timedelta(days=1)
next_week = today + timedelta(weeks=1)
two_hours_later = now + timedelta(hours=2)

print(f"내일: {tomorrow}")
print(f"다음 주: {next_week}")

# 날짜 차이 계산
new_year = date(2025, 1, 1)
diff = new_year - today
print(f"새해까지 {diff.days}일 남았습니다")

# 요일 확인
print(today.weekday())  # 0 (월요일) ~ 6 (일요일)
print(today.strftime("%A"))  # Monday

6.4.6 json - JSON 데이터 처리​

import json

# Python 객체 → JSON 문자열
data = {
    'name': '홍길동',
    'age': 30,
    'skills': ['Python', 'JavaScript'],
    'is_student': False
}

json_str = json.dumps(data, ensure_ascii=False, indent=2)
print(json_str)
# {
#   "name": "홍길동",
#   "age": 30,
#   "skills": ["Python", "JavaScript"],
#   "is_student": false
# }

# JSON 문자열 → Python 객체
parsed_data = json.loads(json_str)
print(parsed_data['name'])  # 홍길동

# 파일에 JSON 저장
with open('data.json', 'w', encoding='utf-8') as f:
    json.dump(data, f, ensure_ascii=False, indent=2)

# 파일에서 JSON 읽기
with open('data.json', 'r', encoding='utf-8') as f:
    loaded_data = json.load(f)
    print(loaded_data)

6.5 패키지 (Packages)​

패키지는 관련된 모듈들을 모아놓은 디렉토리입니다. 계층적 구조로 모듈을 조직화할 수 있습니다.

6.5.1 패키지 구조​

mypackage/
    __init__.py
    module1.py
    module2.py
    subpackage/
        __init__.py
        module3.py
        module4.py

6.5.2 __init__.py 파일​

__init__.py는 디렉토리를 Python 패키지로 인식하게 만듭니다.

mypackage/init.py:

"""
mypackage - 유틸리티 함수 모음
"""

__version__ = '1.0.0'
__author__ = 'Your Name'

# 패키지 레벨에서 바로 접근 가능하게 설정
from .module1 import function1
from .module2 import function2

__all__ = ['function1', 'function2']

mypackage/module1.py:

def function1():
    return "Function 1 from module1"

def helper_function():
    return "Helper function"

mypackage/module2.py:

def function2():
    return "Function 2 from module2"

6.5.3 패키지 사용하기​

# 방법 1: 전체 경로
import mypackage.module1
result = mypackage.module1.function1()

# 방법 2: 특정 모듈
from mypackage import module1
result = module1.function1()

# 방법 3: 특정 함수 (__init__.py에서 정의한 경우)
from mypackage import function1
result = function1()

# 서브패키지 사용
from mypackage.subpackage import module3

6.5.4 상대 임포트​

패키지 내부에서 다른 모듈을 참조할 때 사용합니다.

mypackage/subpackage/module3.py:

# 같은 패키지 내의 모듈
from . import module4

# 상위 패키지의 모듈
from .. import module1

# 상위 패키지의 다른 서브패키지
from ..other_subpackage import module5
상대 임포트 vs 절대 임포트
  • 절대 임포트: from mypackage import module1 (권장)
  • 상대 임포트: from . import module1 (패키지 내부에서만)
  • 명확성을 위해 절대 임포트를 선호하는 것이 좋습니다

6.7 가상 환경 (Virtual Environment)​

가상 환경은 프로젝트별로 독립적인 Python 환경을 제공합니다.

6.7.1 가상 환경이 필요한 이유​

  • 프로젝트마다 다른 버전의 패키지 사용 가능
  • 시스템 Python 환경을 오염시키지 않음
  • 프로젝트 의존성을 명확하게 관리
  • 배포 시 재현 가능한 환경 제공

6.7.2 venv 사용하기​

# 가상 환경 생성
python -m venv myenv
python -m venv venv  # 일반적으로 'venv' 이름 사용

# 가상 환경 활성화
# Windows
myenv\Scripts\activate

# macOS/Linux
source myenv/bin/activate

# 활성화 확인 (프롬프트 변경)
(myenv) C:\project>

# 패키지 설치 (가상 환경에만 설치됨)
pip install requests

# 가상 환경 비활성화
deactivate

6.7.3 가상 환경 작업 흐름​

# 1. 프로젝트 디렉토리 생성
mkdir my_project
cd my_project

# 2. 가상 환경 생성
python -m venv venv

# 3. 활성화
source venv/bin/activate  # macOS/Linux
venv\Scripts\activate     # Windows

# 4. 필요한 패키지 설치
pip install requests pandas

# 5. requirements.txt 생성
pip freeze > requirements.txt

# 6. 작업 완료 후 비활성화
deactivate

6.8 실전 예제​

6.8.1 웹 스크래핑 (requests + BeautifulSoup)​

pip install requests beautifulsoup4
import requests
from bs4 import BeautifulSoup

# 웹 페이지 가져오기
url = 'https://example.com'
response = requests.get(url)

if response.status_code == 200:
    # HTML 파싱
    soup = BeautifulSoup(response.text, 'html.parser')

    # 제목 추출
    title = soup.find('h1')
    print(f"제목: {title.text}")

    # 모든 링크 추출
    links = soup.find_all('a')
    for link in links:
        href = link.get('href')
        text = link.text
        print(f"{text}: {href}")
else:
    print(f"오류: {response.status_code}")

6.8.2 데이터 분석 (pandas)​

pip install pandas
import pandas as pd

6.8.4 파일 관리 자동화​

import os
import shutil
from datetime import datetime

def organize_files(directory):
    """파일을 확장자별로 정리합니다."""

    # 파일 타입별 폴더 매핑
    file_types = {
        'images': ['.jpg', '.jpeg', '.png', '.gif', '.bmp'],
        'documents': ['.pdf', '.doc', '.docx', '.txt', '.xlsx'],
        'videos': ['.mp4', '.avi', '.mkv', '.mov'],
        'music': ['.mp3', '.wav', '.flac'],
        'archives': ['.zip', '.rar', '.7z', '.tar', '.gz']
    }

    # 폴더 생성
    for folder in file_types.keys():
        folder_path = os.path.join(directory, folder)
        os.makedirs(folder_path, exist_ok=True)

    # 파일 이동
    for filename in os.listdir(directory):
        file_path = os.path.join(directory, filename)

        # 디렉토리는 건너뛰기
        if os.path.isdir(file_path):
            continue

        # 확장자 확인
        _, ext = os.path.splitext(filename)
        ext = ext.lower()

        # 적절한 폴더로 이동
        moved = False
        for folder, extensions in file_types.items():
            if ext in extensions:
                dest = os.path.join(directory, folder, filename)
                shutil.move(file_path, dest)
                print(f"{filename} → {folder}/")
                moved = True
                break

        if not moved:
            # 기타 파일
            other_folder = os.path.join(directory, 'others')
            os.makedirs(other_folder, exist_ok=True)
            dest = os.path.join(other_folder, filename)
            shutil.move(file_path, dest)
            print(f"{filename} → others/")

# 사용 예
organize_files('C:/Users/YourName/Downloads')

"""
calculator.py - 기본 계산기 모듈
"""

def add(a, b):
    """두 수를 더합니다."""
    return a + b

def subtract(a, b):
    """두 수를 뺍니다."""
    return a - b

def multiply(a, b):
    """두 수를 곱합니다."""
    return a * b

def divide(a, b):
    """두 수를 나눕니다."""
    if b == 0:
        raise ValueError("0으로 나눌 수 없습니다")
    return a / b

def power(base, exponent):
    """거듭제곱을 계산합니다."""
    return base ** exponent

def factorial(n):
    """팩토리얼을 계산합니다."""
    if n < 0:
        raise ValueError("음수는 팩토리얼을 계산할 수 없습니다")
    if n == 0 or n == 1:
        return 1

    result = 1
    for i in range(2, n + 1):
        result *= i
    return result

연습 문제 2: 날짜 유틸리티​

datetime 모듈을 사용하여 다음 함수들을 작성하세요:

  1. days_until_birthday(birth_month, birth_day): 다음 생일까지 남은 일수
  2. age_calculator(birth_year, birth_month, birth_day): 현재 나이 계산
  3. is_weekend(date_str): 주어진 날짜가 주말인지 확인 (YYYY-MM-DD 형식)
해답 보기
from datetime import datetime, date, timedelta

def days_until_birthday(birth_month, birth_day):
    """다음 생일까지 남은 일수를 계산합니다."""
    today = date.today()
    current_year = today.year

    # 올해 생일
    birthday = date(current_year, birth_month, birth_day)

    # 생일이 이미 지났다면 내년 생일로
    if birthday < today:
        birthday = date(current_year + 1, birth_month, birth_day)

    days_left = (birthday - today).days
    return days_left

def age_calculator(birth_year, birth_month, birth_day):
    """현재 나이를 계산합니다."""
    today = date.today()
    birth_date = date(birth_year, birth_month, birth_day)

    age = today.year - birth_date.year

    # 생일이 아직 안 지났으면 1 빼기
    if (today.month, today.day) < (birth_date.month, birth_date.day):
        age -= 1

    return age

def is_weekend(date_str):
    """주어진 날짜가 주말인지 확인합니다."""
    date_obj = datetime.strptime(date_str, "%Y-%m-%d")
    # weekday(): 0(월) ~ 6(일)
    return date_obj.weekday() >= 5

# 테스트
if __name__ == "__main__":
    print(f"생일까지 {days_until_birthday(12, 25)}일 남았습니다")
    print(f"나이: {age_calculator(1990, 5, 15)}세")
    print(f"2024-01-15는 주말인가요? {is_weekend('2024-01-15')}")

연습 문제 5: 웹 스크래퍼 프로젝트​

가상 환경에서 requests와 BeautifulSoup을 사용하여 Python 공식 문서 페이지(https://docs.python.org/3/)의 제목과 첫 번째 문단을 추출하는 프로그램을 작성하세요.

해답 보기
"""
web_scraper.py - Python 문서 스크래퍼
"""
import requests
from bs4 import BeautifulSoup

def scrape_python_docs():
    """Python 공식 문서의 제목과 첫 문단을 추출합니다."""
    url = 'https://docs.python.org/3/'

    print(f"페이지 요청 중: {url}")
    response = requests.get(url)

    if response.status_code == 200:
        soup = BeautifulSoup(response.text, 'html.parser')

        # 제목 추출
        title = soup.find('title')
        if title:
            print(f"\n제목: {title.text}")

        # 첫 번째 문단 추출
        first_paragraph = soup.find('p')
        if first_paragraph:
            print(f"\n첫 번째 문단:")
            print(first_paragraph.text.strip())

        # 모든 h2 헤더 추출
        print("\n주요 섹션:")
        headers = soup.find_all('h2')
        for header in headers[:5]:  # 처음 5개만
            print(f"- {header.text.strip()}")

        return True
    else:
        print(f"오류: HTTP {response.status_code}")
        return False

def scrape_with_headers():
    """User-Agent 헤더를 포함한 요청"""
    url = 'https://docs.python.org/3/'

    headers = {
        'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36'
    }

    response = requests.get(url, headers=headers, timeout=10)

    if response.status_code == 200:
        print("✓ 요청 성공")
        return response.text
    else:
        print(f"✗ 요청 실패: {response.status_code}")
        return None

if __name__ == "__main__":
    print("=== Python 문서 스크래퍼 ===\n")
    scrape_python_docs()

    │   ├── __init__.py
    │   ├── module1.py
    │   └── module2.py
    ├── tests/                # 테스트 코드
    │   ├── __init__.py
    │   └── test_module1.py
    ├── docs/                 # 문서
    ├── .gitignore
    ├── README.md
    ├── requirements.txt      # 운영 의존성
    ├── requirements-dev.txt  # 개발 의존성
    └── setup.py              # 패키지 설정

정리​

  • 모듈은 재사용 가능한 Python 코드 파일입니다
  • 임포트는 import, from...import, as로 할 수 있습니다
  • 표준 라이브러리는 별도 설치 없이 사용 가능합니다
  • 패키지는 모듈들을 계층적으로 구조화합니다
  • pip로 서드파티 패키지를 설치합니다
  • 가상 환경은 프로젝트별 독립적인 Python 환경을 제공합니다
  • requirements.txt로 의존성을 관리합니다
  • 코드를 모듈화하면 재사용성과 유지보수성이 향상됩니다

다음 단계​

모듈과 패키지의 기초를 익혔다면, 이제 객체 지향 프로그래밍을 학습할 준비가 되었습니다.
