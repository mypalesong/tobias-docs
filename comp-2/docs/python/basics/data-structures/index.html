5. 자료구조

Python의 내장 자료구조는 데이터를 효율적으로 저장하고 관리하는 핵심 도구입니다. 각 자료구조는 고유한 특성과 용도를 가지고 있으며, 상황에 맞는 자료구조를 선택하는 것이 중요합니다.

5.1 리스트 (Lists)​

리스트는 Python에서 가장 많이 사용되는 자료구조로, 순서가 있고 변경 가능한(mutable) 데이터 컬렉션입니다.

5.1.1 리스트 생성​

# 빈 리스트 생성
empty_list = []
empty_list2 = list()

# 값이 있는 리스트 생성
numbers = [1, 2, 3, 4, 5]
fruits = ["apple", "banana", "cherry"]
mixed = [1, "hello", 3.14, True]

# 중첩 리스트 (2차원 리스트)
matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

# list() 함수를 사용한 변환
from_string = list("hello")  # ['h', 'e', 'l', 'l', 'o']
from_range = list(range(5))  # [0, 1, 2, 3, 4]

5.1.2 인덱싱과 슬라이싱​

fruits = ["apple", "banana", "cherry", "date", "elderberry"]

# 인덱싱 (0부터 시작)
print(fruits[0])   # apple
print(fruits[2])   # cherry
print(fruits[-1])  # elderberry (마지막 요소)
print(fruits[-2])  # date (뒤에서 두 번째)

# 슬라이싱 [시작:끝:간격]
print(fruits[1:3])    # ['banana', 'cherry']
print(fruits[:2])     # ['apple', 'banana'] (처음부터 인덱스 2 전까지)
print(fruits[2:])     # ['cherry', 'date', 'elderberry'] (인덱스 2부터 끝까지)
print(fruits[::2])    # ['apple', 'cherry', 'elderberry'] (2칸씩 건너뛰기)
print(fruits[::-1])   # 리스트 뒤집기

# 중첩 리스트 접근
matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
print(matrix[0])      # [1, 2, 3]
print(matrix[1][2])   # 6

5.1.3 리스트 메서드​

# append() - 끝에 요소 추가
fruits = ["apple", "banana"]
fruits.append("cherry")
print(fruits)  # ['apple', 'banana', 'cherry']

# insert() - 특정 위치에 요소 삽입
fruits.insert(1, "blueberry")
print(fruits)  # ['apple', 'blueberry', 'banana', 'cherry']

# extend() - 여러 요소 추가
fruits.extend(["date", "elderberry"])
print(fruits)  # ['apple', 'blueberry', 'banana', 'cherry', 'date', 'elderberry']

# remove() - 특정 값 제거 (첫 번째 일치 항목)
fruits.remove("banana")
print(fruits)  # ['apple', 'blueberry', 'cherry', 'date', 'elderberry']

# pop() - 특정 인덱스의 요소 제거 및 반환
last_fruit = fruits.pop()      # 마지막 요소 제거
print(last_fruit)              # elderberry
second = fruits.pop(1)         # 인덱스 1의 요소 제거
print(second)                  # blueberry

# index() - 특정 값의 인덱스 찾기
position = fruits.index("cherry")
print(position)  # 1

# count() - 특정 값의 개수 세기
numbers = [1, 2, 2, 3, 2, 4]
print(numbers.count(2))  # 3

# sort() - 리스트 정렬 (원본 변경)
numbers = [3, 1, 4, 1, 5, 9, 2]
numbers.sort()
print(numbers)  # [1, 1, 2, 3, 4, 5, 9]

numbers.sort(reverse=True)
print(numbers)  # [9, 5, 4, 3, 2, 1, 1]

# sorted() - 정렬된 새 리스트 반환 (원본 유지)
original = [3, 1, 4, 1, 5]
sorted_list = sorted(original)
print(original)      # [3, 1, 4, 1, 5]
print(sorted_list)   # [1, 1, 3, 4, 5]

# reverse() - 리스트 뒤집기
fruits = ["apple", "banana", "cherry"]
fruits.reverse()
print(fruits)  # ['cherry', 'banana', 'apple']

# clear() - 모든 요소 제거
fruits.clear()
print(fruits)  # []

# copy() - 리스트 복사
original = [1, 2, 3]
copied = original.copy()
copied.append(4)
print(original)  # [1, 2, 3]
print(copied)    # [1, 2, 3, 4]

5.1.4 리스트 컴프리헨션 (List Comprehension)​

리스트 컴프리헨션은 간결하고 효율적으로 리스트를 생성하는 방법입니다.

# 기본 형식: [표현식 for 항목 in 반복가능객체]
squares = [x**2 for x in range(10)]
print(squares)  # [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

# 조건문 포함: [표현식 for 항목 in 반복가능객체 if 조건]
even_squares = [x**2 for x in range(10) if x % 2 == 0]
print(even_squares)  # [0, 4, 16, 36, 64]

# 문자열 처리
fruits = ["apple", "banana", "cherry"]
upper_fruits = [fruit.upper() for fruit in fruits]
print(upper_fruits)  # ['APPLE', 'BANANA', 'CHERRY']

# if-else 포함
numbers = [1, 2, 3, 4, 5]
labels = ["짝수" if n % 2 == 0 else "홀수" for n in numbers]
print(labels)  # ['홀수', '짝수', '홀수', '짝수', '홀수']

# 중첩 루프
matrix = [[i*j for j in range(1, 4)] for i in range(1, 4)]
print(matrix)  # [[1, 2, 3], [2, 4, 6], [3, 6, 9]]

# 2차원 리스트를 1차원으로 평탄화
nested = [[1, 2, 3], [4, 5], [6, 7, 8]]
flattened = [num for sublist in nested for num in sublist]
print(flattened)  # [1, 2, 3, 4, 5, 6, 7, 8]
팁

리스트 컴프리헨션은 for 루프보다 빠르고 간결합니다. 하지만 너무 복잡해지면 가독성이 떨어지므로, 복잡한 경우에는 일반 for 루프를 사용하는 것이 좋습니다.

5.2 튜플 (Tuples)​

튜플은 리스트와 유사하지만 변경 불가능한(immutable) 자료구조입니다.

5.2.1 튜플 생성​

# 소괄호를 사용한 생성
empty_tuple = ()
single_item = (1,)  # 주의: 쉼표 필요!
numbers = (1, 2, 3, 4, 5)
mixed = (1, "hello", 3.14, True)

# 소괄호 없이도 생성 가능 (튜플 패킹)
coordinates = 10, 20
print(type(coordinates))  # <class 'tuple'>

# tuple() 함수 사용
from_list = tuple([1, 2, 3])
from_string = tuple("hello")  # ('h', 'e', 'l', 'l', 'o')

5.2.2 튜플 언패킹​

# 기본 언패킹
point = (10, 20)
x, y = point
print(f"x: {x}, y: {y}")  # x: 10, y: 20

# 여러 값 반환 시 활용
def get_user_info():
    return "Alice", 25, "alice@email.com"

name, age, email = get_user_info()
print(name)   # Alice
print(age)    # 25
print(email)  # alice@email.com

# 언패킹 시 일부 값 무시
name, _, email = get_user_info()  # age는 무시

# 확장 언패킹 (Python 3.x)
numbers = (1, 2, 3, 4, 5)
first, *middle, last = numbers
print(first)   # 1
print(middle)  # [2, 3, 4]
print(last)    # 5

# 변수 값 교환
a, b = 10, 20
a, b = b, a
print(a, b)  # 20 10

5.2.3 튜플 메서드와 연산​

numbers = (1, 2, 2, 3, 2, 4)

# count() - 특정 값의 개수
print(numbers.count(2))  # 3

# index() - 특정 값의 인덱스
print(numbers.index(3))  # 3

# 인덱싱과 슬라이싱 (리스트와 동일)
print(numbers[0])    # 1
print(numbers[1:4])  # (2, 2, 3)

# 튜플 연결
tuple1 = (1, 2, 3)
tuple2 = (4, 5, 6)
combined = tuple1 + tuple2
print(combined)  # (1, 2, 3, 4, 5, 6)

# 튜플 반복
repeated = (1, 2) * 3
print(repeated)  # (1, 2, 1, 2, 1, 2)

5.2.4 튜플 사용 사례​

# 1. 딕셔너리 키로 사용 (리스트는 불가능)
locations = {
    (0, 0): "원점",
    (1, 0): "오른쪽",
    (0, 1): "위쪽"
}

# 2. 함수에서 여러 값 반환
def calculate_stats(numbers):
    return min(numbers), max(numbers), sum(numbers) / len(numbers)

min_val, max_val, avg = calculate_stats([1, 2, 3, 4, 5])

# 3. 상수 집합 정의 (변경되지 않아야 하는 데이터)
DAYS_OF_WEEK = ("월", "화", "수", "목", "금", "토", "일")
RGB_COLORS = (
    (255, 0, 0),    # 빨강
    (0, 255, 0),    # 초록
    (0, 0, 255)     # 파랑
)

# 4. 데이터베이스 레코드 표현
student_record = ("김철수", 20, "컴퓨터공학", 3.8)
팁

튜플을 사용해야 하는 경우:

  • 데이터가 변경되지 않아야 할 때
  • 딕셔너리 키로 사용해야 할 때
  • 함수에서 여러 값을 반환할 때
  • 메모리 효율이 중요할 때 (튜플이 리스트보다 메모리 효율적)

5.3 딕셔너리 (Dictionaries)​

딕셔너리는 키(key)와 값(value)의 쌍으로 데이터를 저장하는 자료구조입니다.

5.3.1 딕셔너리 생성​

# 중괄호를 사용한 생성
empty_dict = {}
empty_dict2 = dict()

# 키-값 쌍으로 생성
student = {
    "name": "김철수",
    "age": 20,
    "major": "컴퓨터공학",
    "gpa": 3.8
}

# dict() 함수 사용
person = dict(name="이영희", age=25, city="서울")

# 키-값 튜플 리스트로부터 생성
items = [("apple", 1000), ("banana", 500), ("cherry", 1500)]
prices = dict(items)
print(prices)  # {'apple': 1000, 'banana': 500, 'cherry': 1500}

# 키 리스트로부터 기본값으로 생성
keys = ["a", "b", "c"]
default_dict = dict.fromkeys(keys, 0)
print(default_dict)  # {'a': 0, 'b': 0, 'c': 0}

5.3.2 딕셔너리 접근과 수정​

student = {
    "name": "김철수",
    "age": 20,
    "major": "컴퓨터공학"
}

# 값 접근
print(student["name"])      # 김철수
print(student.get("age"))   # 20

# get()의 장점: 키가 없어도 에러가 나지 않음
print(student.get("gpa"))              # None
print(student.get("gpa", 0.0))         # 0.0 (기본값)
# print(student["gpa"])                # KeyError 발생!

# 값 수정
student["age"] = 21
print(student["age"])  # 21

# 새 키-값 추가
student["gpa"] = 3.8
print(student)  # {'name': '김철수', 'age': 21, 'major': '컴퓨터공학', 'gpa': 3.8}

# 값 삭제
del student["major"]
removed_age = student.pop("age")  # 삭제하고 값 반환
print(removed_age)  # 21

# 키 존재 여부 확인
print("name" in student)   # True
print("age" in student)    # False

5.3.3 딕셔너리 메서드​

student = {
    "name": "김철수",
    "age": 20,
    "major": "컴퓨터공학",
    "gpa": 3.8
}

# keys() - 모든 키 반환
print(student.keys())  # dict_keys(['name', 'age', 'major', 'gpa'])
print(list(student.keys()))  # ['name', 'age', 'major', 'gpa']

# values() - 모든 값 반환
print(student.values())  # dict_values(['김철수', 20, '컴퓨터공학', 3.8])

# items() - 모든 키-값 쌍 반환
print(student.items())
# dict_items([('name', '김철수'), ('age', 20), ('major', '컴퓨터공학'), ('gpa', 3.8)])

# 딕셔너리 순회
for key in student:
    print(f"{key}: {student[key]}")

for key, value in student.items():
    print(f"{key}: {value}")

# update() - 딕셔너리 병합/업데이트
student.update({"year": 3, "gpa": 4.0})
print(student)

# setdefault() - 키가 없으면 기본값 설정 후 반환
grade = student.setdefault("grade", "A")
print(grade)  # A
print(student)  # grade: A가 추가됨

# clear() - 모든 항목 제거
temp = {"a": 1, "b": 2}
temp.clear()
print(temp)  # {}

# copy() - 딕셔너리 복사
original = {"a": 1, "b": 2}
copied = original.copy()
copied["c"] = 3
print(original)  # {'a': 1, 'b': 2}
print(copied)    # {'a': 1, 'b': 2, 'c': 3}

5.3.4 딕셔너리 컴프리헨션​

# 기본 형식: {키: 값 for 항목 in 반복가능객체}
squares = {x: x**2 for x in range(6)}
print(squares)  # {0: 0, 1: 1, 2: 4, 3: 9, 4: 16, 5: 25}

# 조건문 포함
even_squares = {x: x**2 for x in range(10) if x % 2 == 0}
print(even_squares)  # {0: 0, 2: 4, 4: 16, 6: 36, 8: 64}

# 두 리스트로 딕셔너리 생성
keys = ["name", "age", "city"]
values = ["Alice", 25, "Seoul"]
person = {k: v for k, v in zip(keys, values)}
print(person)  # {'name': 'Alice', 'age': 25, 'city': 'Seoul'}

# 딕셔너리 변환
prices = {"apple": 1000, "banana": 500, "cherry": 1500}
expensive = {fruit: price for fruit, price in prices.items() if price > 800}
print(expensive)  # {'apple': 1000, 'cherry': 1500}

# 키와 값 뒤집기
original = {"a": 1, "b": 2, "c": 3}
swapped = {v: k for k, v in original.items()}
print(swapped)  # {1: 'a', 2: 'b', 3: 'c'}

5.3.5 중첩 딕셔너리​

# 학생 정보 관리 시스템
students = {
    "2024001": {
        "name": "김철수",
        "age": 20,
        "grades": {"수학": 95, "영어": 88, "과학": 92}
    },
    "2024002": {
        "name": "이영희",
        "age": 21,
        "grades": {"수학": 90, "영어": 95, "과학": 87}
    }
}

# 중첩 딕셔너리 접근
print(students["2024001"]["name"])           # 김철수
print(students["2024001"]["grades"]["수학"])  # 95

# 중첩 딕셔너리 순회
for student_id, info in students.items():
    print(f"\n학번: {student_id}")
    print(f"이름: {info['name']}")
    print(f"나이: {info['age']}")
    print("성적:")
    for subject, grade in info["grades"].items():
        print(f"  {subject}: {grade}점")

5.4 집합 (Sets)​

집합은 중복되지 않는 요소들의 순서 없는 컬렉션입니다.

5.4.1 집합 생성​

# 중괄호를 사용한 생성
empty_set = set()  # 주의: {}는 빈 딕셔너리!
numbers = {1, 2, 3, 4, 5}
fruits = {"apple", "banana", "cherry"}

# 중복 요소는 자동으로 제거됨
duplicate = {1, 2, 2, 3, 3, 3}
print(duplicate)  # {1, 2, 3}

# set() 함수 사용
from_list = set([1, 2, 3, 2, 1])
print(from_list)  # {1, 2, 3}

from_string = set("hello")
print(from_string)  # {'h', 'e', 'l', 'o'}

# 리스트에서 중복 제거
numbers = [1, 2, 2, 3, 3, 3, 4, 5, 5]
unique = list(set(numbers))
print(unique)  # [1, 2, 3, 4, 5] (순서는 보장되지 않음)

5.4.2 집합 연산​

# 기본 집합
a = {1, 2, 3, 4, 5}
b = {4, 5, 6, 7, 8}

# 합집합 (Union)
print(a | b)              # {1, 2, 3, 4, 5, 6, 7, 8}
print(a.union(b))         # 동일

# 교집합 (Intersection)
print(a & b)              # {4, 5}
print(a.intersection(b))  # 동일

# 차집합 (Difference)
print(a - b)              # {1, 2, 3}
print(a.difference(b))    # 동일
print(b - a)              # {6, 7, 8}

# 대칭 차집합 (Symmetric Difference) - 합집합에서 교집합을 뺀 것
print(a ^ b)                        # {1, 2, 3, 6, 7, 8}
print(a.symmetric_difference(b))    # 동일

# 부분집합 확인
small = {1, 2}
print(small.issubset(a))     # True
print(small <= a)            # True

# 상위집합 확인
print(a.issuperset(small))   # True
print(a >= small)            # True

# 서로소 확인 (교집합이 없는지)
c = {10, 11, 12}
print(a.isdisjoint(c))       # True

5.4.3 집합 메서드​

fruits = {"apple", "banana", "cherry"}

# add() - 요소 추가
fruits.add("date")
print(fruits)  # {'apple', 'banana', 'cherry', 'date'}

# update() - 여러 요소 추가
fruits.update(["elderberry", "fig"])
print(fruits)  # {'apple', 'banana', 'cherry', 'date', 'elderberry', 'fig'}

# remove() - 요소 제거 (없으면 KeyError)
fruits.remove("banana")
print(fruits)

# discard() - 요소 제거 (없어도 에러 없음)
fruits.discard("grape")     # 에러 없음
fruits.discard("apple")     # 제거됨

# pop() - 임의의 요소 제거 및 반환
item = fruits.pop()
print(f"제거된 항목: {item}")

# clear() - 모든 요소 제거
temp = {1, 2, 3}
temp.clear()
print(temp)  # set()

# copy() - 집합 복사
original = {1, 2, 3}
copied = original.copy()

리스트• 순서가 중요할 때

• 중복 값이 필요할 때
• 인덱스로 접근할 때		• 데이터가 변경되지 않을 때
• 빠른 검색이 필요할 때
• 중복을 허용하지 않을 때
튜플• 데이터가 변경되지 않을 때
• 딕셔너리 키로 사용할 때
• 함수에서 여러 값 반환 시
• 메모리 효율이 중요할 때		• 데이터를 자주 수정할 때
• 요소를 추가/삭제할 때
딕셔너리• 키-값 쌍으로 데이터 저장 시
• 빠른 검색이 필요할 때
• 레이블이 있는 데이터 관리 시
• JSON 같은 구조적 데이터		• 순서가 중요할 때 (Python 3.7+ 제외)
• 키가 불변이 아닐 때
• 단순 값 목록만 필요할 때
집합• 중복을 제거할 때
• 멤버십 테스트가 많을 때
• 수학적 집합 연산이 필요할 때
• 고유한 값만 필요할 때		• 순서가 중요할 때

• 중복 값이 필요할 때

# 학생 성적 관리 시스템
class GradeSystem:
    def __init__(self):
        # 딕셔너리: 학생 정보 (빠른 검색)
        self.students = {}
        # 집합: 등록된 과목 (중복 없음)
        self.subjects = set()
        # 리스트: 시험 일정 (순서 중요)
        self.exam_schedule = []

    def add_student(self, student_id, name):
        self.students[student_id] = {
            "name": name,
            "grades": {}  # 중첩 딕셔너리
        }

    def add_subject(self, subject):
        self.subjects.add(subject)

    def add_grade(self, student_id, subject, grade):
        if student_id in self.students and subject in self.subjects:
            self.students[student_id]["grades"][subject] = grade

    def schedule_exam(self, date, subject):
        # 튜플: 불변 데이터
        self.exam_schedule.append((date, subject))

# 사용 예시
system = GradeSystem()
system.add_student("2024001", "김철수")
system.add_subject("수학")
system.add_subject("영어")
system.add_grade("2024001", "수학", 95)
system.schedule_exam("2024-11-01", "수학")

# 1. 학생별 평균 계산
def calculate_averages(data):
    averages = {}
    for name, grades in data.items():
        avg = sum(grades.values()) / len(grades)
        averages[name] = round(avg, 2)
    return averages

for name, avg in averages.items():
    print(f"{name}: {avg}점")

    subjects = ["국어", "영어", "수학"]
    top_students = {}

    for subject in subjects:
        max_score = 0
        top_student = ""

        for name, grades in data.items():
            if grades[subject] > max_score:
                max_score = grades[subject]
                top_student = name

        top_students[subject] = (top_student, max_score)

    return top_students

    print(f"{subject} 최고점: {name} ({score}점)")

5.9.3 최적화 팁​

# ❌ 비효율적: 리스트에 반복적으로 검색
def find_duplicates_slow(numbers):
    duplicates = []
    for num in numbers:
        if numbers.count(num) > 1 and num not in duplicates:
            duplicates.append(num)
    return duplicates

# ✅ 효율적: 집합 사용
def find_duplicates_fast(numbers):
    seen = set()
    duplicates = set()
    for num in numbers:
        if num in seen:
            duplicates.add(num)
        else:
            seen.add(num)
    return list(duplicates)

# ❌ 비효율적: 리스트 연결
result = []
for i in range(1000):
    result = result + [i]  # 매번 새 리스트 생성!

# ✅ 효율적: append 사용
result = []
for i in range(1000):
    result.append(i)

# 또는 리스트 컴프리헨션
result = [i for i in range(1000)]

연습 문제 1: 리스트 조작​

다음 요구사항을 만족하는 함수를 작성하세요:

  1. 리스트에서 중복을 제거하되 순서는 유지
  2. 짝수만 필터링
  3. 각 요소를 제곱
# 입력: [1, 2, 2, 3, 4, 4, 5, 6]
# 출력: [4, 16, 36]
풀이 보기
def process_list(numbers):
    # 1. 중복 제거 (순서 유지)
    unique = []
    seen = set()
    for num in numbers:
        if num not in seen:
            unique.append(num)
            seen.add(num)

    # 2. 짝수만 필터링
    evens = [num for num in unique if num % 2 == 0]

    # 3. 제곱
    squared = [num**2 for num in evens]

    return squared

# 한 줄로 작성
def process_list_oneliner(numbers):
    seen = set()
    return [x**2 for x in numbers
            if x % 2 == 0 and not (x in seen or seen.add(x))]

# 테스트
numbers = [1, 2, 2, 3, 4, 4, 5, 6]
print(process_list(numbers))  # [4, 16, 36]

연습 문제 2: 딕셔너리 통계​

학생들의 시험 점수가 주어졌을 때, 다음을 계산하는 프로그램을 작성하세요:

  1. 과목별 평균 점수
  2. 각 학생의 총점과 평균
  3. 전체 수석 학생
scores = {
    "김철수": {"국어": 85, "영어": 90, "수학": 95},
    "이영희": {"국어": 92, "영어": 88, "수학": 91},
    "박민수": {"국어": 78, "영어": 85, "수학": 80}
}
풀이 보기
def analyze_scores(scores):
    # 1. 과목별 평균
    subjects = ["국어", "영어", "수학"]
    subject_averages = {}

    for subject in subjects:
        total = sum(student[subject] for student in scores.values())
        average = total / len(scores)
        subject_averages[subject] = round(average, 2)

    # 2. 학생별 총점과 평균
    student_stats = {}
    for name, grades in scores.items():
        total = sum(grades.values())
        average = total / len(grades)
        student_stats[name] = {
            "총점": total,
            "평균": round(average, 2)
        }

    # 3. 수석 학생
    top_student = max(student_stats.items(), key=lambda x: x[1]["평균"])

    return {
        "과목별_평균": subject_averages,
        "학생별_통계": student_stats,
        "수석": top_student
    }

scores = {
    "김철수": {"국어": 85, "영어": 90, "수학": 95},
    "이영희": {"국어": 92, "영어": 88, "수학": 91},
    "박민수": {"국어": 78, "영어": 85, "수학": 80}
}

result = analyze_scores(scores)
print("과목별 평균:", result["과목별_평균"])
print("학생별 통계:", result["학생별_통계"])
print(f"수석: {result['수석'][0]} (평균 {result['수석'][1]['평균']}점)")

연습 문제 3: 집합 연산​

  2. A 쇼핑몰에만 있는 상품

def compare_malls(mall_a, mall_b):
    # 1. 교집합 - 두 쇼핑몰 모두에서 판매
    common = mall_a & mall_b

    # 2. 차집합 - A에만 있는 상품
    only_a = mall_a - mall_b

    # 3. 대칭차집합 - 어느 한쪽에만 있는 상품
    exclusive = mall_a ^ mall_b

    return {
        "공통_상품": common,
        "A_전용": only_a,
        "배타적_상품": exclusive,
        "전체_상품": mall_a | mall_b
    }

mall_a = {"노트북", "마우스", "키보드", "모니터", "헤드셋"}
mall_b = {"노트북", "키보드", "스피커", "웹캠", "마이크"}

result = compare_malls(mall_a, mall_b)

print("A 쇼핑몰 전용:", result["A_전용"])

print("전체 상품:", result["전체_상품"])

연습 문제 4: 중첩 자료구조​

도서 관리 시스템을 만들어보세요. 각 도서는 제목, 저자, 출판연도, 대출 여부를 가집니다.

  1. 대출 가능한 도서 목록
  2. 특정 저자의 도서 찾기
  3. 연도별 도서 수
풀이 보기
class Library:
    def __init__(self):
        self.books = []

    def add_book(self, title, author, year, available=True):
        book = {
            "title": title,
            "author": author,
            "year": year,
            "available": available
        }
        self.books.append(book)

    def available_books(self):
        """대출 가능한 도서 목록"""
        return [book["title"] for book in self.books if book["available"]]

    def books_by_author(self, author):
        """특정 저자의 도서"""
        return [book["title"] for book in self.books if book["author"] == author]

    def books_by_year(self):
        """연도별 도서 수"""
        year_count = {}
        for book in self.books:
            year = book["year"]
            year_count[year] = year_count.get(year, 0) + 1
        return year_count

    def checkout_book(self, title):
        """도서 대출"""
        for book in self.books:
            if book["title"] == title and book["available"]:
                book["available"] = False
                return f"'{title}' 대출 완료"
        return f"'{title}'은(는) 대출할 수 없습니다."

# 사용 예시
library = Library()
library.add_book("파이썬 기초", "김철수", 2023)
library.add_book("자료구조", "이영희", 2022)
library.add_book("알고리즘", "박민수", 2023)
library.add_book("웹 개발", "김철수", 2022)

print("대출 가능:", library.available_books())
print("김철수 저자:", library.books_by_author("김철수"))
print("연도별 도서:", library.books_by_year())
print(library.checkout_book("파이썬 기초"))
print("대출 가능:", library.available_books())

5.11 체크리스트​

자료구조를 사용할 때 다음 사항을 확인하세요:

  • 리스트 사용 시

    • 순서가 중요한가?
    • 데이터를 자주 수정하는가?
    • 인덱스로 접근해야 하는가?
    • append보다 + 연산을 자주 사용하고 있지는 않은가?

  • 튜플 사용 시

    • 데이터가 변경되지 않는가?
    • 딕셔너리 키로 사용할 계획인가?
    • 단일 항목 튜플 생성 시 쉼표를 넣었는가? (item,)

  • 딕셔너리 사용 시

    • 키는 불변 타입인가? (문자열, 숫자, 튜플)
    • 키가 존재하지 않을 수 있다면 get() 사용을 고려했는가?
    • 딕셔너리 순회 시 items() 사용을 고려했는가?

  • 집합 사용 시

    • 중복을 허용하지 않는가?
    • 순서가 중요하지 않은가?
    • 멤버십 테스트가 많은가?
    • 집합 연산(합집합, 교집합 등)이 필요한가?

  • 성능 고려

    • 빈번한 검색이 있다면 리스트 대신 집합/딕셔너리를 사용하는가?
    • 리스트 컴프리헨션이 너무 복잡해지지 않았는가?
    • 대용량 데이터에서 불필요한 복사를 하고 있지는 않은가?

5.12 핵심 정리​

자료구조별 특징​

자료구조순서변경 가능중복 허용인덱싱주요 용도
리스트✓✓✓✓순서 있는 데이터 모음
튜플✓✗✓✓불변 데이터, 다중 반환
딕셔너리✓*✓키: ✗, 값: ✓키로키-값 매핑
집합✗✓✗✗고유값, 집합 연산

*Python 3.7+부터 삽입 순서 보장

기억해야 할 포인트​

  1. 리스트는 가장 다용도이지만, 검색이 느림
  2. 튜플은 리스트보다 빠르고 메모리 효율적이지만 수정 불가
  3. 딕셔너리는 키를 통한 빠른 조회에 최적화
  4. 집합은 중복 제거와 멤버십 테스트에 최고
  5. 컴프리헨션은 간결하고 빠르지만 가독성을 해치지 않도록 주의
자료구조 선택 요령
  1. 순서가 중요하고 수정이 필요하면 → 리스트
  2. 순서가 중요하지만 수정이 불필요하면 → 튜플
  3. 키-값 쌍으로 데이터를 관리하면 → 딕셔너리
  4. 중복 없이 고유한 값만 필요하면 → 집합
