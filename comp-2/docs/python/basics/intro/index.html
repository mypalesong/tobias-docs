1. Python 소개 및 설치

Python의 세계에 오신 것을 환영합니다! 이 문서에서는 Python을 설치하고 첫 프로그램을 작성합니다.

1.1 Python이란?​

Python은 1991년 귀도 반 로섬(Guido van Rossum)이 만든 고수준 프로그래밍 언어입니다.

특징​

✅ **읽기 쉬운 문법**
   들여쓰기로 코드 블록 구분
   영어와 유사한 키워드

✅ **인터프리터 언어**
   컴파일 없이 즉시 실행
   빠른 개발과 테스트

✅ **다중 패러다임**
   절차적, 객체지향, 함수형 프로그래밍 모두 지원

✅ **크로스 플랫폼**
   Windows, macOS, Linux에서 동일하게 동작

Python으로 할 수 있는 것​

**웹 개발**
- Django, Flask로 웹사이트 구축
- 예: Instagram, Spotify

**데이터 분석**
- pandas, NumPy로 데이터 처리
- matplotlib으로 시각화

**인공지능/머신러닝**
- TensorFlow, PyTorch
- scikit-learn

**자동화**
- 반복 작업 자동화
- 웹 스크래핑

**게임 개발**
- Pygame

**네트워크/보안**
- 스크립팅, 보안 도구

1.2 Python 설치​

Windows​

  1. 공식 사이트에서 다운로드

    • python.org/downloads 방문
    • "Download Python 3.x.x" 클릭

  2. 설치 진행

    ⚠️ 중요: "Add Python to PATH" 체크!

  3. 확인

    # 명령 프롬프트(cmd)에서
    python --version

    출력 예시:

    Python 3.11.5

macOS​

방법 1: 공식 설치 파일

# python.org에서 macOS 설치 파일 다운로드 후 설치

방법 2: Homebrew (추천)

# Homebrew 설치 (없는 경우)
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Python 설치
brew install python3

# 확인
python3 --version

Linux (Ubuntu/Debian)​

# 업데이트
sudo apt update

# Python 3 설치
sudo apt install python3 python3-pip

# 확인
python3 --version
pip3 --version

1.3 개발 환경 설정​

VS Code 설치 (추천)​

  1. VS Code 다운로드

    • code.visualstudio.com

  2. Python 확장 설치

    • VS Code 열기
    • Extensions (Ctrl+Shift+X)
    • "Python" 검색하여 설치

  3. 설정 확인

    - 자동 완성 작동 확인
    - 문법 강조 확인
    - 인터프리터 선택 (Ctrl+Shift+P → "Python: Select Interpreter")

PyCharm (선택)​

**Community Edition (무료)**
- jetbrains.com/pycharm/download

**장점:**
- 강력한 디버깅
- 통합 도구
- 프로젝트 관리

**단점:**
- 무겁다
- 초보자에게는 과함

1.4 첫 Python 프로그램​

대화형 모드​

터미널에서 Python을 바로 실행할 수 있습니다:

python

또는

python3

Python 프롬프트가 나타납니다:

>>>

첫 코드를 입력해보세요:

>>> print("Hello, Python!")
Hello, Python!

>>> 2 + 2
4

>>> name = "김철수"
>>> print(f"안녕하세요, {name}님!")
안녕하세요, 김철수님!

종료:

>>> exit()

파일로 실행​

  1. 파일 생성

    hello.py 파일을 만들고 다음 내용을 입력:

    # hello.py
    print("Hello, Python!")
    print("첫 프로그램입니다.")

    name = input("이름을 입력하세요: ")
    print(f"안녕하세요, {name}님!")

  2. 실행

    python hello.py

    출력:

    Hello, Python!
    첫 프로그램입니다.
    이름을 입력하세요: _

1.5 Python의 기본 개념​

주석 (Comments)​

# 한 줄 주석

"""
여러 줄 주석
(독스트링이라고도 함)
"""

'''
작은따옴표로도 가능
'''

출력 (Print)​

# 기본 출력
print("Hello")

# 여러 개 출력
print("Hello", "World")  # Hello World

# 구분자 변경
print("Hello", "World", sep="-")  # Hello-World

# 줄바꿈 제거
print("Hello", end=" ")
print("World")  # Hello World

변수 (Variables)​

# 변수 할당
name = "김철수"
age = 25
height = 175.5

# 여러 변수 동시 할당
x, y, z = 1, 2, 3

# 같은 값 할당
a = b = c = 0

# 변수 이름 규칙
user_name = "good"  # ✅ 스네이크 케이스 (Python 관례)
userName = "ok"     # ⚠️ 카멜 케이스 (가능하지만 비권장)
UserName = "ok"     # ⚠️ 파스칼 케이스 (클래스명에 사용)
123name = "bad"     # ❌ 숫자로 시작 불가
user-name = "bad"   # ❌ 하이픈 불가

입력 (Input)​

# 사용자 입력 받기
name = input("이름: ")
print(f"입력한 이름: {name}")

# 주의: input()은 항상 문자열 반환
age = input("나이: ")  # "25" (문자열)
age = int(input("나이: "))  # 25 (정수)

1.6 간단한 계산기 만들기​

실습으로 간단한 계산기를 만들어봅시다:

# calculator.py
print("=== 간단한 계산기 ===")

# 숫자 입력
num1 = float(input("첫 번째 숫자: "))
num2 = float(input("두 번째 숫자: "))

# 계산
addition = num1 + num2
subtraction = num1 - num2
multiplication = num1 * num2
division = num1 / num2

# 결과 출력
print(f"\n결과:")
print(f"{num1} + {num2} = {addition}")
print(f"{num1} - {num2} = {subtraction}")
print(f"{num1} × {num2} = {multiplication}")
print(f"{num1} ÷ {num2} = {division}")

실행 결과:

=== 간단한 계산기 ===
첫 번째 숫자: 10
두 번째 숫자: 3

결과:
10.0 + 3.0 = 13.0
10.0 - 3.0 = 7.0
10.0 × 3.0 = 30.0
10.0 ÷ 3.0 = 3.3333333333333335

1.7 일반적인 문제 해결​

"python을 찾을 수 없습니다"​

❌ 'python'은(는) 내부 또는 외부 명령... 아닙니다.

해결:

  1. PATH에 Python이 추가되었는지 확인
  2. 재설치 시 "Add Python to PATH" 체크
  3. 또는 python3 사용 (macOS/Linux)

한글 출력 오류​

# 파일 상단에 추가
# -*- coding: utf-8 -*-

또는 Python 3에서는 기본적으로 UTF-8이므로 파일을 UTF-8로 저장

들여쓰기 오류​

❌ IndentationError: unexpected indent

해결:

  • Tab과 Space를 섞어 쓰지 말 것
  • VS Code 설정: Space 4칸 (Python 표준)

1.8 실습 과제​

과제 1: 자기소개 프로그램​

# 사용자 정보 입력받기
# 이름, 나이, 좋아하는 것
# 형식에 맞춰 출력하기
해답 보기
name = input("이름: ")
age = int(input("나이: "))
hobby = input("취미: ")

print(f"\n=== 자기소개 ===")
print(f"이름: {name}")
print(f"나이: {age}세")
print(f"취미: {hobby}")

과제 2: 온도 변환기​

# 섭씨를 화씨로 변환
# 공식: F = C × 9/5 + 32
해답 보기
celsius = float(input("섭씨 온도: "))
fahrenheit = celsius * 9/5 + 32
print(f"{celsius}°C = {fahrenheit}°F")
